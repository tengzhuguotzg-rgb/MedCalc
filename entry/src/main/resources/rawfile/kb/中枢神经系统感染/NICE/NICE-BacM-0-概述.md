## 核心推荐速览

> [!quote] Red Flag Combination（细菌性脑膜炎强疑似）
> 发热 + 头痛 + 颈强直 + 意识/认知改变

> [!quote] Red Flag Symptoms（脑膜炎球菌病）
> 瘀点/紫癜样皮疹 + 发热 + 任一脑膜炎征象

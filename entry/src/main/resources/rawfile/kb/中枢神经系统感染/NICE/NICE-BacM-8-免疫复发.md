## 🧪 1. 免疫缺陷评估（Rec 1.10.1-1.10.5）

### 1.1 HIV 检测

| 人群 | 推荐 |
|------|------|
| 成人 | ✅ 常规检测 |
| 婴儿/儿童/年轻人 | ⚠️ 有免疫缺陷征象或 HIV 风险因素时考虑 |

### 1.2 免疫学转诊

> [!important] 强推荐
> 肺炎球菌脑膜炎婴儿 → 转介儿科免疫学和感染病专科评估**原发性免疫缺陷**（Rec 1.10.3）。

### 1.3 解剖结构检查

| 检查 | 适用人群 |
|------|---------|
| 背部/头皮检查（排查窦道）| 婴儿和小幼儿细菌性脑膜炎 |

### 1.4 病史采集

| 采集内容 | 说明 |
|---------|------|
| 头部创伤/手术/CSF漏 | 复发风险 |
| 疫苗接种史 | — |
| 用药史（含免疫抑制剂如补体抑制剂）| — |

## ⚠️ 2. 复发危险因素（Rec 1.14.1）

| 危险因素 | 说明 |
|---------|------|
| 原发或继发性免疫缺陷 | 含先天性补体缺乏或获得性抑制 |
| 脾功能减退或缺失 | — |
| CSF漏 | 头部创伤/手术/先天畸形 |
| 既往细菌性脑膜炎/脑膜炎球菌病 | — |
| 耳蜗植入物 | — |

---

## 📊 3. 复发管理原则

| 要求 | 说明 |
|------|------|
| 详细评估 | 所有复发患者需全面评估潜在病因 |
| 免疫科转诊 | 存在免疫缺陷时及时转诊 |
| 预防措施 | 疫苗接种（脑膜炎球菌、Hib、肺炎球菌） |

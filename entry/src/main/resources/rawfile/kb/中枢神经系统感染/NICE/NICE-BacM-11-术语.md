## 👥 人群定义

| 术语 | 年龄定义 |
|------|---------|
| **婴儿**（Babies）| < 12月龄 |
| **幼儿**（Young children）| 1-4岁 |
| **儿童**（Children）| 5-11岁 |
| **年轻人**（Young people）| 12-17岁 |
| **成人**（Adults）| ≥ 18岁 |
| **老年人**（Older adults）| 通常指 > 65岁（临床判断）|
| **年轻成人**（Young adults）| 18-25岁（学生人群相关）|

## 🔬 特殊术语

| 术语 | 说明 |
|------|------|
| **肠杆菌（coliforms）** | 又称大肠杆菌，涉及抗生素方案 |
| **感染专科医生**（Infection specialist）| 感染科/传染科专科会诊 |
| **高级临床决策者**（Senior clinical decision maker）| 负责初始评估的高年资医生 |
| **严重抗生素过敏** | 过敏性休克/严重皮肤反应等 |
| **强疑似**（Strongly suspected）| 符合Red Flag Combination/Symptoms |

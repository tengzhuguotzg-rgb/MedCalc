# 神经重症撤机 - 概述

## 定义与核心概念

**成功撤机**定义为：拔管后 48 小时内无需机械通气维持自主通气[^1]。该时间窗口存在争议，部分作者建议延伸至 72 小时乃至 7 天[^2]。

**神经重症监护（Neurocritical care）** 是重症医学的亚专科，专注于管理伴有危及生命的神经系统或神经外科疾病的急性危重症患者，或伴有全身性疾病的危及生命的神经系统表现的患者。

> [!tip] 临床提示
> 急性脑损伤（Acute Brain Injury, ABI）定义为 GCS ≤ 12 分，且头颅 CT 至少有一项与急性病变相关的异常（硬膜外血肿、蛛网膜下腔出血、脑挫伤、血肿、脑水肿、颅骨骨折、卒中、脑脓肿）。

## 核心挑战

急性脑损伤患者撤机的核心矛盾：

1. **撤机延迟的危害**：延长机械通气增加 ICU 死亡率[^3]，晚期拔管增加非计划性拔管风险，影响预后
2. **撤机失败的后果**：拔管失败率高达 10%～35%，再插管与死亡率增加相关[^4]

> [!warning] 关键挑战
> 神经重症患者常被排除在随机对照试验之外，专门针对这一人群的撤机数据极为匮乏。临床上需要在"延迟拔管"与"再插管"之间找到精确平衡点。

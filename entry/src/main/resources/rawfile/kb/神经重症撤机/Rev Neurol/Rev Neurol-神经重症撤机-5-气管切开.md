# 神经重症撤机 - 气管切开

## 气管切开的潜在优势[^1]

| 优势 | 说明 |
|------|------|
| 降低非计划性拔管风险 | 固定可靠 |
| 减少气道创伤 | 避免长期经口气管插管的并发症 |
| 减少鼻窦炎 | 经口气管插管更易并发 |
| 降低气道阻力 | 减少呼吸功 |
| 提高患者耐受性 | 减少镇静需求 |

## 气管切开的相关风险[^3]

**不是零风险手术**：手术部位感染、出血、纵隔气肿、气胸，死亡均有报告。

## 急性脑损伤中的气管切开现状[^4]

| 数据 | 数值 |
|------|------|
| 拔管失败率 | ~38% |
| 气管切开率 | 32%～45% |
| 完全未尝试撤机即行气管切开 | 高达 79% |

> [!warning] 异常现象
> 高达 79% 的急性脑损伤患者在未尝试任何撤机的情况下直接行气管切开——这一比例极高，提示该人群中撤机和拔管决策存在显著困难或临床惯性。

**颅内压或脑灌注压不稳定时，建议推迟气管切开**[^6]。

## 5.1 需要气管切开的预测因素

气管切开需求因脑损伤类型而异[^1]：

**严重急性脑损伤（卒中/TBI/心脏骤停后缺血缺氧性脑病）**

- 年轻
- 男性
- 非白人种族
- 大型、城市、大学附属医院（可能反映转诊偏倚或疾病严重度差异）

**创伤性脑损伤（TBI）独立预测因素**：

| 评分/指标 | 说明 |
|------|------|
| CRASH 评分 | 更大分值 = 更可能需要气管切开 |
| IMPACT 评分 | 同上 |
| SAPS II / APACHE II | 疾病严重度评分 |
| 年龄 | 年龄越大，气管切开风险越高 |
| 修订创伤评分 | 整体创伤严重度 |
| 瞳孔反应性 | 无反应 = 风险增加 |
| 小脑幕切迹疝（basal cistern collapse） | 结构改变提示预后差 |
| 硬膜下血肿 | 特定病理类型 |

**卒中相关**：吞咽困难 + GCS < 10 + 脑积水 + 脑干病变 + 颅内出血 + 手术干预 + 全身器官功能（COPD、PaO₂/FiO₂ < 150、脓毒症、肺损伤评分 > 1、APS > 20）→ SET 评分

**颅内出血**：低 GCS、COPD、颅内出血量与位置、中线移位、脑室内出血、脑积水 → **TRACH 评分**（GCS + 影像学三因素：脑积水、室间隔移位、丘脑部位出血），ROC = 0.92，辨别力极强[^7]。

> [!tip] 临床意义
> TRACH 评分可早期识别需要气管切开的高危患者，帮助提前规划呼吸道管理路径。

## 5.2 早期 vs 晚期气管切开[^2]

**通用 ICU 人群**：指南建议 MV 第 4 天之前避免行气管切开[^8]。

**急性脑损伤中的证据**：

| 研究人群 | 发现 |
|------|------|
| 卒中/SAH 观察性研究 | 早期（≤3 天）vs 晚期（7～14 天）气管切开，ICU 时长相似 |
| 严重缺血性/出血性卒中 RCT | 早期（1～3 天）vs 标准（7～14 天）气管切开：ICU 死亡率和 6 个月死亡率**更低**[^9] |
| SAH | 气管切开率 17%[^10]；气管切开时间越长，肺部并发症（气胸、ARDS、术后肺部并发症）、静脉血栓栓塞、肺炎越多；时间越短，住院时长越短[^11] |
| 433 例 TBI 队列 | 7 天内气管切开：更好的神经系统预后、更低死亡率、更短住院时长[^13] |
| TBI Meta 分析（8 项 RCT） | 早期（≤10 天）vs 晚期（> 10 天）：MV 时间更短、VAP 发生率更低[^14] |
| TBI 回顾性队列（3 天内 vs 3 天后） | 30 天死亡率：3% vs 8%（P = 0.38）；3 天内组 MV 时间和 ICU 时长更短，不良事件和肺炎无差异[^15] |
| 98 例 TBI 队列 | 超早期气管切开：ICU 时长、抗生素使用、住院费用、VAP 率均更低，死亡率无差异[^16] |

> [!note] 总体趋势
> 在急性脑损伤人群中，**倾向于更早气管切开**（7～10 天内），可缩短 MV 时间、减少肺部并发症、降低住院时长，且不增加不良事件。

## 5.3 气管切开技术[^3]

指南推荐**床旁经皮气管切开**作为 ICU 标准技术[^17]——相比外科切开，手术时间更短，手术部位感染和炎症更少。

在急性脑损伤患者中，**由重症医学科医生在床旁操作** vs 外科医生在手术室：前者气管切开时机更早、MV 时间更短[^18]。

## 5.4 康复与拔管（decannulation）[^4]

气管切开保留至气道保护反射、吞咽功能、意识和认知恢复至能有效清除分泌物并控制上气道。

**拔管标准**（各评估组合的最优预测规则）[^19]：

1. **气管切开套囊放气 ≥ 72 小时**（耐受良好）
2. **内镜评估气道通畅性**
3. **吞咽仪器评估**
4. **蓝染测试**

> [!tip] 安全拔管原则
> 及时、安全地拔除气管切开管可改善预后[^20]。在神经重症患者中，拔管决策应综合评估咳嗽、吞咽、气道通畅性和意识状态，而非单纯依赖 GCS。

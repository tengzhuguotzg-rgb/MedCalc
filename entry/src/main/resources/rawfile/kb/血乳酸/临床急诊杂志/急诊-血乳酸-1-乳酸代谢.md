# 血乳酸在急危重症应用的急诊专家共识 第1章 乳酸代谢

## 1.1 乳酸的产生

乳酸产生在细胞质中进行，主要通过糖酵解途径。丙酮酸进入线粒体后被氧化，或被乳酸脱氢酶还原为乳酸（无氧糖酵解）。乳酸在细胞中以乳酸盐形式存在，通过单羧酸转运蛋白跨细胞膜转运。

**无氧糖酵解（anaerobic glycolysis）**是乳酸产生的主要来源。正常情况下乳酸是代谢的副产物而非代谢废物。

## 1.2 乳酸的代谢与清除

乳酸在体内的清除主要通过三条途径：**氧化代谢**（50%~75%经氧化途径直接转化为丙酮酸进入三羧酸循环）、**糖异生**（经Cori循环在肝脏和肾脏转化为葡萄糖）、**肾脏排泄**（仅贡献约10%~12%）。

乳酸清除效率受组织血流量、氧供应、乳酸转运蛋白功能等多种因素影响。乳酸清除速率是评估组织灌注和代谢状态的重要指标。

## 1.3 乳酸的分类

乳酸分为**L-乳酸**和**D-乳酸**两种旋光异构体：

- **L-乳酸**：主要由糖酵解产生，是缺氧状态下主要的乳酸产物，是"乳酸穿梭"理论的核心，参与细胞内和细胞间的能量传递。
- **D-乳酸**：主要由肠道细菌（乳酸菌）通过D-乳酸脱氢酶产生。人体缺乏代谢D-乳酸的酶，主要通过肾脏代谢。研究表明D-乳酸可能参与癌症细胞代谢（Warburg效应）。

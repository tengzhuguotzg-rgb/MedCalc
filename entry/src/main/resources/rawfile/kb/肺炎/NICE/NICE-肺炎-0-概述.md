## 指南定位与背景

### 编制背景

2022年NICE监测报告指出CG191（成人肺炎）需要更新，尤其是纳入<18岁年龄段——该年龄段此前被排除在外。此外，存在两份独立的NICE儿童和成人抗生素处方指南（CAP和HAP）。本次更新将抗生素处方指南与CG191的**部分更新**合并，形成**统一肺炎推荐**。

### 流行病学

| 核心数据 | 数值 |
|---------|------|
| CAP年发病率 | 5~10/1,000成人 |
| CAP占基层LRTI比例 | 5%~12% |
| CAP需住院比例 | 22%~42% |
| HAP发生率 | 0.5%~2%住院患者 |
| HAP延长住院 | 平均7~9天/人 |
| 英国肺炎年死亡数 | **29,000例** |
| CAP住院30天死亡率 | 5%~15%（ICU亚组升至30%） |

## 流行病学背景

- **美国年 AMI 发病数**：约 80.5 万例（首次 60.5 万，复发 20 万）
- **死亡率改善**：2011–2021 年冠心病年死亡率下降 15.0%（归因于及时再灌注和二级预防）
- **经济负担**：年总成本约 849 亿美元（含直接医疗费用和间接生产力损失）
- **种族差异**：美国 Black 男性 AMI 发病率高于 non-Hispanic White 男性（多因素）
- **性别差异**：女性平均比男性晚 10 年发生 ACS；女性更常表现为非胸痛伴随症状，导致识别和治疗的延迟

## 推荐强度与证据质量（COR & LOE）

**Table 2. COR & LOE 定义**

| 等级 | 含义 |
|------|------|
| **COR 1（强推荐）** | 获益 >> 风险；推荐执行 |
| **COR 2a（中等推荐）** | 获益 > 风险；建议执行 |
| **COR 2b（弱推荐）** | 获益 ≥ 风险；可能建议执行 |
| **COR 3（不推荐）** | 无获益或有害 |

| 等级 | 含义 |
|------|------|
| **LOE A** | 高质量 RCT 或高质量 Mate 分析 |
| **LOE B-R** | 中等质量 RCT 或观察性研究（随机的） |
| **LOE B-NR** | 中等质量非随机化观察性研究 |
| **LOE C-LD** | 设计有限或结论不强的观察性研究 |
| **LOE C-EO** | 专家共识、病例标准 |

## 核心管理策略（Top Take-Home Messages）

### 1. DAPT 与 P2Y12 抑制剂
- 所有 ACS 患者推荐 DAPT
- **替格瑞洛或 prasugrel** 优于氯吡格雷（PCI 患者）
- NSTE-ACS 拟行择期侵入策略（血管造影时间 >24h）：可考虑预先使用氯吡格雷或替格瑞洛减少 MACE

### 2. DAPT 时长
- 默认策略：**至少 12 个月** DAPT（非高出血风险患者）
- 出血风险管理策略：
  - 胃肠道出血风险：推荐 PPI
  - 已耐受替格瑞洛 DAPT：≥1个月后转为替格瑞洛单药
  - 长期抗凝需求：PCI 后 1–4 周停用阿司匹林，继续 P2Y12 抑制剂（首选氯吡格雷）

### 3. 调脂治疗
- **高强度他汀**推荐用于所有 ACS 患者
- 最大耐受他汀仍 LDL-C ≥70 mg/dL：推荐加用非他汀类降脂药（依折麦布、PCSK9 抑制剂、inclisiran、bempedoic acid）
- LDL-C 55–<70 mg/dL 且已用最大耐受他汀：可考虑进一步强化降脂

### 4. NSTE-ACS 侵入策略
- **中高危**：推荐住院期间行侵入策略（意图血运重建）以减少 MACE
- **低危**：推荐常规侵入或选择性侵入策略，进一步风险分层以识别需要血运重建的患者

### 5. PCI 操作要点
- **桡动脉入路**优于股动脉（减少出血、血管并发症和死亡）
- **腔内影像（IVUS/OCT）**推荐用于复杂冠脉病变 PCI 指导

### 6. 多支病变血运重建策略
- **STEMI 或 NSTE-ACS**：推荐完全血运重建
- NSTE-ACS 多支病变：血运重建方式（CABG vs 多支 PCI）应根据冠脉病变复杂度和合并症决定
- STEMI 多支病变：非罪犯血管 PCI 可单次或分次完成，倾向单次多支 PCI
- **心源性休克**：紧急罪犯血管血运重建；不推荐同时常规 PCI 非罪犯血管

### 7. 微轴流泵（心源性休克）
- 基于一项 RCT：特定心源性休克患者使用微轴流泵可能合理，可**减少死亡**
- 代价：出血、肢体缺血、肾功能衰竭发生率更高；需注意血管入路和脱机

### 8. 输血策略
- ACS 伴急性或慢性贫血（未活动性出血）：维持血红蛋白 ≥10 g/dL 可能合理

### 9. 出院后二级预防
- **空腹血脂检测**：开始或调整降脂治疗后 **4–8 周**复查
- **心脏康复**：推荐转诊（可选择居家项目）

## 指南结构（Contents）

| 章节 | 主题 | 核心内容 |
|------|------|---------|
| **1** | ACS 概览与定义 | 定义、三类分类（UA/NSTEMI/STEMI）、ECG 判读标准、MI Type 分型（1–5型） |
| **2** | 疑似 ACS 的初始评估 | 院前评估、院内初始评估（ECG ≤10分钟、hs-cTn 0/1h 方案）、GRACE/TIMI 风险评分 |
| **3** | 心脏骤停管理 | OHCA 转运决策、ROSC 后 PCI 时机（0.5–24h）、昏迷患者个体化评估（CMfcMcs） |
| **4** | 标准药物治疗 | 氧疗（SpO2 <94%）、镇痛（避免 NSAIDs）、阿司匹林、P2Y12、GP IIb/IIIa、肠外抗凝（UFH/依诺肝素/比伐卢定/磺达肝素）、调脂（高强度他汀 ±依折麦布 ±PCSK9）、β阻、RAASi |
| **5** | STEMI 再灌注策略 | 区域系统建设、PPCI 时间目标（FMC–器械 ≤90min 直达/≤120min 转诊）、溶栓（tenecteplase/reteplase/alteplase）、紧急 CABG |
| **6** | NSTE-ACS 侵入策略 | 常规侵入 vs 选择性侵入定义、侵入时机（<2h/<24h/<48–72h）、相对禁忌证（Table 15） |
| **7** | 导管室注意事项 | 桡动脉入路（首选）、血栓抽吸（不常规推荐）、腔内影像（IVUS/OCT 对比）、多支病变（Complete vs Culprit-only） |
| **8** | 心源性休克管理 | SHOCK 试验（紧急血运重建获益）、CULPRIT-SHOCK（仅罪犯血管 PCI）、微轴流泵（DanGer-SHOCK 降低死亡26%）、IABP（不降低死亡率） |
| **9** | ACS 并发症 | 机械并发症（室间隔破裂/乳头肌断裂/游离壁破裂 + MCS + 手术时机）、ICD 一级预防（WCD/永久起搏器）、心梗后心包炎（左室血栓抗凝3个月） |
| **10** | 住院管理与出院计划 | CICU 收治标准、贫血输血（MINT：目标10 g/dL 优于7–8 g/dL）、遥测、住院时长、早期出院（Zwolle 评分 ≤3）、患者教育（6项核心内容）、心脏康复（居家 CR 等效） |
| **11** | 二级预防与长期管理 | DAPT 时程（替格瑞洛单药 ≥1月后/高出血风险30天后单药）、调脂（4–8周复查）、SGLT-2i（心衰/糖尿病）、GLP-1RA（ASCVD+糖尿病/超重）、秋水仙碱（0.5–0.6 mg/d）、免疫接种（流感） |
| **12** | 证据缺口与未来方向 | ARNI 时机、高出血风险 PCI 策略、微轴流泵最佳患者选择等 |

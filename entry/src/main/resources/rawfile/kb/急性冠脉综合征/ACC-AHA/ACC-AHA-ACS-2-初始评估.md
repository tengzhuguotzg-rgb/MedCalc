## 3. 疑似ACS的初始评估与管理

### 3.1. 疑似ACS的初始评估

---

#### 3.1.1 院前评估与处理要点

**核心要点：**

- 疑似 ACS 患者应由急救医疗服务（EMS）转运至急诊科，而非自行就医
- EMS 转运允许在转运途中进行评估、监测和治疗（如心律失常或心脏骤停）
- 训练有素的急救人员应获取**聚焦病史+体格检查（含生命体征）+ 至少一份 12 导联 ECG**

**院前 ECG：**

- 基于 12 导联 ECG 结果（有无 ST 段抬高）进行分诊
- 心电图可远程传输至 PPCI 中心，缩短门-球时间
- 合并心衰、室性心律失常或心源性休克的 ACS 患者死亡风险最高，应优先转运至 PCI 中心

**院前处理推荐（Recommendation-Specific Supportive Text）：**

1. **院前 12 导联 ECG**：训练有素的急救人员记录院前 ECG 与 STEMI 早期再灌注时间缩短和死亡率降低相关；ST 段抬高或等效征象时按 STEMI 流程处理；NSTE-ACS 诊断不要求 ECG 一定异常。

2. **复查 ECG**：首份 ECG 无诊断意义时，转院途中应复查（尤其临床怀疑高、症状持续或病情恶化时）；院前第二或第三次 ECG 可额外识别约 15% 的 STEMI；后壁导联（V7–V9）适用于持续症状伴 V1–V3 孤立 ST 段压低的患者。

3. **PPCI 作为首选**：STEMI 患者首选直接经皮冠状动脉介入治疗（PPCI）；每延迟 30 分钟，1 年死亡率相对风险增加 7.5%；院前识别 STEMI 者应优先转运至 PCI 中心，目标首次医疗接触（FMC）至首次器械时间 ≤90 分钟；EMS 直接送入医院时，时间起点为 FMC；FMC-至-器械时间 ≤120 分钟在无法满足 ≤90 分钟时也可考虑。

4. **系统层面干预**：多项诊疗流程改进可显著缩短 STEMI 再灌注时间，包括：院前导管室激活、非 PCI 设施单次呼叫转诊协议、绕行急诊直达导管室。

---

#### 3.1.2 院内初始评估

**核心要点：**

- 胸痛是美国急诊科第二大主诉（仅次于发热）
- ACS 初始评估三要素：**病史+体格检查 + 及时心电图 + 心脏肌钙蛋白（cTn）**
- 高度疑似 ACS 者应立即启动初始药物治疗
- 合并心源性休克、血流动力学不稳定或疑似机械并发症时，应尽快完成床旁超声（PoCUS）
- 疑似 STEMI 者：快速再灌注优先，诊断检查不应延误导管室操作
- 临床决策路径（CDP）可减少 21%–43% 的不必要检查，同时维持对 MACE 的高阴性预测值

**ECG 要求：**
- 目标：患者到达后 **≤10 分钟**内完成 12 导联 ECG 并由训练有素的临床医师解读
- 初始无诊断意义的 ECG **不能排除 ACS**（ECG 异常可能动态演变）
- 应与既往 ECG 对比；复查 ECG 时机根据症状决定
- 下壁 STEMI 应加做右室导联评估右室受累
- NSTE-ACS 患者的 ECG 可能正常或显示短暂 ST 段抬高（高风险征象）、ST 段压低、T 波倒置或非特异性改变
- 前壁间隔导联（V1–V3）ST 段压低可能提示后壁 STEMI 进展

**心肌标志物——cTn 是首选：**

| 要点 | 说明 |
|------|------|
| **首选检测** | 高敏心肌肌钙蛋白（hs-cTn）优于传统 cTn |
| **诊断阈值** | cTn 峰值 >99th percentile URL = 心肌损伤 |
| **动态监测** | AMI 的特征为 cTn 升高伴至少一次高于 99th percentile 的值，且有升高和/或下降趋势 |
| **1–2 小时 CDP** | 重复 hs-cTn 检测（就诊时 + 1–2 小时后），阴性预测值 >99.5%（根据 assay 特异性阈值） |
| **单次 hs-cTn** | 若使用单次检测，须在症状发生后 ≥3 小时采样 |
| **传统 cTn** | 若使用传统 cTn 采样时间窗需延长至就诊后 3–6 小时 |
| **STEMI 溶栓** | 心电图已显示 STEMI 时，不应因等待标志物结果而延迟再灌注治疗 |
| **性别差异** | hs-cTn  assays 可能存在性别不同的截断值 |
| **正常范围内变化** | hs-cTn 在正常参考范围内但有变化时也可能提示心脏缺血，需进一步评估 |

### 3.1.3 风险分层工具

**核心要点：**
- ACS 患者短期死亡率和 MACE 风险存在较大个体差异
- 风险评估有助于判断预后、指导治疗决策
- GRACE 和 TIMI 风险评分是最经过充分验证的工具
- 风险评分**不能用于诊断**，仅用于风险分层

#### 主要评分工具比较（Table 5）

| | **GRACE 风险评分（2.0）** | **TIMI 风险评分（UA/NSTEMI）** | **TIMI 风险评分（STEMI）** |
|---|---|---|---|
| **适用人群** | ACS | 不稳定心绞痛或 NSTEMI | STEMI |
| **目标事件** | 住院期间/6月/1年/3年死亡或死亡/MI | 14天全因死亡、MI 或紧急血运重建 | 30天全因死亡 |
| **评分变量** | 年龄、Killip 分级、收缩压、心率、ST 段改变、心脏骤停入院、血肌酐、心肌标志物升高 | 年龄≥65岁（1分）、≥3个 CAD 危险因素（1分）、已知冠状动脉狭窄≥50%（1分）、ST 段改变≥0.5 mm（1分）、24小时内≥2次心绞痛发作（1分）、近7天使用阿司匹林（1分）、心肌标志物升高（CK-MB 或肌钙蛋白）（1分） | 年龄 65–74岁/≥75岁（2–3分）、Killip II–IV级（2分）、收缩压 <100 mmHg（3分）、心率 >100 bpm（2分）、前壁 ST 段抬高或 LBBB（1分）、糖尿病/高血压/心绞痛（1分）、体重 <67 kg（1分）、症状至治疗时间 >4 h（1分） |

> **GRACE 评分表现**：在 STEMI 和中危 NSTE-ACS 患者中，GRACE 评分预测死亡或 MI 优于主观临床判断。

**其他风险标记物：**
- 心衰证据（肺淤血等）是 Killip 分级的重要组成部分，也是 GRACE 和 TIMI-STEMI 评分的核心变量
- 较高的 cTn 水平与死亡和 MACE 风险增加相关
- 利钠肽（BNP、NT-proBNP）可提供额外的预后信息，识别死亡、心衰和复发性 MACE 高危患者

**心源性休克早期识别：**
- AMI 相关心源性休克占 ACS 人群的 **7%–10%**，死亡率极高
- 早期快速血运重建与生存率改善相关
- 应优先转运至 PCI 中心（理想情况下转运至提供高级机械循环支持（MCS）的中心）

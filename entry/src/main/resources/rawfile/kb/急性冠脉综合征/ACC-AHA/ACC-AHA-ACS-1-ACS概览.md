## 2. ACS 概览

### 2.1 定义与分类

ACS 由动脉粥样硬化斑块不稳定（破裂或侵蚀）引起，伴随部分或完全冠状动脉血栓形成和/或微栓塞，导致心肌血流减少和心肌缺血。

**ACS 三类临床类型（同一疾病谱系）：**

- **不稳定心绞痛（Unstable Angina）**：短暂心肌缺血，血流减少，但无明显心肌坏死（cTn 正常）
- **非ST段抬高型心肌梗死（NSTEMI）**：部分冠状动脉闭塞，心内膜下缺血；cTn 升高
- **ST段抬高型心肌梗死（STEMI）**：完全血管闭塞，透壁心肌缺血和梗死

**初始诊断依据**：临床病史 + ECG（Table 3）+ 心脏肌钙蛋白（cTn）

> 病理生理是动态的：患者可在病程中从一种临床类型快速进展为另一种。

**其他少见病因**：冠状动脉痉挛、冠状动脉栓塞、冠状动脉夹层。

### ECG 判读标准（Table 3）

| | 非ST段抬高型ACS（NSTE-ACS） | ST段抬高型心肌梗死（STEMI） |
|---|---|---|
| **缺血性 ECG 改变** | 新发或推定新发的水平或下斜型 ST 段压低 ≥0.5 mm（≥2个相邻导联）和/或 T 波倒置 >1 mm（≥2个相邻导联，R波为主或R/S>1），或短暂 ST 段抬高 | 新发或推定新发 ST 段抬高：≥2个解剖相邻导联，J点测量 ≥1 mm（V2-V3除外）；V2-V3：男性≥40岁 ≥2 mm，男性<40岁 ≥2.5 mm，女性不限年龄 ≥1.5 mm |
| **其他 ECG 改变** | 多数患者可为非特异性 ST/T 波改变或正常 ECG；ECG 正常不能排除 ACS | 应加做后壁导联（V7–V9）：疑似左回旋支闭塞、且 V1–V3 孤立 ST 段压低 ≥0.5 mm 时 |

> **ST 段改变的其他原因**：急性心包炎、左室肥厚、LBBB、Brugada 综合征、右室起搏、Takotsubo 综合征、早期复极——可干扰 STEMI 诊断。
>
> **新发 LBBB**：不應单独作为 AMI 诊断依据，需结合临床。

**本指南范围**：聚焦 Type 1 MI（动脉粥样硬化斑块破裂/侵蚀导致的急性事件）的急性管理。Type 2 MI、SCAD、MINOCA 的评估见专项文件。

### MI 通用定义——Type 分类（Table 4）

| Type | 描述 |
|------|------|
| **Type 1** | 急性冠状动脉粥样硬化血栓形成引起，通常由动脉粥样硬化斑块破裂或侵蚀引发，常伴有部分或完全血管血栓形成 |
| **Type 2** | 与心肌氧供需失衡相关，与急性冠状动脉粥样硬化血栓形成无关 |
| **Type 3** | 心源性死亡，有心肌缺血症状和推定缺血性 ECG 改变或室性心律失常，在获取心肌标志物样本前死亡或尸检发现 |
| **Type 4a** | PCI 围术期 MI，由操作并发症引起，PCI 后 ≤48 h 内检出 |
| **Type 4b** | PCI 后再发 MI，由冠状动脉支架或支架血栓形成引起 |
| **Type 4c** | PCI 后再发 MI，由冠状动脉支架再狭窄引起 |
| **Type 5** | CABG 围术期 MI，由操作并发症引起，CABG 后 ≤48 h 内检出 |

### 风险轨迹

- **最高风险期**：ACS 急性期（院前阶段及早期住院阶段）
- 适当管理后，风险开始降低
- **复发风险**：数月至数年持续升高（可能与 ACS 后急性炎症环境有关）
- ACS → 慢性冠状动脉疾病（CCD）的转换时间点尚不明确
- 出院时多项药物与 CCD 管理相似，应延长至 ACS 后 **>1 年**

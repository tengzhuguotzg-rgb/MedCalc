# 5岁以下儿童哮喘诊断

> [!important] 三条核心标准（Box 10-1/10-2）
> 三项标准**必须全部满足**才能确诊哮喘。仅满足 1-2 项 → 记录为"疑似哮喘"（suspected asthma）。

## 儿童喘息与哮喘

哮喘是儿童最常见的慢性疾病，也是导致缺课、急诊和住院的首要原因。最多一半的哮喘在儿童期起病。

> [!note] 喘息表型分类（历史概念，临床实用性有限）
> - **病毒诱发喘息**：典型与 URTI 相关，但 <12 月龄首次发作多为毛细支气管炎
> - **时间趋势分类**：一过性（<3岁止）、持续性（3岁前起>6岁止）、晚发性（>3岁起）
> - **症状分类**：仅发作性喘息 vs 多触发因素喘息
> 这些分类多为回顾性、预测价值差，不宜作为个体诊断依据。临床目标是在当前个体做出诊断。

## 诊断标准1：反复急性喘息发作

**满足以下任一**：

**选项A**：过去12个月至少 **2 次急性喘息发作**（每次持续 >24 小时）

**或选项B**：至少 1 次急性喘息发作 **+ 发作间期有哮喘样症状**（干咳、阵咳、睡眠中或哭笑运动后加重）

**喘息确认要求**：需由受训医护人员直接观察到呼吸道症状的不同，或由能明确区分喘息与其他呼吸音的家长可信报告。年幼儿的"呼吸嘈杂"可能来自上气道。

## 诊断标准2：排除其他诊断

基于详细病史和体检，除并发的病毒性呼吸道感染外，无其他可能解释症状的疾病（**Box 10-4**）。

### 关键鉴别诊断

| 症状/体征 | 需考虑疾病 |
|---------|-----------|
| 咳嗽+流涕<10天，无喘息 | 上呼吸道病毒感染 |
| 喂养时咳嗽、反复胸部感染 | GERD ± 咽期吞咽困难 |
| 突发症状、单侧喘息 | 异物吸入/结核 |
| 阵发性痉挛性咳嗽+鸡鸣样回声+呕吐 | 百日咳 |
| 持续湿咳 | 迁延性细菌性支气管炎/结核 |
| 哭闹/进食时嘈杂呼吸、粗厉咳嗽 | 气管软化/喉软化 |
| 心脏杂音、生长迟缓 | 先天性心脏病 |
| 早产史、出生即有症状 | 支气管肺发育不良 |
| 大量咳嗽+黏液、胃肠症状、生长发育迟缓 | 囊性纤维化 |
| 持续性响亮呼吸、喂养困难 | 血管环 |

> [!tip] 年龄越小，替代诊断可能性越大
> <12 月龄首次喘息发作通常为毛细支气管炎而非哮喘。
> <12 月龄反复喘息（≥2次）建议转诊儿科专科。

## 诊断标准3：治疗反应评估

**满足以下任一途径**：

1. **急性期住院/就诊时**：吸入 SABA 后 20-60 分钟内症状体征改善；中重度加重需 OCS 者在 3-4 小时内改善

2. **居家抢救**：家长报告 SABA 后 20-60 分钟内改善

3. **ICS 试验性治疗**：每日 ICS（如 FP 100-250 mcg/日）+ 按需 SABA 2-3 个月后，急性发作频率/严重度减少

> [!important] ICS 试验剂量
> 临床试验中证实有效的 ICS 剂量中位数为 FP 200 mcg/日。一旦确认诊断，尽快降至最低有效剂量。

**无反应的处理**：检查吸入技术和依从性 → 如技术正确+依从性好 → 考虑替代诊断或转诊。

## 辅助诊断工具

| 项目 | 价值 |
|------|------|
| **过敏检测**（皮试/sIgE） | ≥3岁儿童多数哮喘患儿致敏。但阴性不排除哮喘 |
| **胸片** | 不常规，但怀疑结构异常/异物/TB时有用 |
| **肺功能（肺功能仪、振荡肺功能）** | 多数 ≤5 岁儿童无法配合常规肺功能，但 5-6 岁可在指导下完成。**振荡肺功能**（oscillometry）从 3 岁起可用，评估支气管舒张反应性 |
| **FeNO** | 离线潮气法可测，目前仍为研究工具。反复咳嗽喘息且 FeNO 高的学龄前儿童，后续被诊断为哮喘的几率增加 |

## 疑似哮喘

不符合全部三项标准但高度怀疑 → 诊断"疑似哮喘"，继续随访、记录治疗反应和随时间变化。

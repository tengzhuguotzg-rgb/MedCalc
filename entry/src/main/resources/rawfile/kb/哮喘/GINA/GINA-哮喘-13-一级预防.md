# 哮喘一级预防

> [!note] 关键窗口期
> 哮喘的发生和持续由基因-环境相互作用驱动，**宫内和早期生活**被认为是预防干预的"机会窗口"。但目前干预研究有限，缺乏能改变哮喘病程的确切策略。

## 儿童预防建议

**Box 13-1** 核心建议：

- **孕期和出生后避免环境烟草暴露**
- **鼓励阴道分娩**（剖宫产与儿童哮喘患病率更高相关）
- **生后第一年尽量避免使用广谱抗生素**
- 母乳喂养的益处主要是其他健康获益，**不建议以预防过敏/哮喘为理由鼓励母乳喂养**
- 纠正维生素D不足的哮喘孕妇可能降低早期喘息风险

## 单一过敏原回避无效

针对单一过敏原的回避策略**未能降低哮喘发病率**（A级）。多策略干预（如 Isle of Wight 研究、加拿大哮喘初级预防研究、儿童哮喘预防研究）在 <5 岁儿童中观察到保护效应，但关键有效成分尚未确定。

**屋尘螨/宠物暴露与哮喘的关系复杂**，一些研究显示暴露增加致敏和喘息风险，另一些显示减少风险。出生队列数据显示家中宠物与儿童哮喘患病率无显著关联。**目前数据不足以推荐儿童早期减少或增加常见致敏原暴露来预防哮喘。**

## 环境因素

| 因素 | 证据与推荐 |
|------|-----------|
| **孕期吸烟** | 最强的产前暴露风险。产后吸烟影响年长儿童。Meta分析证实 |
| **空气污染** | 居住近主干道增加哮喘风险。全球每年约 400 万儿童新发哮喘病例可归因于交通相关空气污染（13%全球发病率） |
| **微生物/卫生假说** | 农场儿童（接触牲畜、生牛奶）哮喘发生率更低。卧室高LPS内毒素也与低过敏率相关 |
| **RSV感染** | 早期 RSV 住院与 25 岁前哮喘诊断率增加 3 倍相关。RSV 疫苗和单克隆抗体的长期哮喘预防效果仍不确定 |
| **剖宫产** | 与更高的哮喘患病率相关，可能通过影响婴儿肠道微生物群 |
| **肥胖** | 母体肥胖（每增加 1 kg/m² BMI，儿童哮喘几率增加 2-3%）和儿童超重/肥胖（尤其女孩）均为危险因素 |
| **早产/低出生体重** | 增加婴儿期和儿童期喘息障碍风险 |

## 饮食因素

- **母体饮食**：目前证据未明确显示孕期摄入特定食物增加哮喘风险。实际上花生和牛奶摄入与后代过敏/哮喘减少相关
- **维生素D**：两项 RCT 数据合并显示，孕期补充维生素D降低 0-3 岁哮喘/反复喘息风险 **25%**。6 岁时效果消失
- **鱼油**：一项高危儿童研究中，孕晚期大剂量鱼油降低学龄前喘息/哮喘，但最佳方案未确定
- **益生菌**：证据不足以推荐用于过敏性疾病预防
- **对乙酰氨基酚/叶酸**：孕期高频使用与儿童哮喘风险略有增加相关，但临床获益远大于风险

## 成人预防

### 职业暴露（最重要的可预防因素）
- **5-20% 成人新发哮喘可归因于职业暴露**
- 所有成人起病哮喘必须系统询问职业史和暴露史
- 早期识别和消除职业致敏物、脱离暴露（A级）
- 因经济和法律影响，尽快转诊专家（A级）

### 其他可变的成年起病风险因素
- 吸烟/电子烟
- 室内污染（二手烟、家用清洁剂）
- 交通相关空气污染
- 超重/肥胖

# 哮喘管理原则

> [!important] 长期目标
> GINA 哮喘管理的长期目标是：为每位患者实现尽可能最佳的长期结局——**长期症状控制 + 长期风险最小化**。

## 患者-医护合作

有效的哮喘管理需要建立患者（或其照护者）与医护之间的**伙伴关系**，使患者获得知识、信心和技能，承担自我管理的主要角色。

自我管理教育可减少成人（A级证据）和儿童（A级证据）的哮喘发病率。共享决策（shared decision-making）与改善结局相关。

### 沟通策略（Box 3-1）

**促进良好沟通的关键策略**：
- 友善幽默、关注患者
- 让患者表达目标、信念和担忧
- 共情、安慰、及时处理concerns
- 提供个性化信息

**降低低健康素养影响的策略**：
- 语速慢、用词简单，避免医学术语
- 用具体数字而非百分比
- 使用"回教法"确认理解（请患者复述）
- 请第二名医护人员或家属重复核心信息

### 数字健康

远程监测和管理手段（telemedicine、远程医疗）在 COVID-19 疫情期间加速发展，但需要更多高质量研究评估其在真实临床实践中的效用、有效性和可扩展性。

## 长期管理目标（Box 3-2）

**临床视角的长期目标**：
- **症状控制**：无/极少哮喘症状、无因哮喘睡眠障碍、不受限的体力活动
- **风险最小化**：无急性加重、改善或稳定个人最佳肺功能、无需维持性全身激素、无药物副作用

> [!warning] 症状控制 ≠ 总风险控制
> 仅评估症状控制是不够的。症状少或无症状的患者仍可能发生严重甚至致死性加重（触发因素：病毒感染、过敏原暴露、污染等）。**必须同时评估危险因素**。

同时要了解患者本人的哮喘治疗目标——可能与医疗目标不同。与患者讨论最佳结局时，需考虑其目标、表型、合并症、危险因素、可用性和成本以及副作用。

## 哮喘缓解（Remission）概念

> [!note] 什么是哮喘缓解？
> 缓解分为**临床缓解**（特定时间内无症状/加重）和**完全缓解**（同时肺功能、AHR、炎症标志物正常化）。可发生在治疗终止后或治疗过程中。

**儿童缓解**：
- 报告率差异大（一项研究：59% 学龄前喘息儿童6岁时无喘息；另一项：9岁后持续喘息的儿童中仅 15% 在 26 岁时无喘息）
- **缓解不是治愈**：儿童期/青少年期缓解后，哮喘常在后期复发。缓解的儿童成年后肺功能下降加速风险增加
- 目前无证据表明儿童期干预可增加缓解概率

**成人缓解**：
- 成人起病哮喘中约 16% 在 5 年内出现临床缓解（无症状且不用药）
- 儿童起病哮喘的成人缓解率高于成人起病
- 重症哮喘生物制剂治疗后的治疗中缓解越来越受关注

> [!tip] 与患者沟通注意事项
> 使用"缓解"一词需谨慎——患者可能理解为"治愈"，或联想到癌症/白血病。建议说"哮喘暂时安静了"（gone quiet for now），并强调不意味着永久治愈，不要自行停药。

## 个性化控制管理

哮喘控制的两个维度（症状控制 + 风险降低）驱动着治疗决策的**持续循环**（**Box 3-3**）：

```
评估 → 调整 → 复核 →（循环）
```

1. **评估**：症状控制 + 风险因素（重点：吸入技术和依从性）+ 合并症 + 患者目标与偏好
2. **调整**：治疗可调控危险因素 + 合并症 + 非药物策略 + 教育培训 + 药物调整（成人/青少年首选 Track 1 ICS-福莫特罗方案）
3. **复核**：对照治疗目标、复核影响因素、考虑 T2 生物标志物（若仍失控）、进一步检查

### 群体级 vs 患者级决策（Box 3-4）

**群体级（指南/医保/处方集层级）**：
- 每步"首选"方案基于 RCT、真实世界研究和观察性数据的疗效、效果、安全性和可及性
- Track 1 是大多数患者首选——整体加重风险更低、OCS需求更少、症状控制相似、方案更简便

**患者级（个体化决策）**：
- 考虑患者特征/表型预测差异风险和治疗反应
- 合并症评估
- 患者目标、偏好和实际问题（成本、吸入器可用性、依从性）

### 药物副作用最小化

- **首选 Track 1**：因所需 ICS 总剂量更低
- **确保正确的吸入技术**，包括使用储雾罐（pMDI）
- **吸药后漱口吐掉**
- 良好控制维持 3 个月后 → **寻找每位患者的最低有效 ICS 剂量**
- 检查 CYP450 药物相互作用

### OCS 管理策略（OCS Stewardship）

> [!important] OCS 使用原则
> 优化吸入治疗（尤其是转用 Track 1 抗炎缓解方案）可大幅减少 OCS 需求。即使 4-5 次 OCS 疗程即有累积性长期不良反应（糖尿病、骨质疏松、白内障、青光眼、心衰等）。

- 仅在明确指征时用：中/重度/危及生命的急性哮喘
- 最短有效剂量和最短疗程
- >2 周的 OCS 才需逐渐减停
- 任何需 OCS 的加重应提示全面审查并预防下一次

## 可调控危险因素处理

**Box 3-5** 汇总了可调控危险因素的策略：

| 危险因素 | 策略 |
|---------|------|
| **任何加重危险因素**（含症状未控） | 确保 ICS 处方（A）；转用 AIR 方案（A）；提供书面行动计划（A）；更频繁复诊（A）；频繁检查技术和依从性（A） |
| **过去1年 ≥1次严重加重** | 转用 AIR 方案（A）；无调控因素时考虑升级（A）；转诊考虑生物制剂（C） |
| **吸烟/电子烟** | 鼓励戒烟（A）；控制差时考虑更高 ICS 剂量（B） |
| **低 FEV1（尤其 <60%）** | 解决依从性和技术（A）；考虑高剂量 ICS 试验3月（B）；排除其他肺病（D） |
| **肥胖** | 减重策略（B）；区分哮喘症状与去条件化/机械限制/OSA 症状（D） |
| **重大心理问题** | 转诊心理评估（D）；帮助区分焦虑与哮喘症状（D） |
| **社会经济问题** | 选择最具成本效益的 ICS 方案（D）；优化吸入技术（D） |
| **确诊食物过敏** | 适当食物回避 + 过敏行动计划 + 肾上腺素（A）；转诊专家（D） |
| **职业/生活暴露** | 尽快脱离暴露（A）；尽快转诊（D） |
| **过敏原暴露（致敏）** | 简单回避策略（C）；升级治疗（D）；考虑 HDM-SLIT（A） |

## 非药物策略

**Box 3-6** 汇总各非药物干预措施的证据等级和推荐意见：

### 戒烟与避免二手烟和电子烟
- 每次就诊强烈鼓励吸烟/电子烟者戒烟，提供咨询和戒烟计划（A）
- 建议患儿家长不在儿童使用的房间或车内吸烟/电子烟（A）
- 评估 >10 包·年吸烟者为 COPD 或哮喘+COPD（D）

### 体力活动
- 鼓励规律体力活动（A）——对一般健康有益
- 运动诱发性支气管收缩可通过维持 ICS（A）或运动前 SABA（A）/低剂量 ICS-福莫特罗（B）/热身（A）预防
- 儿童游泳训练可增加肺功能和心肺耐力（B）
- **肺康复计划**：对成人哮喘的功能性运动能力和生活质量有临床意义的短期改善（A）

### 避免职业/生活暴露
- 所有成人起病哮喘者系统询问职业史和其他吸入暴露（D）
- 职业性哮喘：尽早识别和消除致敏物，脱离暴露（A）；尽快转诊（A）

### 避免可加重哮喘的药物
- 处方 NSAID 前始终询问哮喘史（A）
- 哮喘不是心脏选择性 β 受体阻滞剂治疗急性冠脉事件的绝对禁忌症（D）

### 室内过敏原
- **过敏原回避不作为哮喘的常规策略推荐**（A）
- 大多数单一干预未能充分降低过敏原负荷以改善临床结局（A）
- 家中除湿或霉菌修复可减少成人哮喘症状和用药（A）

### 健康饮食
- 鼓励多吃水果蔬菜——对一般健康有益（A）。增加果蔬摄入可改善控制、降低加重风险

### 肥胖患者减重
- 纳入治疗计划（B）。减重 5-10%（限食±运动）可改善控制和 QOL。减重手术效果最显著（A）
- GLP-1RA（如用于糖尿病）可能降低加重率（前瞻性研究进行中）

### 呼吸练习
- 可作为药物治疗的补充，改善症状和 QOL，但不降低加重风险，对肺功能无一致效果（A）

### 室内空气污染
- 鼓励使用非污染供暖/烹饪源，污染源排至室外（B）

### 情绪压力
- 鼓励患者识别应对情绪压力的目标和策略（D）
- 焦虑/抑郁症状者安排心理评估（D）

### 户外过敏原/空气污染/食物回避
- 良好控制时无需改变生活方式回避不利条件
- 不利环境条件（严寒/高污染）时避免剧烈户外活动并留在空调环境（D）
- 食物回避必须有明确诊断（经口服激发证实）（D）
- 确诊食物过敏者回避可减少哮喘加重（D）

## 转诊指征（Box 3-8）

需转诊专家的情况（存在其中任意一项）：

| 类别 | 具体指征 |
|------|---------|
| **诊断困难** | 慢性感染症状/提示心源或非肺源性病因；治疗试验后诊断仍不明；哮喘+COPD 优先治疗顺序不确定 |
| **怀疑职业性哮喘** | 确认测试 + 识别致敏物/刺激物 |
| **持续性或重度失控** | Step 4 治疗仍持续症状/加重/低肺功能；频繁使用急诊/紧急医疗 |
| **哮喘相关死亡危险因素** | 曾因哮喘入 ICU/机械通气；确诊食物过敏/过敏反应 |
| **显著治疗副作用** | 长期 OCS 使用；频繁 OCS（≥2次/年） |
| **特殊亚型** | AERD、ABPA 等 |
| **儿童6-11岁附加指征** | 中剂量 ICS 仍失控；怀疑生长延迟等副作用 |

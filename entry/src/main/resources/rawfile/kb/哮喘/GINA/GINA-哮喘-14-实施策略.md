# 哮喘管理策略的实施

## 引言

GINA 年度报告不是正式的临床实践指南，而是每年更新的循证策略。必须在各国、各地区和医疗机构层面进行本地化实施。

## 实施步骤（Box 14-2）

1. 组建多学科工作组（医生、专科护士、药师、患者、社区成员）
2. 评估当前哮喘诊疗现状（加重、住院、死亡、诊疗缺口和需求）
3. 选择实施内容，确定主要目标，根据本地环境调整
4. 识别实施障碍和促进因素
5. 选择实施框架和策略
6. 制定逐步实施计划（目标人群、可评估结局、数据编码、本地资源、时间线、任务分配、结局评估）
7. 持续审查进展和结果，必要时调整策略

## 实施障碍（Box 14-3）

| 医护方 | 患者方 |
|-------|--------|
| 对推荐了解不足 | 低健康素养 |
| 不认同推荐或认为无效 | 对哮喘及其管理理解不足 |
| 抵制改变 | 不认同推荐 |
| 外部障碍（组织/政策/资金） | 文化/经济障碍 |
| 时间和资源不足 | 同辈影响 |
| 法律问题 | 错误认知 |

## 高影响力干预（Box 14-4）

**已有实证证据支持的干预措施**：
- 近期住院/重症哮喘患者**免费提供 ICS**
- 早期 ICS 治疗 + 自我管理教育 + 减少烟草暴露 + 改善哮喘教育可及性
- 基层医疗清单式提醒工具（提示评估控制水平和治疗策略）
- 书面行动计划作为自我管理教育的一部分
- 儿科哮喘急性和慢性管理循证流程模式（多医院实施）

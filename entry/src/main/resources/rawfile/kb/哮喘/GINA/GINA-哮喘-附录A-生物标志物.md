# T2 生物标志物

临床应用中最有价值的 T2 气道炎症生物标志物：**血嗜酸粒细胞计数（BEC）**、**FeNO**、**血清总 IgE** 和 **过敏原特异性 IgE**。

## 关键阈值

| 标志物 | 诊断中"高"的标准 | 重症哮喘 T2 炎症判断 |
|-------|----------------|--------------------|
| **血EOS** | ≥国家/地区实验室正常上限 | 高剂量 ICS 下 ≥150/μL 提示 T2 炎症；≥300/μL 为生物制剂准入常见阈值 |
| **FeNO** | ICS初治 >50 ppb；中等剂量 ICS ≥25 ppb | 依从性好但持续高水平需警惕类固醇难治性 |
| **总IgE** | 正常值因年龄和地区而异 | 不能单独用于预测生物制剂反应 |
| **过敏原-sIgE** | 阳性 + 临床病史一致性 | 支持过敏性哮喘 |

## 影响因素

| 因素 | BEC | FeNO |
|------|-----|------|
| **男性 > 女性** | 升高 | 升高 |
| **日间变异** | 清晨高、下午低 | 下午高、清晨低 |
| **吸烟** | 升高 | 降低 |
| **ICS/OCS** | 降低 | ICS 剂量依赖性降低 |
| **过敏原暴露**（致敏者） | 升高 | 约 24h 后升高 |
| **寄生虫感染** | 升高 | — |
| **非哮喘条件** | EGPA、嗜酸粒细胞性支气管炎等 | 特应性皮炎、鼻炎等 |

> [!tip] 临床解读要点
> - 有大量变异因素，**需谨慎解读单一阈值**
> - 重症哮喘评估生物制剂资格时，如初次未达标准，应重复至少 **3 次**
> - BEC 和 FeNO 提供**互补**的预后和治疗预测信息——两者均高时未来加重风险最高

## 临床效用

| 场景 | BEC | FeNO |
|------|-----|------|
| **初始诊断** | 典型症状时高BEC支持T2哮喘 | 高FeNO支持T2诊断，低不排除 |
| **预后（加重风险）** | 独立预测未来加重（尤其中高ICS者） | 独立预测未来加重 |
| **预测生物制剂反应** | 高BEC预测抗IL-5、奥马珠单抗等反应更好 | 高FeNO预测杜普利尤单抗、奥马珠单抗、特泽普单抗反应更好；与抗IL-5疗效无关 |
| **MART疗效（轻-中度）** | 基线水平不影响疗效 | 基线水平不影响疗效 |
| **低BEC+低FeNO** | 两者均低 → 考虑非药物策略后再升级ICS | |

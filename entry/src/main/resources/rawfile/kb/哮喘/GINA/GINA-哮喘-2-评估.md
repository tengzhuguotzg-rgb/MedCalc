# 哮喘评估

> [!important] 核心概念：哮喘控制（Asthma Control）的双重维度
> 哮喘控制评估必须同时涵盖两个独立维度：**症状控制**（过去4周）和**未来不良结局风险**（加重风险、肺功能下降、药物副作用）。两者可能不一致——症状控制良好的患者仍可能发生严重急性加重。

## 评估总览

**Box 2-1** 概述哮喘评估的三要素：

1. **评估哮喘控制** = 症状控制 + 未来风险
   - 症状控制：过去4周内哮喘症状频率（日间/夜间/活动受限/SABA使用）
   - 风险因素：加重危险因素、持续气流受限风险、药物副作用风险
   - 肺功能：诊断/初始治疗时测定，ICS治疗3-6月后记录个人最佳值，此后定期复查（至少每1-2年1次，高风险患者更频繁）

2. **评估治疗相关问题**
   - 确认当前治疗阶梯
   - 观察吸入技术、评估依从性和副作用
   - 确认患者有无书面行动计划
   - 询问患者对哮喘的态度、目标和偏好

3. **评估多病共存**
   - 鼻炎/鼻窦炎、GERD、肥胖、OSA、抑郁和焦虑均可能贡献症状和降低生活质量

## 症状控制评估

**Box 2-2A** GINA 症状控制分级（过去4周）：

| 评估项目 | 是/否 |
|---------|-------|
| 日间哮喘症状 >2次/周？ | 是 □ 否 □ |
| 因哮喘夜间憋醒？ | 是 □ 否 □ |
| SABA缓解剂使用 >2次/周？（不包括运动前预防性使用） | 是 □ 否 □ |
| 因哮喘活动受限？ | 是 □ 否 □ |

- 0项为是 → **良好控制**
- 1-2项为是 → **部分控制**
- 3-4项为是 → **未控制**

> [!warning] AIR缓解剂 vs SABA缓解剂的区别
> 对于使用 AIR（抗炎缓解剂，如按需 ICS-福莫特罗）的患者，上述"缓解剂使用频率"标准**不适用**——因为 AIR 使用同时提供了额外 ICS 治疗。应根据过去4周平均使用频率综合判断。

**症状控制评估工具**：
- **ACQ-5**（5项症状问卷）：≤0.75 良好控制，≥1.5 未控制。GINA 推荐使用 ACQ-5 而非 ACQ-6/7
- **ACT**（哮喘控制测试）：5-25分，20-25 良好控制，16-19 未良好控制，5-15 非常差控制。最小临床重要差异 3 分
- **CAAT**（慢性气道评估测试）：8项通用健康状态工具

**儿童 6-11 岁评估**（**Box 2-3**）：
- 覆盖症状、活动水平、危险因素、肺功能、副作用（至少每年测身高）、吸入技术、依从性、合并症
- **c-ACT**：有独立家长版和儿童版
- 儿童为避免体育活动可能表现得控制良好，需主动询问活动受限情况

> [!tip] 儿童评估技巧
> 确保在不被家长打断的情况下倾听儿童本人的描述。儿童仅能回忆最近几天的症状，而家长的回忆跨度更长。应结合双方信息判断。

## 未来风险评估

**Box 2-2B** 列出三类未来风险危险因素：

### 加重风险因素（独立于症状控制水平）
| 类别 | 具体因素 |
|------|---------|
| **症状** | 未控制的哮喘症状 |
| **药物** | SABA过度使用（≥3罐/年增风险，≥1罐/月与死亡相关）、未用ICS/依从性差/技术错误 |
| **合并症** | 肥胖、慢性鼻窦炎、GERD、确诊食物过敏、妊娠 |
| **暴露** | 吸烟、电子烟、过敏原（致敏）、空气污染 |
| **心理社会** | 重大心理/社会经济问题 |
| **肺功能** | 低 FEV1（尤其 <60%）、高支气管舒张反应性 |
| **炎症标志物** | 血嗜酸粒细胞升高、FeNO升高 |
| **加重史** | 曾因哮喘插管/入ICU、过去1年 ≥1次严重加重 |

### 持续气流受限风险因素
- 早产、低出生体重、婴儿期体重增长过快、频繁咳痰
- 严重加重史中未用ICS
- 吸烟/化学物暴露（职业/生活）
- 初始 FEV1 低、痰或血嗜酸粒细胞增多

### 药物副作用风险因素
- **全身性**：频繁 OCS、长期/高剂量/强效 ICS、CYP450 抑制剂合用
- **局部**：高剂量/强效 ICS、错误吸入技术

## 严重度评估

> [!important] 严重度定义基于"治疗的困难程度"
> 哮喘严重度是**回顾性**评估：至少 2-3 个月治疗后，根据控制症状和加重所需的治疗强度判断。该定义主要用于识别可从生物制剂等附加治疗中获益的患者。

- **重症哮喘**：优化高剂量 ICS-LABA 仍失控，或需高剂量 ICS-LABA 防止失控
- **中度哮喘**：3-4阶梯治疗（中低剂量 ICS-LABA）良好控制
- **轻度哮喘**：低强度治疗（按需低剂量 ICS-福莫特罗或低剂量 ICS+按需 SABA）

> [!warning] "轻度哮喘"的警示
> GINA 建议临床实践中**尽量避免使用"轻度哮喘"**一词，因为患者和医生常误认为"轻度"即"低风险"，无需 ICS。然而高达 **30% 的哮喘加重和死亡发生在间歇症状（<每周1次）患者中**。该标签是回顾性的，不能用于决定起始治疗。

## 肺功能在评估中的作用

> [!tip] 确诊后肺功能的主要作用是**评估未来风险**
> FEV1 <60% 预计值是独立的加重风险预测因子，但与症状相关性不一定强。

- 诊断/治疗开始时、ICS 治疗 3-6 月后（确定个人最佳值）、定期（至少 1-2 年 1 次，高风险者更频繁）测定
- 症状频繁而 FEV1 正常 → 排查非哮喘原因
- 症状少但 FEV1 低 → 气流受限感知受损可能
- 持续性支气管舒张反应性 → 提示未控制

## 与重症哮喘的区分

**Box 2-4** 基层医疗可筛查的常见可调控因素——诊断重症哮喘前必须排除：
- **吸入技术错误**（社区患者中高达 80%）
- **依从性差**
- **诊断错误**（症状实际来自 ILO、心衰、体能差等）
- **多病共存**（鼻窦炎、GERD、肥胖、OSA）
- **持续接触致敏物或刺激物**

> [!note]
> 区分**难治性哮喘**和**真性重症哮喘**至关重要。详见[[GINA/GINA-哮喘-8-难治性与重症哮喘]]。

# 线粒体功能障碍

## 核心断言

**线粒体功能障碍是脓毒症器官功能衰竭的"终末机制"之一，表现为 ATP 合成下降、线粒体氧化应激损伤和代谢重编程。**

在脓毒症中，线粒体面临三重打击：① 微循环障碍致氧输送减少；② 直接的氧化应激（ROS↑）；③ mPTP（线粒体通透性转换孔）异常开放。这导致 ATP 耗竭、细胞代谢"僵败"，即使组织氧供暂时恢复，线粒体功能也可能无法恢复。

---

## 关键病理改变

| 改变 | 机制 |
|------|------|
| **mPTP 异常开放** | 钙超载 + ROS → 线粒体膜电位崩溃 → ATP 合成骤停 |
| **氧化磷酸化解耦联** | 质子漏 → ATP 合成效率↓ |
| **线粒体钙稳态破坏** | 胞质 Ca²⁺ 超载 → mPTP 开放 |
| **线粒体 DNA（mtDNA）损伤** | ROS 攻击 mtDNA → 呼吸链复合物功能↓ |
| **线粒体自噬受损** | 自噬体-溶酶体融合障碍 → 损伤线粒体累积 |
| **代谢重编程** | 糖酵解↑ + 氧化磷酸化↓（Warburg 效应样改变） |

---

## 关键分子

| 分子 | 作用 |
|------|------|
| **mPTP**（线粒体通透性转换孔） | 异常开放 → 线粒体肿胀 → 能量崩溃 |
| **mtDNA** | 损伤后释放入血 → 激活 NLRP3 → 炎症加重 |
| **NADPH** | 抗氧化系统底物，耗竭后 ROS 清除能力↓ |
| **PINK1/Parkin** | 线粒体自噬通路，脓毒症中受损 |

---

## 临床决策点

> [!warning] 临床意义
> 线粒体功能障碍是脓毒性心肌病（SIMD）的核心机制，表现为心脏舒张功能障碍和收缩力下降。
> 线粒体损伤也参与脓毒性脑病（SAE）和 AKI 的发生。

> [!tip] 通路记忆
> 感染/炎症 → 线粒体氧化应激↑ + mPTP 开放 → mtDNA 损伤/mtDNA 释放 → NLRP3 激活 → 炎症加重 → 细胞能量崩溃 → 器官功能障碍

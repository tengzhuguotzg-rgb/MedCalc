# 微循环功能障碍

## 核心断言

**微循环功能障碍是脓毒症器官灌注不足的核心环节，表现为毛细血管密度下降、血流分布异常和氧摄取障碍。**

尽管全身血流动力学参数（CO、MAP）可能正常，微循环层面的变化（毛细血管密度↓↓、血流间歇流、红细胞变形性↓）已导致组织氧供需失衡。这一机制解释了为何部分患者对液体复苏和血管活性药反应不佳——全身参数改善不等于微循环改善。

---

## 微循环功能障碍的核心改变

| 改变 | 机制 |
|------|------|
| **毛细血管密度下降** | 微血栓形成 + 内皮水肿 + 血管内皮细胞凋亡 |
| **血流分布异常** | 毛细血管血流间歇流（rolling停滞） |
| **红细胞变形性↓** | 氧化应激损伤红细胞膜 |
| **氧摄取率↓** | 线粒体功能障碍 + 微血管分流 |
| **内皮糖萼降解** | 炎症因子/胰蛋白酶/透明质酸酶降解糖萼 |

---

## 微循环与全身循环的脱节

> [!warning] 关键概念
> **全身血流动力学正常 ≠ 微循环灌注正常**
> 脓毒症中微循环与全身循环"脱节"（loss of microvascular reactivity），
> 是血管麻痹和内皮损伤在微血管层面的表现。

- 全身血管阻力下降（血管麻痹）
- 微血管自身调节能力丧失
- 内皮屏障功能障碍 → 液体渗漏至组织间隙

---

## 关键分子与通路

| 分子 | 作用 |
|------|------|
| **VEGF** | 促进血管新生，但功能异常导致血管渗漏 |
| **Ang-2**（血管生成素-2） | 竞争性抑制 Ang-1/Tie2 → 内皮屏障破坏 |
| **透明质酸酶** | 降解细胞外基质和糖萼 |
| **VE-cadherin** | 内皮细胞间连接蛋白，被组蛋白酶降解 |
| **ICAM-1/VCAM-1** | 上调 → 白细胞黏附 → 毛细血管阻塞 |

---

## 临床评估方法

| 方法 | 内容 |
|------|------|
| **舌下微循环显像** | 侧流暗视野（SDF）/在体显微镜 → 可视化毛细血管密度和血流 |
| **GUM（胃黏膜内推 pCO₂）** | 胃黏膜 pCO₂↑（> 20 mmHg gap）→ 微循环低灌注 |
| **乳酸** | 乳酸 > 4 mmol/L 提示组织低灌注 |
| **ScvO₂**（中心静脉血氧饱和度） | < 70% 提示氧供需失衡 |

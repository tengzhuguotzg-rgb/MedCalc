# 脑功能障碍

## 核心断言

**脓毒性脑病（Sepsis-Associated Encephalopathy，SAE）是最常见的脓毒症器官损伤之一，表现为意识改变、谵妄和认知障碍，机制涉及血脑屏障（BBB）破坏、神经炎症和脑代谢异常。**

SAE 在脓毒症中发生率高达 30–70%，可出现在脓毒症病程早期（不依赖血流动力学改变），病死率显著增加。

---

## 核心机制

| 机制 | 说明 |
|------|------|
| **血脑屏障破坏** | 炎症因子（TNF-α、IL-1β）→ 紧密连接蛋白（Claudin-5）降解 → BBB 通透性↑ |
| **神经炎症** | 星形胶质细胞和小胶质细胞被激活 → 神经毒性介质释放 |
| **脑代谢异常** | 线粒体功能障碍 → 神经元 ATP 合成↓ |
| **微循环障碍** | 脑灌注↓ + 微血栓 |
| **色氨酸代谢异常** | 犬尿氨酸途径激活 → 兴奋性毒素积累 |
| **神经递质失衡** | 乙酰胆碱↓、多巴胺↓、谷氨酸↑ |

---

## 临床特点

| 表现 | 说明 |
|------|------|
| **谵妄（Delirium）** | 急性意识改变，注意力和意识波动障碍 |
| **昏迷** | 严重 SAE |
| **认知障碍** | ICU 获得性认知障碍（长期） |

**评估工具**：ICU 谵妄评估（CAM-ICU）、ICDSC

# 凝血-免疫交互

## 核心断言

**凝血系统与固有免疫在脓毒症中相互激活，形成"免疫血栓（Immunothrombosis）"，是 DIC 和多器官衰竭的核心病理基础。**

炎症激活外源性凝血途径（组织因子 TF 表达），同时中性粒细胞释放 NETs 激活内源性凝血途径（FXII），导致全身凝血级联反应失控。生理性抗凝系统（PC、PS、AT、TAMM）被消耗或抑制，纤溶亢进最终被 PAI-1 介导的纤溶抑制取代，形成微血管血栓 → 组织低灌注 → 多器官功能障碍。

---

## 免疫血栓的形成机制

```
炎症因子（TNF-α、IL-1β、IL-6）
    ↓
单核细胞/中性粒细胞表达组织因子（TF）
    ↓
外源性凝血途径激活 → 凝血酶生成 ↑↑
    ↓
纤维蛋白原 → 纤维蛋白
    ↓
中性粒细胞释放 NETs（含组蛋白 + 蛋白酶）
    ↓ 激活 FXII
内源性凝血途径激活 → 进一步放大凝血
    ↓
微血管血栓沉积 → 组织缺血/器官损伤
```

**三条通路汇聚微血栓：**
1. 外源性凝血（TF → FVIIa → FX → 凝血酶）
2. NETs 触发的内源性凝血（FXII 激活）
3. 血小板-中性粒细胞聚集（ platelet-NET complexes）

---

## 凝血病分期（SIC → DIC）

| 分期 | 标志物 | 临床意义 |
|------|--------|---------|
| **SIC**（脓毒症诱导凝血病） | PLT ↓、PT-INR ↑、SOFA ≥ 2 | 早期 DIC，ISTH 2019 标准 |
| **显性 DIC** | DIC 评分 ≥ 5 | 消耗性凝血病，发生率 30–50% |

**SIC 评分**（ISTH 2019）：
- PLT 100–150 × 10⁹/L：1分；< 100 × 10⁹/L：2分
- PT-INR 1.2–1.4：1分；> 1.4：2分
- SOFA 评分（呼吸/肝脏/心血管/肾脏）：≥ 2 分
- **诊断标准**：总分 ≥ 4 分

---

## 关键分子与通路

| 分子 | 作用 |
|------|------|
| **组织因子（TF）** | 外源性凝血途径启动因子，单核细胞/中性粒细胞在炎症下表达 |
| **NETs** | 含组蛋白 H4、弹性蛋白酶，激活 FXII，触发血小板聚集 |
| **凝血酶** | 催化纤维蛋白原 → 纤维蛋白；激活血小板；激活 PAR 受体 |
| **血栓调节蛋白（TM）** | 内皮表面，激活蛋白 C → 生理性抗凝；脓毒症时内皮损伤导致 TM↓ |
| **PAI-1**（纤溶酶原激活物抑制剂-1） | 由内皮细胞释放，抑制 tPA → 纤溶抑制 → 微血栓不易被清除 |
| **蛋白 C / 蛋白 S** | 生理性抗凝系统，脓毒症时大量消耗 |
| **抗凝血酶 III（AT-III）** | 主要抗凝物质，脓毒症时因消耗而↓ |

## 临床决策点

> [!warning] 临床阈值
> - D-dimer 和凝血酶原片段 F1+2 升高 → 提示凝血激活
> - 蛋白 C < 正常 50% → 预后不良
> - AT-III < 70% → 考虑补充
> - SIC 评分 ≥ 4 → 应启动抗凝评估

> [!tip] 治疗决策树
> SIC 早期 → 肝素/ thrombomodulin 考虑；显性 DIC → 替代治疗（FFP/冷沉淀）取决于出血/血栓倾向

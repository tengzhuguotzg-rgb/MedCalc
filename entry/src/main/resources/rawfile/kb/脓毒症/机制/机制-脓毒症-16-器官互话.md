# 器官互话

## 核心断言

**器官互话（Organ Cross-talk）是脓毒症多器官衰竭（MODS）的核心机制，各器官通过循环介质（细胞因子、VGF、BNP 等）、血流动力学和神经体液通路相互影响，形成恶性循环。**

单一器官损伤可经器官互话途径引发连锁反应，最典型的例子是心肾综合征（AKI → 心衰加重 → 肾脏灌注↓）。

---

## 典型器官互话通路

| 互话组合 | 机制 | 临床后果 |
|---------|------|---------|
| **心-肾** | AKI → 容量超负荷 → 心衰加重；心输出量↓ → 肾脏低灌注 | 心肾综合征 |
| **心-肺** | 左心衰 → 肺淤血 → 低氧 → 肺血管阻力↑ → 右心衰 | 心源性肺损伤 |
| **肾-脑** | AKI → 尿毒症毒素↑ → 脑病加重 | 尿毒症脑病 |
| **肠-肺** | 肠屏障破坏 → 内毒素入血 → 肺损伤加重 | 肠源性肺损伤 |
| **肝-肾** | 肝衰竭 → 胆红素↑ + 血管扩张物质↑ → AKI | 肝肾综合征 |

---

## 循环介质

| 介质 | 来源 | 作用 |
|------|------|------|
| **VEGF** | 内皮受损 | 血管通透性↑ |
| **BNP/NT-proBNP** | 心肌牵张 | 心功能评估 + 肾功能影响 |
| **IL-6** | 全身炎症 | 多器官损伤 |
| **TNF-α** | 巨噬细胞 | 心肌抑制 + 血管扩张 |
| **LPS** | 肠道 | 全身炎症 + 内皮损伤 |

# 肾脏损伤

## 核心断言

**脓毒症相关急性肾损伤（SA-AKI）发生率约 50%，是脓毒症患者预后不良的独立预测因素，机制涉及低灌注、炎症、肾内凝血和线粒体损伤。**

SA-AKI 的特殊之处：与经典缺血性 AKI 不同，脓毒症即使在血流动力学"稳定"状态下也可发生 AKI（炎症/代谢机制主导），且对液体负荷耐受性差。

---

## AKI 分期标准

### KDIGO 标准

| 分期 | 血清肌酐 | 尿量 |
|------|---------|------|
| **1期** | 升高×1.5–1.9 或 ≥ 0.3 mg/dL（48h内） | < 0.5 mL/kg/h 持续 6–12h |
| **2期** | 升高×2.0–2.9 | < 0.5 mL/kg/h 持续 ≥ 12h |
| **3期** | 升高×3.0 或 ≥ 4 mg/dL 或启动 RRT | < 0.3 mL/kg/h 持续 ≥ 24h 或无尿 ≥ 12h |

### RIFLE 标准
Risk → Injury → Failure → Loss → End-stage renal disease

---

## SA-AKI 的特殊机制

| 机制 | 说明 |
|------|------|
| **炎症损伤** | TNF-α/IL-1β 直接损伤肾小管上皮 |
| **微循环障碍** | 肾内血管收缩 + 微血栓 → 肾灌注↓ |
| **代谢重编程** | 线粒体功能障碍 → 肾小管上皮 ATP 耗竭 |
| **肾内凝血** | 炎症激活凝血 → 肾内微血栓 |

---

## 临床决策点

> [!warning] 液体管理
> - SA-AKI 对液体超负荷耐受性差，过多输液易导致液体过负荷（FO）与多器官衰竭加重相关
> - 早期目标导向液体复苏仍必要，但需监测液体反应性（PPV、PLR）

> [!tip] RRT 时机
> - 传统指标：严重酸中毒（pH < 7.2）、高钾（K⁺ > 6.5 mmol/L）、液体过负荷
> - 脓毒症：考虑早期启动 RRT 可改善预后（IDEAL-ICU 研究提示争议，需个体化）

# 免疫抑制

## 核心断言

**脓毒症相关免疫抑制综合征（PICS）表现为持续的炎症反应加免疫抑制状态，是脓毒症后期（>72h）继发感染的核心机制。**

PICS 的特征：持续炎症（C-反应蛋白↑、铁蛋白↑）+ 获得性免疫缺陷（CD4+ T 细胞减少、mHLA-DR 表达↓）+ 代谢异常（骨骼肌消耗）。免疫抑制状态下的患者对院内感染（革兰阴性菌、真菌、病毒再激活）高度易感。

---

## 免疫抑制的核心改变

| 改变 | 具体表现 |
|------|---------|
| **固有免疫耗竭** | 中性粒细胞吞噬功能↓、NK 细胞毒性↓、巨噬细胞失活 |
| **适应性免疫抑制** | CD4+ T 细胞减少、CD4+/CD8+ 比值倒置、Treg 增加 |
| **单核细胞失活** | mHLA-DR < 8000 抗体/细胞 |
| **T 细胞耗竭** | PD-1/CTLA-4 上调 |
| **继发感染风险↑** | 医院获得性感染（VAP、CRBSI、UTI、Candida） |

---

## PICS 的临床标志

| 标志物 | 临床意义 |
|--------|---------|
| **mHLA-DR < 8000 AB/cell** | 单核细胞失活，阈值以下需考虑免疫增强治疗 |
| **CD4+ T 细胞 < 400/μL** | 严重免疫缺陷 |
| **CD4+/CD8+ < 1.0** | 免疫倒置 |
| **PD-1 表达 > 90%** | T 细胞耗竭晚期标志 |

---

## 治疗策略

| 策略 | 用药 | 适应症 |
|------|------|--------|
| **免疫增强** | IFN-γ（100 μg/m²，每周3次） | mHLA-DR 持续低 |
| **G-CSF** | GM-CSF（250 μg/m²/d） | 中性粒细胞功能低下 |
| **免疫检查点抑制剂** | 研究中 | T 细胞耗竭 |
| **预防继发感染** | 感控Bundle | 高危患者 |

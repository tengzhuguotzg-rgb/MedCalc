# 炎症风暴

## 核心断言

**炎症风暴（SIRS）是脓毒症早期（0–72h）的核心病理生理事件。**

感染或组织损伤触发固有免疫细胞释放大量促炎细胞因子（ TNF-α > 500 pg/mL、IL-6 > 1000 pg/mL、IL-1β ），同时补体 C5a > 40 ng/mL 过度激活，形成全身性瀑布样炎症反应。晚期（4–14 天）免疫抑制占优，抗炎介质（IL-10、TGF-β）占据主导地位，导致 T 细胞耗竭和继发感染易感。

---

## 炎症风暴的分期模型

```
早期（0–72h）：超炎症相（SIRS）
├── 促炎细胞因子主导：TNF-α、IL-1β、IL-6、IL-8
├── 补体 C5a 过度激活
├── NETosis 释放
└── 内皮激活 → 血管通透性↑ → 组织水肿

晚期（4–14d）：免疫抑制相（CARS）
├── 抗炎介质主导：IL-10、TGF-β
├── T 细胞耗竭（PD-1 表达 >90%）
├── 单核细胞失活化（mHLA-DR <8000 抗体/细胞）
└── 继发感染风险↑
```

---

## 关键通路与分子

### 1. 经典炎症通路
| 通路 | 关键分子 | 效应 |
|------|---------|------|
| **NF-κB 通路** | TLR4/2 → MyD88 → IKK → NF-κB | TNF-α、IL-1β、IL-6 转录 |
| **NLRP3 炎症小体** | Caspase-1 激活 | IL-1β、IL-18 成熟释放 |
| **MAPK 通路** | p38/JNK/ERK | 细胞应激反应、细胞凋亡 |

### 2. 焦亡（Pyroptosis）通路
- **Caspases**：Caspase-1（经典通路）、Caspase-4/5/11（非经典通路）
- **GSDMD**（Gasdermin D）：被 Caspase 切割后形成膜孔 → 细胞膜破裂 → DAMP 释放
- **触发物**：PAMPs（细菌脂多糖 LPS） + DAMPs（ATP、mtDNA、HMGB1）

> [!warning] 临床阈值
> - TNF-α > 500 pg/mL、IL-6 > 1000 pg/mL 提示重度炎症风暴
> - C5a > 40 ng/mL 提示补体过度激活
> - 内毒素耐受：LPS 诱导 TNF-α < 200 pg/mL

## 与其他机制的交互

### 补体-炎症交互
- C3a/C5a 招募并激活中性粒细胞 → NETosis
- 凝血酶可独立激活 NLRP3 炎症小体 → 放大炎症

### 线粒体-炎症交互
- 线粒体氧化应激 → mtDNA 释放 → 激活 NLRP3
- 线粒体通透性转换孔（mPTP）开放 → 细胞死亡 → DAMP 释放

### NETosis-炎症交互
- NETosis 释放的组蛋白和蛋白酶本身是 DAMPs → 进一步激活炎症
- NET-血小板复合物 → 微血栓形成（见凝血-免疫章节）

## 临床决策点

| 场景 | 决策要点 |
|------|---------|
| **超炎症期识别** | TNF-α、IL-6、PCT 联合监测；CD64 上调区分感染性/非感染性 SIRS |
| **免疫抑制期识别** | mHLA-DR 流式（<15,000 AB/cell）；CD4+/CD8+ 比值（<1.0）；PD-1 表达 |
| **治疗窗口** | 早期（<72h）抗炎；晚期（>4d）免疫增强 |
| **TNF-α 拮抗** | 英夫利昔单抗（5 mg/kg）可考虑超炎症表型 |
| **糖皮质激素** | 液体复苏后仍需血管活性药时考虑；推荐 Hydrocortisone 200 mg/d |
| **CytoSorb® 血液吸附** | 清除 IL-6、TNF-α 等循环细胞因子 |
| **GSDMD 靶向** | 仍在临床前研究阶段（Dexamethasone 间接抑制 GSDMD） |

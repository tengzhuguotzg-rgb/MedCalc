# 感染触发与固有免疫

## 核心断言

**PMNs（中性粒细胞）是脓毒症中固有免疫的核心效应细胞，同时扮演"杀菌"与"免疫病理"的双重角色。**

感染发生后，病原体相关分子模式（PAMPs）被模式识别受体（PRRs，如 TLR4）识别，触发中性粒细胞趋化、吞噬、ROS 爆发和 NETosis 释放。这一过程在感染控制中不可或缺，但过度激活或失调的 PMNs 会通过以下机制加重组织损伤：
- 延迟凋亡导致 PMNs 在组织中持续存在
- NETosis 过度激活形成微血栓
- 获得免疫抑制表型（PMNs-MDSCs）抑制 T 细胞功能

---

## 关键通路与分子

| 通路/分子 | 作用 |
|---------|------|
| **PAD4**（肽酰基精氨酸脱亚胺酶 4） | 催化组蛋白瓜氨酸化 → 染色质解聚 → NETosis |
| **Mcl-1**（髓系细胞白血病-1） | 抗凋亡蛋白，G-CSF/GM-CSF/LPS 上调 → PMNs 延迟凋亡 |
| **PD-L1** | 炎症/代谢应激下上调 → 诱导 T 细胞耗竭 |
| **Arginase-1** | 由 PMNs-MDSCs 分泌，耗竭精氨酸 → 抑制 T 细胞增殖 |
| **S100A8/A9** | 独立预后风险因素，诱导内皮线粒体功能障碍和 PANoptosis |
| **MLCK**（肌球蛋白轻链激酶） | MLCK/MYOCD/SRF 轴介导肺损伤中 PMNs 跨内皮迁移 |
| **PI3K/Akt/HIF-1α** | 调控糖酵解重编程（Warburg 效应）→ 损害 PMNs 趋化和吞噬功能 |

---

## 临床决策点

### 生物标志物
- **CD64**（FcγRI）：感染鉴别生物标志物，上调可区分脓毒症与术后 SIRS
- **PCT**：指导抗菌药物停药，PCT 引导策略平均缩短抗菌疗程 3.2 天
- **IL-6**：早期升高生物标志物， kinetics 反映疾病轨迹和死亡风险

### PMNs 功能评估
- 吞噬功能测定（荧光标记细菌或颗粒）
- 呼吸爆发活性（化学发光/流式细胞术）
- 表面标志物表型（CD11b、CD64、CD62L）

### 治疗策略
| 干预 | 机制 | 效果 |
|------|------|------|
| **IFN-γ** | JAK-STAT 通路激活 → 促进吞噬溶酶体成熟，稳定 NADPH 氧化酶 | 增强 PMNs 杀菌功能 |
| **Pentoxifylline** | PDE4 抑制 → cAMP↑；IKK 抑制 → NF-κB↓；Mcl-1 转录下调 | 血浆 IL-8 降低 48%，降低 MODS 发生率 |
| **CXCR2 拮抗剂（如 Reparixin）** | 优化趋化方向精度 | 减少破坏性旁观组织浸润（临床验证中） |
| **GSK484（PAD4 抑制剂）** | 阻断病理性 NETosis | 保留保护性 NETosis，限制微血栓形成 |

---

## 关键阈值速记

> [!warning] 临床阈值
> - 抗菌每延迟 1 小时 → 病死率增加 7.6%（OR 1.32）
> - CD4+ T 细胞耗竭 >80% 提示免疫抑制期
> - mHLA-DR <8000 抗体/细胞 提示单核细胞失活化
> - LPS 诱导 TNF-α <200 pg/mL 提示内毒素耐受

> [!tip] 通路记忆
> 感染 → TLR4 激活 → NF-κB → 炎症因子风暴（TNF-α/IL-1β/IL-6）→ PMNs 激活 → NETosis + ROS + 延迟凋亡 → 组织损伤 + 微血栓

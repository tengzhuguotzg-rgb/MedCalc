# 血管舒缩麻痹

## 核心断言

**血管舒缩麻痹（Vasoplegia）是脓毒性休克的血流动力学核心特征，表现为全身血管阻力（SVR）显著下降和对血管活性药反应性降低，需大剂量儿茶酚胺维持 MAP。**

机制核心：诱导型一氧化氮合酶（iNOS）上调 → NO 大量生成 → 血管平滑肌 cGMP↑ → 血管扩张；再加之 β-受体下调、肾上腺素能信号传导受损，血管对内源性和外源性儿茶酚胺的反应均下降。

---

## 核心机制

| 机制 | 通路/分子 |
|------|---------|
| **NO 过量生成** | iNOS 被炎症因子激活 → NO↑ → cGMP↑ → 血管平滑肌松弛 |
| **肾上腺素能受体下调/脱敏** | β-AR 磷酸化异常 → 对 NE 反应↓ |
| **血管紧张素受体1（AT1R）下调** | RAAS 系统失调 |
| **组蛋白介导的血管舒张** | DAMPs/组蛋白 → 直接舒张血管平滑肌 |
| **内皮素-1（ET-1）相对不足** | 血管收缩物质相对减少 |

> [!warning] 临床阈值
> - MAP 初始目标：≥ 65 mmHg（SSC 2021）
> - 需大剂量去甲肾上腺素（NE > 0.5–1 μg/kg/min 或等效剂量）提示严重血管麻痹
> - 乳酸 > 4 mmol/L + 需大剂量血管活性药 → 高死亡风险

---

## 治疗策略

| 药物 | 机制 | 注意事项 |
|------|------|---------|
| **去甲肾上腺素（NE）** | α1 受体激动 → 血管收缩 | 首选用药，但部分患者反应差 |
| **血管加压素（VP）** | V1a 受体激动 → 不依赖肾上腺素能通路 | 与 NE 联用可减少 NE 剂量 |
| **血管紧张素II（ATII）** | AT1R 激动 | 对 NE 低反应患者可能有效 |
| **亚甲蓝（Methylene Blue）** | 抑制 iNOS/可溶性鸟苷酸环化酶 | 难治性血管麻痹的补救治疗 |
| **氢化可的松** | 改善血管对儿茶酚胺敏感性 | 合并肾上腺皮质功能不全时 |

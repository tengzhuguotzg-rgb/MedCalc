# 心肌抑制

## 核心断言

**脓毒性心肌病（Sepsis-Induced Myocardial Dysfunction，SIMD）表现为左室射血分数（LVEF）可逆性下降和舒张功能障碍，机制与乳酸酸中毒、线粒体损伤和 β-肾上腺素能信号下调有关。**

SIMD 的特征是"可逆性"——存活患者的 LVEF 通常在 7–10 天内恢复，但恢复速度个体差异大。LVEF 下降程度与病死率正相关，LVEF < 40% 提示显著心肌抑制。

---

## SIMD 的病理生理机制

| 机制 | 关键分子/通路 |
|------|-------------|
| **乳酸酸中毒** | H⁺ 竞争性抑制 Ca²⁺ 与肌钙蛋白结合 → 心肌收缩力↓ |
| **线粒体损伤** | mPTP 开放 → ATP 耗竭 → 心肌能量崩溃 |
| **β-肾上腺素能信号下调** | β-AR 密度↓ + 磷酸化障碍 → 心肌对儿茶酚胺反应↓ |
| **TNF-α 直接心肌毒性** | TNF-α → 诱导心肌细胞凋亡 |
| **心肌血流分布异常** | 微血栓形成 + 血管内皮损伤 → 心肌灌注↓ |
| **代谢重编程** | 糖酵解↑ + 氧化磷酸化↓（心肌细胞 Warburg 效应） |

---

## 临床特点

| 指标 | 改变 |
|------|------|
| **LVEF** | 可逆性下降（< 40% 提示显著抑制） |
| **左室舒张末期内径（LVEDD）** | 脓毒症早期可扩大 |
| **E/e' 比率** | 舒张功能障碍标志 |
| **BNP/NT-proBNP** | 显著升高，反映心功能受损 |
| **肌钙蛋白** | 轻-中度升高，提示心肌损伤 |

> [!warning] 临床意义
> SIMD 与预后关系：LVEF < 40% + 乳酸 > 4 mmol/L 组合病死率显著升高。

# 内皮损伤

## 核心断言

**内皮损伤是脓毒症病理生理的"结构基础"，表现为血管内皮糖萼（Glycocalyx）降解、通透性增加和促凝表面转化，是微循环障碍、免疫血栓和多器官衰竭的共同上游。**

正常内皮维持抗凝、抗粘附和血管屏障功能。脓毒症中，PAMPs/DAMPs、炎症因子（ TNF-α、IL-6）和 NETs 直接损伤内皮，导致：
- 糖萼降解 → 通透性↑ → 组织水肿
- TF 表达↑ + TM/PS 表达↓ → 促凝表面
- ICAM-1/VCAM-1 上调 → 白细胞粘附/跨内皮迁移

---

## 关键分子标志物

| 标志物 | 意义 |
|--------|------|
| **Syndecan-1** | 糖萼降解标志物，升高提示糖萼损伤 |
| **Ang-2** | 竞争性抑制 Ang-1/Tie2 → 内皮屏障破坏，与病死率相关 |
| **VWF（血管性血友病因子）** | 内皮损伤后释放入血，VWF:Ag ↑提示内皮激活/损伤 |
| **可溶性 TM（sTM）** | 内皮损伤标志物，也是潜在抗凝治疗靶点 |
| **ET-1（内皮素-1）** | 强效血管收缩肽，升高提示内皮功能失调 |

---

## 临床决策点

> [!warning] 临床意义
> - 入院时 VWF:Ag 和 Ang-2 升高与脓毒症病死率正相关
> - 糖萼降解标志物（Syndecan-1）升高提示严重内皮损伤
> - 内皮损伤参与所有器官（肺/肾/脑/心）的功能障碍

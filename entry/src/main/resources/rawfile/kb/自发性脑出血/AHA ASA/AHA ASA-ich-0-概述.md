## TOP 10 TAKE-HOME MESSAGES

1. **区域医疗系统建设**：建议发展区域性卒中照护系统，确保ICH患者能快速转运至具备神经重症和神经外科能力的机构。

2. **血肿扩大预测**：血肿扩大与不良预后密切相关。CT平扫影像标记物联合临床标记（发病时间、抗栓药物使用）可预测血肿扩大风险。

3. **识别血管病变机制**：ICH是多种血管病理的结果，指南强调识别微血管和大血管出血机制的标志物。

4. **急性降压策略**：轻至中度ICH患者，限制血压变异、实现平稳持续的BP控制可减少血肿扩大并改善功能预后。

5. **抗凝相关ICH的逆转**：口服抗凝药相关ICH死亡率和致残率极高；指南更新了VKA（4因子PCC）、达比加群（idarucizumab）、Xa因子抑制剂（andexanet alfa）的逆转推荐。

6. **部分既往治疗无益或有害**：预防性糖皮质激素、持续高渗治疗对预后无益；非手术患者输注血小板可能恶化预后；逐级加压袜单独使用不能预防DVT；无癫痫证据时预防性抗癫痫药不能改善预后。

7. **微创手术降低死亡率**：微创血肿清除或脑室出血溶栓引流可降低死亡率，但功能预后改善不明确。小脑出血>15 mL新增手术指征。

8. **生命维持治疗决策**：DNR决策与限制其他治疗干预需分开考虑。预后量表可用于评估出血严重程度，但不应作为限制生命维持治疗的唯一依据。

9. **康复锻炼时机**：中度ICH患者发病24-48小时后可行拉伸和功能任务训练；24小时内积极早期活动可能增加14天死亡率。氟西汀不能改善功能康复且增加骨折风险。

10. **照护者支持**：指南强调对家庭照护者进行心理教育、实践支持和技能培训，以改善患者平衡、活动能力和生活质量。

## 证据等级体系

本指南采用AHA/ASA推荐的**COR + LOE双栏体系**：

| COR | 含义 |
|-----|------|
| 1 | 强推荐（Benefit >> Risk） |
| 2a | 中等推荐（Benefit > Risk） |
| 2b | 弱推荐（Benefit ≥ Risk） |
| 3 (No Benefit) | 不推荐（无明确益处） |
| 3 (Harm) | 禁忌/有害（Risk > Benefit） |

| LOE | 含义 |
|-----|------|
| A | 源自≥1项高质量RCT的证据 |
| B-R | 源自≥1项中等质量RCT或这些RCT的事后分析 |
| B-NR | 源自≥1项良好设计的观察性研究 |
| C-LD | 随机或观察性研究的局限性，或观察性研究的倾向性 |
| C-EO | 专家共识、标椎护理或病例报告 |

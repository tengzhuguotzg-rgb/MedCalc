## 1. INTRODUCTION

### 核心流行病学数据

- 美国每年约79.5万卒中，其中约10%为ICH
- ICH发病率：黑人比白人高约1.6倍；墨西哥裔比非西班牙裔白人高约1.6倍
- 中低收入国家ICH占全部卒中的比例和绝对发病率均显著高于高收入国家
- ICH是急性卒中中致死性最强的形式，早期死亡率约30-40%，近年无明显改善
- 随着人口老龄化，即便BP控制改善，ICH发病率仍将维持较高水平
- 抗凝药物使用增加是ICH的日益增长的病因来源

### 1.1 方法学与证据审查

- 文献检索：2020年10月至2021年3月，涵盖MEDLINE、PubMed、Cochrane Library等
- 2021年3月至11月间发表的、可能影响推荐内容、COR或LOE的重要试验已纳入
- 所有推荐意见经全体撰写组充分审查讨论
- 采用改良Delphi流程达成共识
- 相关RWI撰写组成员在涉及利益的推荐上回避投票
- 所有推荐获得88.9%-100%的投票成员同意

### 1.2 撰写组构成

撰写组成员包括：血管神经科医生、神经重症医生、神经外科医生、急诊医生、血液科医生、康复医学医生、急性护理执业护士、住院医师和患者代表。

联署机构：AHA/ASA、AANS/CNS、美国神经病学学会（AAN）。

### 1.3 指南范围

本指南针对成人自发性ICH的诊断、治疗和预防，**排除**以下情况：
- 创伤性脑出血
- 结构性病因（血管畸形、囊状动脉瘤、易出血性肿瘤）

本指南与以下指南**不重叠**：AVM、动脉瘤性SAH、未破裂囊状动脉瘤的治疗指南。

涵盖范围：院前急救组织(L3) → 初步诊断评估(L4) → 急性内科和外科干预(L5.1、L5.2、L6) → 住院并发症管理(L5.3-5.6) → 治疗目标评估(L7) → 康复与恢复(L8) → 复发预防(L9)。

## 4. DIAGNOSIS AND ASSESSMENT

---

### 4.1.1 体格检查与实验室评估

#### 推荐意见

| COR | LOE | 推荐内容 |
|-----|-----|---------|
| — | C-LD | 1. 自发性ICH患者入院时应完成：血常规、PT/INR/PTT、肌酐/eGFR、血糖、心肌肌钙蛋白+心电图、毒筛、炎症指标 |

#### 核心实验室检查临床意义

| 检查 | 临床意义 |
|------|---------|
| CBC | 极度血小板减少（如<10,000）可致ICH；贫血与HE和不良预后相关；血小板减少与死亡率增加相关 |
| PT/INR/PTT | 抗凝相关ICH与更大血肿体积、扩大和死亡率增加相关 |
| DOAC特异性检测（稀释凝血酶时间、抗Xa活性） | 可指导重启抗凝决策 |
| 心肌肌钙蛋白 | 入院升高与院内死亡率增加相关 |
| 心电图 | 左室肥厚提示慢性高血压；识别心肌缺血或既往心脏损伤 |
| eGFR | 肾功能影响DOAC清除；入院肾衰与不良功能预后和住院/12个月死亡率相关 |
| 血糖 | 入院高血糖与短期和长期不良预后及死亡率相关 |
| 毒筛 | 可卡因等拟交感神经药物与ICH相关；妊娠妇女需妊娠试验 |
| 炎症标志物（ESR/CRP） | 与感染性心内膜炎相关 |

#### 知识缺口

- 血小板或凝血活性测定是否可识别从血小板输注、氨甲环酸等治疗中获益的亚组
- 传统凝血试验不可靠地确定DOAC抗凝水平；特定Xa抑制水平尚未普及
- 粘弹性止血试验（血栓弹力图/旋转血栓弹力图）在ICH中的预后价值待明确

### 4.1.2 影像学诊断

#### 推荐意见

| COR | LOE | 推荐内容 |
|-----|-----|---------|
| 1 | B-NR | 1. 疑似卒中症状患者应快速行CT或MRI以确诊自发性ICH |
| 2a | B-NR | 2. 自发性ICH和/或IVH患者，发病24小时内系列头CT有助于评估HE |
| 2a | C-LD | 3. ICH/IVH伴低GCS或ND患者，系列头CT有助于评估HE、脑积水、脑肿胀或脑疝 |
| 2b | B-NR | 4. ICH发病数小时内行CTA可能有助于识别HE风险患者 |
| 2b | B-NR | 5. 使用NCCT影像标记物（heterogeneous density、swirl sign、black hole sign等）识别HE风险可能合理 |

#### 影像学核心数据

- **CT**：最广泛应用，确诊ICH最快速、准确；MRI梯度回波或SWI同样可高准确度检测急性ICH
- **HE时间窗**：发病3小时内CT扫描患者，1小时扫描时26%已发生显著HE，20小时扫描时额外12%；6-12小时扫描15%；12-24小时6%；24小时后罕见（0%）
- **CTA斑点征**：与HE、死亡率、不良mRS相关；发病至扫描0-2小时内敏感性和阳性预测值最高
- **NCCT标记物**：blend sign、black hole sign、swirl sign、satellite sign等与HE和不良功能预后相关（高度异质性）
- **复查CT指征**：神经功能恶化或意识障碍者；GCS≥5稳定患者6小时和24小时各扫一次可排除HE并记录最终血肿体积

#### 延迟IVH

- 21%初始无IVH患者会出现延迟IVH；与延迟HE相关；独立增加死亡率和不良预后；常需急诊手术干预

#### 知识缺口

- 无论神经状态如何，系列CT的成本收益比及适用人群
- NCCT标记物作为HE预测评分组成部分的标准化定义和前瞻性验证
- 超早期（<2小时）启动氨甲环酸等止血治疗是否改善预后

### 4.2 出血机制诊断评估

#### 推荐意见

| COR | LOE | 推荐内容 |
|-----|-----|---------|
| — | B-NR | 1. 脑叶ICH且<70岁、深部/后颅窝ICH且<45岁、或深部/后颅窝45-70岁且无高血压病史患者：急性CTA+静脉成像排除大血管病因或脑静脉血栓 |
| — | B-NR | 2. 单纯IVH且无可见实质出血患者：应进行导管动脉DSA排除大血管病因 |
| — | C-LD | 3. CTA或MRA提示大血管病因的ICH患者：应尽快行导管动脉DSA确认并处理血管畸形 |
| 2a | B-NR | 4. 脑叶ICH<70岁、深部/后颅窝ICH<45岁、或45-70岁深部/后颅窝ICH无高血压病史且非侵入性检查（CTA±静脉成像+MRI/MRA）阴性时：DSA合理排除大血管病因 |
| 2a | B-NR | 5. CTA/静脉成像阴性的ICH患者：MRI和MRA有助于确定非大血管性病因（如CAA、深穿通血管病、脑海绵状血管瘤或恶性肿瘤） |
| 2a | C-LD | 6. 入院已行CT或MRI的ICH患者：急性期行CTA+静脉成像或MRA+静脉成像有助于排除大血管病因或脑静脉血栓 |
| 2b | C-LD | 7. 导管DSA阴性且无明确微血管诊断或其他结构性病变的ICH患者：发病3-6个月后重复导管DSA可能合理，以识别先前被遮蔽的血管病变 |

#### 大血管病因检查策略

**DIAGRAM研究关键数据**：

| 人群 | 大血管病因阳性率 |
|------|----------------|
| 脑叶ICH <70岁 | 高（需综合评分） |
| 深部/后颅窝ICH <45岁无高血压 | ~17% |
| 深部ICH伴小血管病征象（白质病变、腔隙性脑梗死）51-70岁 | ~1% |
| 18-50岁脑叶/后颅窝ICH无小血管病 | >50% |

**无创成像敏感性**：CTA和MRA对颅内血管畸形检测敏感性和特异性均>90%（高度选择人群vs.导管DSA）

**MRI/MRA在DSA阴性患者的作用**：
- 检测CAA标志物：微出血、皮层浅表铁沉积（敏感序列：SWI、susceptibility-weighted imaging）
- 排除肿瘤或其他占位病变；建议3-6个月后复查增强T1以排除肿瘤
- 深穿通血管病、脑海绵状血管瘤识别

**重复DSA指征**：<65岁皮质下ICH患者，初始DSA阴性者中4/22在重复DSA时发现AVM

#### 知识缺口

- 选择检查标准不应仅基于高血压、糖尿病等危险因素（这些因素难以确定）
- 小血管病标志物（白质高信号、腔隙、微出血、浅表铁沉积）可纳入高/低风险分层
- 需要开发和验证诊断标准帮助临床医生和研究人员按出血病因对ICH患者分类
- 非侵入性成像（动脉自旋标记、血管壁成像）能否替代DSA

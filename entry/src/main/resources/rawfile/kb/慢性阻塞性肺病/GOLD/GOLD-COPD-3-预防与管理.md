## 关键要点

> [!abstract] 管理目标
> > - **核心目标**：减轻症状 + 降低未来风险（急性加重、住院、死亡）
> - 2026 新概念：「疾病活动度」—— 无急性加重、无症状恶化、无加速肺功能下降 → 低疾病活动度

## 疾病活动度

2026 版新增章节。**低疾病活动度**定义为：一段时间内无急性加重、无症状恶化、无加速肺功能下降。目标是预防短期和长期的所有急性加重。

| 状态 | 定义 |
|------|------|
| **疾病稳定** | 低疾病活动度：无急性加重 + 无症状恶化 + 无加速肺功能下降 |
| **疾病控制** | 低疾病活动度 + 症状低于阈值水平（更严格） |

## 风险降低、生活方式与患者教育

### 戒烟

> [!important] 关键推荐
> - 应积极鼓励并支持所有吸烟者戒烟
> - 尼古丁替代疗法和药物治疗可可靠地提高长期戒烟率
> - 立法的吸烟禁令和医务人员咨询可提高戒烟率
> - **目前没有充分证据支持电子烟作为戒烟辅助手段的安全性和有效性**

**5A 策略**：
| 步骤 | 内容 |
|------|------|
| Ask（询问） | 每次就诊系统性地识别所有烟草使用者 |
| Advise（建议） | 强烈敦促所有烟草使用者戒烟 |
| Assess（评估） | 评估患者戒烟意愿 |
| Assist（协助） | 制定戒烟计划，提供咨询和药物治疗 |
| Arrange（安排） | 安排随访 |

> [!tip] 戒烟效果
> COPD 患者持续戒烟率可达 14-27%。戒烟对疾病自然史的影响最大，可改善日常症状、减少急性加重频率。

### 疫苗接种

| 疫苗 | 推荐级别 |
|------|---------|
| 年度流感疫苗 | 证据 B |
| SARS-CoV-2（COVID-19） | 证据 B（按最新推荐） |
| PCV21 或 PCV20（肺炎球菌） | 证据 B |
| RSV 疫苗（≥ 50 岁和/或慢性心肺疾病） | 证据 A |
| Tdap（百白破，未接种者） | 证据 B |
| 带状疱疹疫苗（> 50 岁） | 证据 B |

### 其他环境因素

- 减少室内外空气污染暴露
- 避免职业粉尘、烟雾和气体暴露
- 应对气候变化影响：热浪/极寒条件下 COPD 患者死亡风险增加

## COPD稳定期维持药物治疗

### 原则

- 初始药物治疗应个体化，基于：症状严重程度、急性加重风险、副作用、合并症、药物可及性与成本、患者偏好和吸入装置使用能力
- 定期评估吸入技术和依从性

### ABE 分组初始药物治疗

| 组别 | 初始治疗推荐 |
|------|-------------|
| **A 组** | 支气管扩张剂（LABA 或 LAMA） |
| **B 组** | LABA + LAMA（双联） |
| **E 组** | LABA + LAMA（双联）；若血 EOS ≥ 300/μL，考虑 LABA + LAMA + ICS（三联） |

### 随访药物治疗

- 规律评估后若症状仍明显或仍有急性加重 → 升级治疗
- 降级治疗仅在明确判断无获益或出现严重副作用时考虑

> [!warning] ICS 使用原则
> - 血 EOS < 100/μL → ICS 效果有限，不建议使用
> - 血 EOS 100-299/μL → ICS 可作为辅助治疗
> - 血 EOS ≥ 300/μL → ICS 获益最大，强烈建议在双联基础上加用

### 吸入装置

> [!tip] 装置选择
> 需根据患者手口协调能力、吸气流量等选择适当装置。每年至少评估一次吸入技术。

## COPD非药物治疗

| 干预 | 推荐 | 证据等级 |
|------|------|---------|
| **肺康复** | 改善运动能力、症状和生活质量（所有GOLD分级） | A |
| **长期氧疗 (LTOT)** | PaO₂ ≤ 55 mmHg 或 < 60 mmHg 伴肺源性心脏病或红细胞增多症；不推荐用于中重度去饱和 | A |
| **长期 NIV** | 精选患者，特别是日间显著高碳酸血症且近期住院者 | B |
| **疫苗接种** | 见上 | 见上 |
| **肺减容手术** | 精选上叶肺气肿 + 低运动耐量患者 | A |

## 姑息治疗、介入治疗与手术治疗

- 对优化药物治疗无效的晚期肺气肿患者，可选择手术或支气管镜介入治疗
- 姑治疗方法可有效控制晚期 COPD 的症状

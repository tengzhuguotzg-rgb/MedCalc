## 关键要点

> [!abstract] 诊断核心
> - **任何**有呼吸困难、慢性咳嗽、咳痰、反复下呼吸道感染和/或危险因素暴露史的患者，均应考虑 COPD
> - **舒张后 FEV₁/FVC < 0.7** 是确诊的气道阻塞标准
> - 初始评估目标：确定气流阻塞严重程度、当前症状影响、未来事件风险（急性加重、住院、死亡）
> - 规律随访肺功能、症状和急性加重是调整治疗的基础

## 诊断

### 需考虑诊断的临床线索

当出现以下线索（尤其多个同时存在时）需考虑 COPD 并应行肺功能检查：

| 类别 | 具体表现 |
|------|---------|
| **呼吸困难** | 进行性加重、运动时加重、持续性 |
| **慢性咳嗽** | 可间歇性，可无咳痰 |
| **反复下呼吸道感染** | |
| **危险因素暴露史** | 烟草烟雾、烹饪取暖燃料烟雾、职业粉尘/气体/化学物、宿主因素（遗传、发育异常、早产、低出生体重、儿童期呼吸道感染等） |

### 症状

- 慢性呼吸困难是 COPD 最特征性症状
- 咳痰见于 ≤ 30% 患者
- 症状可逐日变化，可能早于气流阻塞数年出现

### 鉴别诊断

| 疾病 | 提示特征 |
|------|---------|
| COPD | 症状缓慢进展，吸烟或危险因素暴露史 |
| 哮喘 | 可变性气流阻塞，过敏/鼻炎/湿疹史，儿童期起病 |
| 充血性心衰 | 胸片示心脏扩大、肺水肿 |
| 支气管扩张 | 大量脓痰，HRCT 示支气管扩张 |
| 结核 | 各年龄均可起病，微生物学确认 |

### 肺功能（Spirometry）

- 用力肺功能是最可重复、最客观的气流阻塞测量方法
- **舒张后 FEV₁/FVC < 0.7** 是 GOLD 诊断 COPD 的标准
- 舒张前肺功能可用于排除诊断；如舒张前提示阻塞需用舒张后值确认

> [!warning] 固定比值的局限性
> 固定 FEV₁/FVC < 0.7 标准可能导致老年人过度诊断，并在约 1% 年轻成人中（特别是轻度疾病）漏诊。若怀疑年轻患者（< 50 岁）COPD 但比值 ≥ 0.7，可参考 LLN 或 z 评分。

## 筛查与病例发现

- **未诊断 COPD 的影响**：81.4% 的肺功能定义 COPD 未被诊断，导致无治疗或不正确治疗
- GOLD 不推荐在无症状人群中普遍筛查，推荐在目标人群中进行**机会性病例发现**

> [!tip] 病例发现途径
> - 初级保健中的机会性病例发现
> - 肺癌 CT 筛查中发现的机会性病例
> - 疑似肺癌的偶发影像发现

## 初始评估

### GOLD 分级系统（基于舒张后 FEV₁）

| GOLD 分级 | FEV₁ 占预计值% |
|-----------|---------------|
| GOLD 1（轻度） | ≥ 80% |
| GOLD 2（中度） | 50-79% |
| GOLD 3（重度） | 30-49% |
| GOLD 4（极重度） | < 30% |

### ABE 分组（2023年后）

**2026 版更新**：仅 1 次中度/重度急性加重即归入 E 组（之前需 ≥ 2 次）。

| 组别 | 特征 | 标准 |
|------|------|------|
| **A** | 低症状、低风险 | mMRC 0-1 或 CAT < 10；急性加重 0-1 次/年（且未因此住院） |
| **B** | 高症状、低风险 | mMRC ≥ 2 或 CAT ≥ 10；急性加重 0-1 次/年（且未因此住院） |
| **E** | 高风险 | 过去 1 年 ≥ 1 次中度/重度急性加重住院或 ≥ 2 次中度加重（门诊） |

### 血嗜酸性粒细胞计数

作为 ICS 治疗反应的预测标志物。目前阈值：血 EOS ≥ 300 cells/μL 提示 ICS 获益更大。

## 随访与监测

- 评估每 1-3 个月一次（起始或调整治疗后），病情稳定后可延长间隔
- 监测内容：症状、急性加重、依从性与吸入技术、吸烟状态、肺功能、影像、合并症

## 补充检查

| 检查 | 应用指征 |
|------|---------|
| **肺容量测定** | 气流阻塞严重程度与症状不匹配时 |
| **DLco** | 评估肺气肿对气体交换的影响 |
| **血氧/动脉血气** | 静息 SpO₂ < 92%、呼吸困难重、右心衰或红细胞增多症 |
| **运动试验** | 评估功能障碍程度、鉴别呼吸困难原因 |
| **胸部CT** | 肺气肿评估、肺结节管理、支气管扩张筛查、AATD 评估 |
| **α-1抗胰蛋白酶筛查** | 早发 COPD（< 45 岁）、家族史、无吸烟史的下叶肺气肿 |

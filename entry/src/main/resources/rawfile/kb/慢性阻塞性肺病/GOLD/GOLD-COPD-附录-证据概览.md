## 附录3：证据概览——药物治疗

### 戒烟药物

| 药物 | 效果 | 备注 |
|------|------|------|
| 尼古丁替代疗法（贴片/口香糖/吸入剂/鼻喷雾/舌下片/含片） | 可靠提高长期戒烟率 | 近期心梗或中风后有争议 |
| 安非他酮（Bupropion） | 14-27% 持续戒烟率 | 证据支持 |
| 伐尼克兰（Varenicline） | 58.3%（9-24周持续戒断率） | 与安非他酮效果相似 |
| 电子烟/雾化 | 有效但安全性不明确 | **不推荐用于COPD患者戒烟**（缺乏长期安全性数据） |

> [!warning] 电子烟警告
> 电子烟可能导致肺部急性损伤（EVALI）、细胞毒性、炎症反应、纤毛功能受损，短期和长期安全性均不清楚，**不应推荐**。

### 稳定期维持治疗药物

| 药物类别 | 主要作用 | 常见药物 |
|----------|---------|---------|
| **SABA**（短效β2激动剂） | 按需使用，4-6小时 | 沙丁胺醇、特布他林、非诺特罗 |
| **LABA**（长效β2激动剂） | 12/24小时，改善FEV₁和症状 | 福莫特罗、沙美特罗（bid）；茚达特罗、奥洛特罗、维兰特罗（qd） |
| **SAMA**（短效抗胆碱药） | 6-8小时 | 异丙托溴铵 |
| **LAMA**（长效抗胆碱药） | 12/24小时，减少急性加重 | 噻托溴铵（qd）、格隆溴铵、乌美溴铵 |
| **LABA+LAMA** | 优于单药改善FEV₁和症状 | 双联，首选 |
| **LABA+ICS** | 改善肺功能+减少急性加重 | ICS用于EOS ≥ 100/μL |
| **LABA+LAMA+ICS（三联）** | 尤其EOS ≥ 300/μL | 单装置三联：糠酸氟替卡松+乌美溴铵+维兰特罗等 |
| **PDE3/4抑制剂** | 罗氟司特减少慢性支气管炎型加重 | 恩塞芬特林为PDE3/4双重抑制剂 |
| **祛痰药** | 厄多司坦、羧甲司坦、N-乙酰半胱氨酸 | 精选患者可减少加重 |
| **生物制剂** | 度普利尤单抗（dupilumab, q2w） | EOS ≥ 300/μL 且慢性支气管炎型 |

### 吸入装置对比

> [!tip] 吸入器选择
> - 不同装置之间疗效差异小于装置选择错误的影响
> - 关键：患者手口协调能力、吸气流量
> - MDI（定量吸入器）+ 储雾罐 vs DPI（干粉吸入器）vs SMI（软雾吸入器）
> - **每年至少评估一次吸入技术**

### 支气管扩张剂证据摘要

- 支气管扩张剂规律使用改善肺功能、减少动态过度充气、改善运动耐量（证据 A）
- LABA/LAMA 优于短效药物（证据 A），显著改善呼吸困难、健康状态和减少加重（证据 A）
- LAMA 减少加重效果优于 LABA（证据 A），并减少住院（证据 B）
- 起始长效治疗时**首选 LABA + LAMA 双联**（证据 A）
- 单药仍有呼吸困难 → 升级至双联（证据 A）

### ICS 证据摘要

- ICS + LABA 联合优于单药（证据 A）
- 三联（LABA + LAMA + ICS）优于双联，尤其加重高风险且 EOS ≥ 300/μL 时（证据 A）
- ICS 长期使用增加肺炎风险（证据 A）
- EOS < 100/μL 时 ICS 效果有限（证据 B）

## 附录4：证据概览——非药物治疗

### 教育、自我管理与整合照护

| 干预 | 效果 | 证据 |
|------|------|------|
| 患者教育 | 提高应对能力、疾病认知 | B |
| 自我管理干预 | 减少住院；对死亡率/肺功能影响无结论 | B |
| 整合照护 | 改善生活质量、运动能力 | B |

### 肺康复

> [!important] 肺康复核心推荐
> - 含运动训练+疾病特异性教育，改善运动能力、呼吸困难和乏力（证据 A）
> - 所有 GOLD 分级的 COPD 患者均有获益
> - 住院期间早期开始可加快恢复

### 氧疗与通气支持

| 干预 | 指征 | 证据 |
|------|------|------|
| 长期氧疗（LTOT） | PaO₂ ≤ 55 mmHg 或 PaO₂ < 60 mmHg 伴肺心病/红细胞增多症 | A |
| 长期 NIV | 精选患者，尤其日间高碳酸血症+近期住院 | B |
| 居家无创通气 | 慢阻肺相关慢性呼吸衰竭 | B |

### 手术与介入治疗

- 肺减容手术（LVRS）：上叶肺气肿+低运动量精选患者，减少死亡（证据 A）
- 支气管镜肺减容：改善运动能力、生活质量（证据 B）
- 肺移植：精选晚期 COPD 患者（证据 C）

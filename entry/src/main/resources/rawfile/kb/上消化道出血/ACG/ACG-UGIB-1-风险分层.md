## 推荐意见

### Rec 1 低风险患者识别与出院管理

上消化道出血（UGIB）患者就诊急诊时，建议使用风险评估工具识别**极低风险患者**（如 Glasgow-Blatchford 评分 ≤1），定义为评分对应的住院干预或死亡假阴性率 ≤1%，可安排出院门诊随访，而非住院（条件推荐，极低质量证据）。

> [!tip] 临床提示
> Glasgow-Blatchford 评分（GBS）0 分患者敏感性点估计 99%–100%（95%CI 下限 98%），特异性仅 8%–22%；GBS ≤1 敏感性 99%（95%CI 下限 97%–98%），特异性提升至 27%–40%。

**支持文本：**

**评估目的**：识别极低风险患者，使部分患者能够安全地从急诊出院门诊随访，降低医疗费用，同时不会使患者面临需要住院干预或死亡的风险。

**假阴性率设定**：研究显示 GBS = 0 在4项个体研究中敏感性点估计为 99%–100%（95%CI 下限 98%）；GBS ≤1 敏感性点估计 99%（95%CI 下限 97%–98%）。敏感性 99% 意味着每 100 例需要住院干预或死亡的患者中，约有 1 例被错误分类为不需要干预。

**人群分布**：两项大型多中心研究显示，UGIB 患者中 GBS ≤1 占比 19%–24%，即约五分之一的患者可能无需住院。

**GBS = 0 的出院证据**：一项前瞻-回顾研究显示，实施 GBS = 0 患者非必要不入院规则前，105 例患者中 0 例需要住院干预或 30 天内死亡；规则实施后，84 例未入院患者同样 0 例出现不良结局。另有回顾性病例系列显示，急性 UGIB 且 GBS ≤1 患者出院后，103 例中 0 例需要住院干预或 30 天内死亡。

**评分阈值应用**：Shung 等机器学习模型可设定敏感性为 99%（与 GBS ≤1 相当）或 100%（与 GBS = 0 相当），且特异性高于传统 GBS；但因仅在单一环境中验证，指南暂以 GBS 为推荐工具。

**Glasgow-Blatchford 评分表（GBS）**：

| 风险因素 | 评分 |
|---------|------:|
| **血尿素氮（mg/dL）** | |
| 18.2 – <22.4 | 2 |
| 22.4 – <28.0 | 3 |
| 28.0 – <70.0 | 4 |
| ≥70.0 | 6 |
| **血红蛋白（g/dL）** | |
| 男 12.0 – <13.0；女 10.0 – <12.0 | 1 |
| 男 10.0 – <12.0 | 3 |
| <10.0 | 6 |
| **收缩压（mmHg）** | |
| 100 – 109 | 1 |
| 90 – 99 | 2 |
| <90 | 3 |
| **心率（次/分钟）** | |
| ≥100 | 1 |
| **黑便（Melena）** | 1 |
| **晕厥（Syncope）** | 2 |
| **肝病史** | 2 |
| **心力衰竭史** | 2 |

> [!warning] 注意
> 肝病和心力衰竭的定义在原始报告中未明确。最近研究定义：肝病为已知病史或临床/实验室急性或慢性肝病证据；心力衰竭为已知病史或临床/超声心动图心力衰竭证据。

---

### Rec 2 限制性输血策略

UGIB 患者建议采取限制性输血策略，输血阈值为**血红蛋白 ≥7 g/dL**（条件推荐，低质量证据）。

**支持文本：**

指南采用限制性输血策略对比开放性输血策略的研究证据，在 UGIB 人群中评估输血阈值对预后的影响。

> [!note] 证据等级说明
> 本条推荐基于低质量证据。输血阈值具体数值及获益幅度建议结合患者个体情况判断。

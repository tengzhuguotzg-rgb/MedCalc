# 诊断标准

## 推荐

**推荐意见 13**（中等推荐，证据等级 C）

推荐使用**"病因-分型-机制-功能"四步诊断范式**诊断重症凝血病。

### 四步诊断法

| 步骤 | 内容 | 说明 |
|------|------|------|
| 第一步 | 病因诊断 | 识别致病因素（感染/创伤/肿瘤等） |
| 第二步 | 分型诊断 | 确定表型（血栓型/出血型）+ 亚型（高凝/低凝） |
| 第三步 | 机制诊断 | 识别病理生理机制（内皮损伤/凝血因子缺乏/纤溶异常等） |
| 第四步 | 功能诊断 | 评估凝血功能状态（障碍/衰竭） |

**临床价值**：保证诊断全面准确，直接对应治疗原则，便于制定个体化治疗方案。

---

**推荐意见 14**（强推荐，证据等级 B）

推荐使用**中国SIC**、**ISTH-DIC**、**凝血衰竭**诊断标准。

### 中国 SIC 诊断标准（2019 ISTH）

≥4分可诊断 SIC，并可启动抗凝治疗。

| 指标 | 1分 | 2分 | 3分 |
|------|-----|-----|-----|
| SOFA评分（分） | 1 | 2 | — |
| 血小板计数（×10⁹/L） | ≥100~<150 | ≥50~<100 | <50 |
| INR | ≤1.2 | >1.2~≤1.4 | >1.4 |
| PT（s） | 延长<3s | 延长3~6s | 延长≥6s |
| 纤维蛋白原（g/L） | ≥1.0 | <1.0 | — |
| 乳酸（mmol/L） | — | — | >2 |

**注意**：中国以血小板计数<100×10⁹/L为血小板减少症标准（而非欧美的<150×10⁹/L），本共识推荐采用中国SIC标准，避免过早诊断SIC而启动抗凝治疗。

### ISTH-DIC 诊断标准（2025更新）

| 指标 | 得分 |
|------|------|
| 血小板计数（×10⁹/L） | ≥50→1分；<50→2分 |
| PT延长 | 3~6s→1分；≥6s→2分 |
| 纤维蛋白原 | <1.0g/L→1分 |
| D-二聚体 | 中度升高→1分；重度升高→2分 |
| **总分≥5分** → 显性DIC；<5分 → 非显性DIC（需每日再评估） |

### 凝血衰竭诊断标准

**ISTH-DIC积分 = 8分 + 血乳酸>2mmol/L** → 诊断凝血衰竭

### 三者关系

- **SIC**：脓毒症诱发的早期凝血病，SIC积分≥4分即可诊断
- **DIC**：多种致病因素导致凝血酶广泛激活为核心特征的病理生理过程；显性DIC=失代偿期；非显性DIC=代偿期
- **凝血衰竭**：DIC积分8分+乳酸>2mmol/L，代表最严重的凝血功能失代偿状态

### DIC 分型（按病理生理机制）

| 类型 | 机制 | 治疗原则 |
|------|------|---------|
| 血栓型DIC | PAMP/DAMP → 系统性炎症/内皮损伤 → 凝血酶大量活化 → 抗凝/纤溶抑制 → 广泛微血栓 → MOF | 早期抗凝 + 血管内皮保护 |
| 纤溶型DIC | t-PA大量释放 → 纤溶亢进 → 纤维蛋白过度降解 → 出血 | 早期抗纤溶 + 替代治疗 |

**警告**：不理解DIC分型而错误选择抗凝治疗或抗纤溶治疗，可能加重凝血障碍乃至死亡。

### 概念辨析

| 概念 | 强调点 |
|------|--------|
| 凝血衰竭 | 强调凝血功能状态和支持治疗（"贵在重症"） |
| DIC | 强调凝血酶广泛激活的病理生理过程和对因治疗（"精在懂病"） |

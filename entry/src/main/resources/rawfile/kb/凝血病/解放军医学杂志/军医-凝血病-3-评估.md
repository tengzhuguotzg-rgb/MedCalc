# 评估

## 推荐

**推荐意见 5**（强推荐，证据等级 B）

重症凝血病的评估需关注患者的**基础疾病、用药史、年龄、性别及体重**。

### 出血风险增加的情况

- 遗传性凝血因子缺乏（血友病）、遗传性纤维蛋白原减少症、血小板功能缺陷
- 获得性凝血因子减少、血小板减少、病理性抗凝物质形成、纤溶亢进
- 消化道溃疡、肝硬化、支气管扩张、痔疮（潜在出血风险疾病）
- 抗凝或抗血小板聚集药物用药史
- 严重创伤或外科手术后 24h 内

**流行病学数据**：
- 韩国 2013-2017 年 1,319,807 例手术患者：术后胃肠道出血发生率 0.27%（3505例）
- 急性冠脉综合征 1379 例随访：严重出血累积发生率 8.6%；年龄>75岁、贫血、高血压、心衰是主要危险因素

---

**推荐意见 6**（强推荐，证据等级 B）

推荐使用**血涂片进行外周血形态学检查**辅助重症凝血病的诊断。

### 血涂片诊断价值

外周血涂片可初步判断白细胞、红细胞及血小板的数量、大小及形态异常。

**高诊断价值疾病**：
- 假性血小板减少症
- TTP（血栓性血小板减少性紫癜）
- HUS（溶血尿毒症综合征）
- 急性白血病
- 淋巴瘤

---

**推荐意见 7**（强推荐，证据等级 C）

充分利用**常规凝血试验及纠正试验**进行重症凝血病的筛查。

### 常规凝血试验体系

| 检测类别 | 指标 | 临床意义 |
|---------|------|---------|
| 凝血因子活性 | APTT | 内源性凝血途径；延长→因子缺乏/抗凝物质/出血风险；缩短→高凝状态 |
| | PT/INR | 外源性凝血途径；延长→因子缺乏/抗凝物质 |
| | TT + 纤维蛋白原 | 共同途径；TT延长+Fib正常→抗凝物质；TT延长+Fib↓→低纤维蛋白原血症 |
| 血小板 | 血小板计数 | 减少→出血风险↑；增多→血栓风险↑ |
| 抗凝系统 | AT（抗凝血酶） | AT+凝血酶→灭活凝血酶；肝素与AT结合可使灭活反应提升千倍以上；AT↓→抗凝能力↓ |
| 纤溶系统 | FDP + D-二聚体 | 两者均↑→存在纤溶活动；FDP↑为主、D-二聚体不明显→原发性纤溶亢进；两者均明显↑→继发性纤溶亢进 |

### 纠正试验

用于不明原因凝固时间延长的筛查：
- **混合血浆纠正试验**：加入正常混合血浆后 APTT 恢复正常 → 因子缺乏；仍不纠正 → 存在抑制物/抗凝物质
  - 计算 Rosner 指数（RI）：RI<10%→因子缺乏；>15%→凝血抑制物；10%-15%→临界值
- **鱼精蛋白纠正试验**：凝血时间延长加入鱼精蛋白后纠正 → 血液含有肝素

### 不同病因凝血病的时相特征

| 病种 | 早期 | 中期 | 晚期 |
|------|------|------|------|
| SIC（脓毒症性凝血病） | 纤溶指标（D-二聚体/FDP）↑ → 血小板破坏 → PT/APTT 延长 → 纤维蛋白原↓ | | |
| TIC（创伤性凝血病） | 凝血酶活化→D-二聚体/FDP↑ → 纤维蛋白原快速消耗 → PT/APTT 延长 → 血小板计数↓（相对较晚） | | |

**推荐意见 8**（强推荐，证据等级 C）

推荐使用**黏弹力凝血试验**作为重症凝血病的常规诊断方法。

### 两大技术对比

| 指标 | TEG（血栓弹力图） | 凝血与血小板功能分析仪 |
|------|-----------------|----------------------|
| 标本 | 全血 | 全血 |
| 优势 | 纤溶系统功能监测（量化优势） | 凝血因子及血小板功能监测更敏感 |
| 共同指标 | 凝血因子、纤维蛋白原、血小板功能、整体凝血状态 | |

### TEG 五种试验类型

| 类型 | 主要指标 | 正常值 |
|------|---------|------|
| 普通试验 | R时间（凝血因子）、α角/K时间（纤维蛋白原）、MA（血小板）、LY30%（纤溶）、CI（整体） | — |
| 肝素酶对比试验 | R普通试验/R肝素酶试验 | 1.5~2.0 为宜（普通肝素监测） |
| 功能性纤维蛋白原（FF）检测试验 | RFF时间、MAFF、FLEV | RFF 1.3~2.5min；MAFF 10.1~25.3mm；FLEV 18.43~46.17 mg/L |
| 快速检测试验 | ACT（代替R时间） | 86~118s（反映外源性凝血途径因子） |
| 血小板图检测试验 | AA途径/ADP受体血小板抑制率 | >50%说明起效；过高需评估出血风险 |

### 凝血与血小板功能分析仪

| 方法 | 激活剂 | 指标 |
|------|--------|------|
| 高岭土法 | 高岭土 | ACT（80~116s自然血/90~145s抗凝血）、CR（9~56自然血/13~65抗凝血） |
| 玻璃珠法 | 玻璃珠 | ACT、CR、PF（>1） |

---

**推荐意见 9**（强推荐，证据等级 C）

推荐**联合使用常规凝血试验及黏弹力凝血试验**诊断重症凝血病。

常规凝血试验（血浆基础凝血理论）+ 黏弹力凝血试验（细胞基础凝血理论）→ 全面评估类型及严重程度。

**联合优势**：
- 常规凝血试验：快速提供凝血障碍证据
- TEG普通试验：准确判断凝血因子活性
- 肝素酶对比试验：准确判断肝素残留/抗凝效果
- FF水平+TEG普通试验：辅助诊断血小板功能障碍

---

**推荐意见 10**（中等推荐，证据等级 C）

推荐使用**新型凝血分子标志物**判断重症凝血病的分型及预后。

### 四种新型标志物

| 标志物 | 正常值 | 意义 |
|--------|--------|------|
| **TM**（血栓调节蛋白） | 3.8~13.3 TU/ml | 血管内皮损伤的敏感指标；肾功能不全可↑ |
| **TAT**（凝血酶-抗凝血酶复合物） | <4 ng/ml | 凝血酶生成增多；敏感度高 |
| **PIC**（纤溶酶-抗纤溶酶复合物） | <0.8 µg/ml | 纤溶酶活化最敏感指标；6h半衰期；纤溶亢进时↑；PIC不高+D-二聚体↑→纤溶抑制 |
| **t-PAIC**（t-PA-PAI-1复合物） | 男<17.0 女<10.5 ng/ml | 内皮损伤+纤溶系统激活；评价脓毒性/心源性休克严重程度 |

### 临床应用场景

| 场景 | TM | TAT | PIC | t-PAIC |
|------|----|----|-----|--------|
| 脓毒症 | ↑ | ↑ | — | — |
| 脓毒症休克 | ↑↑ | ↑↑ | — | ↑↑ |
| 热射病 | ↑↑ | ↑↑ | — | — |
| 肺梗死 | ↑ | ↑↑ | — | — |
| 创伤 | — | ↑↑ | ↑↑ | — |
| 创伤失血性休克 | ↑ | ↑↑ | ↑↑ | ↑ |
| 内科出血 | — | ↑ | ↑↑ | — |
| 白血病性DIC | — | ↑↑ | ↑↑ | — |
| 原发性纤溶亢进 | — | — | ↑↑ | — |
| 溶栓术后 | — | ↑↑ | ↑↑ | — |

**推荐意见 11**（中等推荐，证据等级 C）

推荐使用**抗Ⅹa活性测定试验**评估肝素类药物的相对浓度。

### 原理

发色底物法：肝素类药物与抗凝血酶结合 → 抑制Ⅹa因子活性 → 加入过量Ⅹa后测定剩余活性 → 计算药物浓度（抗Ⅹa单位）

**解读**：
- 未接受肝素治疗者：抗Ⅹa活性 = 0
- 阳性（>0）→ 体内有肝素作用 → 可根据水平决定是否用鱼精蛋白中和
- 抗Ⅹa活性也可用于监测 DOAC（但试剂与肝素不同）

**推荐意见 12**（中等推荐，证据等级 C）

临床医师需对重症凝血病的**出血风险与血栓风险进行综合评估**。

### 风险分级

**出血风险**：

| 等级 | 标准 |
|------|------|
| 低 | 年龄>65岁；肝功能不全（肝酶3倍/胆红素2倍↑）；肾功能不全（肌酐≥200µmol/L或透析/肾移植） |
| 中 | 年龄>75岁；血友病病史；未控制的高血压；近期脑出血/消化道出血病史；长期口服抗血小板药物/抗凝药；PT延长>6s；血小板计数<50×10⁹/L或TEG-MA<43mm；TEG-R>15min或凝血分析仪ACT>280s |
| 高 | 肝衰竭；脑水肿；血小板计数<20×10⁹/L或凝血分析仪PF<1；纤维蛋白原<1g/L；具备两条及以上中风险因素 |

**血栓风险**：

| 等级 | 标准 |
|------|------|
| 低 | 年龄>65岁；肥胖；糖尿病/冠心病史；口服避孕药/激素替代治疗史；腹腔手术史；血小板计数>400×10⁹/L |
| 中 | 长期卧床；实体肿瘤病史；髋/骨盆/下肢骨折；心功能不全；TEG-R<4min或凝血分析仪ACT<100s；TEG-MA>70mm或凝血分析仪PF>4 |
| 高 | 近期脑梗死/肺栓塞/心肌梗死/下肢深静脉血栓/易栓症病史；具备两条及以上中风险因素 |

**评估频率**：至少1次/d；高风险/正在接受溶栓/抗凝/替代治疗时：3-4次/d

**总体决策原则**：以综合获益为根本，以凝血复衡为目标，兼顾出血与血栓风险。

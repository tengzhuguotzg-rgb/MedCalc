# 病因与机制

## 推荐

**推荐意见 2**（强推荐，证据等级 C）

临床医师应注意重症凝血病往往**同时存在多种病因和多种机制**。

### 四大特点

1. **致病因素多**：物理（创伤/高温/低温/辐射）、化学（药物/化合物/中毒）、生物（细菌/病毒/真菌/蛇咬）、免疫（SIRS/免疫抑制）、营养（维生素缺乏）、内环境（酸碱/水电解质紊乱）
2. **基础疾病多**：出血高风险（胃溃疡/痔疮/血友病）、血栓高风险（糖尿病/冠心病）、自身免疫病（狼疮/抗磷脂综合征）、恶性肿瘤
3. **异常机制多**：血管内皮损伤 + 凝血因子缺乏 + 纤溶异常 + 血小板减少/功能异常
4. **医源性因素多**：多种影响凝血功能的药物、侵入式治疗、大量血制品输注

### "同病异制"与"异病同制"

- **同病异制**：一种疾病通过多重机制形成同样的凝血异常表现。例：热射病性凝血病 → 肝功能损害（凝血因子合成↓）+ 微血栓形成（凝血因子消耗↑）+ 过量补液（血液稀释）+ 肝素抗凝（因子活性抑制）→ 均可表现为 APTT 明显延长
- **异病同制**：不同病因通过类似病理生理机制导致同样的实验室表现。例：热射病 APTT 延长 + 并发 SIC → 凝血底物消耗加速 → APTT 延长更明显

---

**推荐意见 3**（中等推荐，证据等级 C）

临床医师应了解**血管、血流、血液三要素的失稳态**是重症凝血病的主要特征。

### 三要素失衡与凝血病

| 要素 | 功能 | 失衡后果 |
|------|------|---------|
| 血管 | 内皮完整性/凝血酶激活 | 内皮损伤 → 促凝物质释放 → 高凝 |
| 血流 | 氧输送/凝血底物运输 | 血流缓慢 → 血栓形成；大出血 → 低凝 |
| 血液 | 凝血因子/血小板/纤溶系统 | 底物耗竭 → 出血；纤溶亢进 → 出血 |

**局部血栓病**（心梗/脑梗）：血管内皮损伤 → 凝血酶高凝 + 血液高黏度 → 局部血流缓慢 → 血栓
**系统性血栓病**（SIC）：PAMP/DAMP → 内皮损伤 → 组织因子释放 + 抗凝↓ + 纤溶抑制 → 血液高凝 → 微血栓 → MOF
**出血型疾病**：血管破裂 → 凝血酶大量活化（早期高凝）→ 未能止血 → 凝血底物丢失 + 纤溶亢进（转为低凝）→ 失血性休克 → 血流恶化 → 凝血病加重

---

**推荐意见 4**（中等推荐，证据等级 C）

临床医师应注重评价重症凝血病中的**免疫紊乱**。

### 免疫血栓理论演化

1. **NETs（2004）**：中性粒细胞释放 DNA + 组蛋白，与纤维蛋白原组成网状结构 → 捕捉/清除病原体；为组织因子、vWF、EV 等促凝成分提供支架 → 促进血栓形成
2. **免疫血栓学说（2013）**：中性粒细胞/单核细胞/巨噬细胞与纤维蛋白原/血小板共同形成免疫血栓 → 控制病原体扩散
3. **免疫凝血聚合理论（2024）**：损伤因素一方面通过 DAMP 激活凝血酶促反应，另一方面通过 NETs 促进免疫血栓形成

**临床意义**：免疫反应失调 → 免疫血栓形成过量 → 血栓炎症 → 重症凝血病。"炎凝交互"提示处理重症凝血病时应同步纠正免疫紊乱。

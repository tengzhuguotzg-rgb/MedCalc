# 分型与分类

## 推荐

**推荐意见 1**（强推荐，证据等级 C）

临床医师应掌握重症凝血病的**表型(phenotype)分型**、**亚型(subtype)分型**和**功能分类**。

### 表型分型（按临床症状）

| 类型 | 特征 |
|------|------|
| 血栓型 | 微血栓形成为主，常见于 SIC 早期 |
| 出血型 | 出血为主，常见于大出血、创伤早期 |

两者可相互转化——SIC 早期血栓型 → 后期凝血底物耗竭 → 转为出血型；创伤早期出血型 → 出血控制后 → 转为血栓型。

### 亚型分型（按凝血状态）

| 类型 | 特征 |
|------|------|
| 高凝血症 | 血液凝固性增高，易致血栓形成 |
| 低凝血症 | 血液凝固性下降，易发生出血 |

**判定方法**：需通过黏弹力凝血试验（TEG/凝血与血小板功能分析仪）。D-二聚体或 TAT 升高仅提示纤溶/凝血酶活化，**不能**据此诊断高凝血症。

### 亚表型分型（结合表型+亚型）

| 亚表型 | 临床特征 | 治疗原则 |
|--------|---------|---------|
| 血栓高凝型 | 血栓性疾病、SIC 早期 | 抗凝治疗 |
| 血栓低凝型 | SIC 晚期 | 评估出血风险后抗凝，滴定剂量 |
| 出血高凝型 | 出血量不大/速度不快 | 局部压迫止血为主，慎防血栓 |
| 出血低凝型 | 大出血 | 止血 + 替代治疗 |

### 功能分类（按严重程度）

| 阶段 | 定义 |
|------|------|
| 凝血障碍 | 血液凝固能力下降，具有出血倾向；实验室异常但可无临床出血 |
| 凝血衰竭 | 三要素（血管内皮严重损伤 + 凝血底物严重障碍 + 严重氧代谢障碍/组织低灌注）共同失衡；可表现为皮肤瘀斑、黏膜出血、肢端坏疽、多器官功能障碍 |

凝血衰竭的预测：血浆 TM 水平（血管内皮）+ PT-INR（血液状态）+ 血乳酸（血流状态）→ 构建列线图可准确判断预后。

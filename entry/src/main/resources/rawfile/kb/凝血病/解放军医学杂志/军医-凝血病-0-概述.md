# 概述

> 来源：解放军医学杂志，2025年第50卷第9期，页1055-1069。DOI: 10.11855/j.issn.0577-7402.1480.2025.0820

重症患者凝血障碍发生率40%~67.6%， INR≥1.5发生率可超过66%。一旦发生凝血障碍，出血事件和病死率是正常重症患者的**4倍以上**。

→ 相关：[[噬血细胞性淋巴组织细胞增生症/NEJM/NEJM-HLH-2-临床特征]]（HLH 常见 DIC 等凝血功能异常，属于凝血病谱系）

早期识别凝血障碍并准确评估凝血功能，是尽快纠正凝血障碍的前提。

## 推荐强度分级

| 等级 | 释义 |
|------|------|
| **Ⅰ（强）** | 证据肯定或良好（A-B级）；或证据一般（C-D级）但指南明确推荐，利大于弊 |
| **Ⅱ（中等）** | 证据一般（C-D级），可改善健康结局 |
| **Ⅲ（弱）** | 证据不足或矛盾，但可能改善健康结局 |

**证据等级**：A=多个RCT系统评价/B=高质量RCT或规范观察性研究/C=非随机病例对照或观察性研究/D=回顾性研究或专家共识

## 核心概念图

- **表型分型**：血栓型 ↔ 出血型（可相互转化）
- **亚型分型**：高凝血症 ↔ 低凝血症（需黏弹力凝血试验判定）
- **功能分类**：凝血障碍 → 凝血衰竭（三要素：血管、血流、血液共同失衡）
- **三要素**：血管内皮损伤 + 凝血底物耗竭 + 组织低灌注 = 凝血衰竭

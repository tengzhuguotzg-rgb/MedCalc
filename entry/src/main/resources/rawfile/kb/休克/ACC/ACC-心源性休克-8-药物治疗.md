# 药物治疗

## 8.1 药物治疗总体目标

| 目标 | 说明 |
|------|------|
| **减轻充血** | 存在充血表型时（襻利尿剂 ± 噻嗪类；难治性考虑肾脏替代治疗超滤）|
| **优化心输出量** | 静脉血管活性药物（正性肌力药/升压药）|
| **改善终末器官灌注** | 维持 MAP > 60~65 mmHg 是合理的血流动力学目标 |

> [!warning] 充血不处理的后果
> 不处理充血可导致微循环缺血，损伤肾脏、肝脏和胃肠道。

**跨肾灌注压**（MAP - CVP）是评估肾脏灌注的有用概念模型：CVP 升高意味着肾脏灌注压降低。

## 8.2 药物分类与 Table 2 总表

| 类别 | 代表药物 | 核心机制 | 常用剂量 |
|------|----------|-----------|----------|
| **Inopressor**（升压正性肌力药）| 去甲肾上腺素 | α1(+++) β1(++) | 0.05-1 μg/kg/min |
| | 肾上腺素 | β1(+++) α1(++) | 0.01-0.5 μg/kg/min |
| | 多巴胺 | D1(+++) β1(++) | 低2-5/中间5-10/高10-20 μg/kg/min |
| | 多巴酚丁胺 | β1(+++) β2(++) | 2-10 μg/kg/min |
| **Inodilator**（正性肌力扩血管药）| 米力农 | PDE-3 抑制剂 | 0.125-0.5 μg/kg/min |
| **Vasopressor**（纯升压药）| 去氧肾上腺素 | α1(+++) | 0.1-10 μg/kg/min |
| | 加压素 | 加压素受体 | 0.01-0.04 U/min |
| **Vasodilator**（血管扩张药）| 硝普钠 | NO | 0.3-10 μg/kg/min |
| | 硝酸甘油 | → NO | 25-200 μg/min |
| **Chronotrope**（增心率药）| 异丙肾上腺素 | β1(+++) β2(+++) | 2-20 μg/min |
| **Inotrope** | 左西孟旦* | 与肌钙蛋白C结合，增强钙敏感性 | 0.05-0.2 μg/kg/min |

\*美国未获 FDA 批准临床使用。

### 五类药物的核心机制

| 类别 | 对心输出量影响 | 对 SVR 影响 |
|------|----------------|-------------|
| **纯升压药** | ↓（反射性心动过缓）| ↑↑ |
| **Inopressor** | ↑ | ↑ |
| **Inodilator** | ↑ | ↓ |
| **Vasodilator** | ↑（后负荷降低）| ↓ |
| **Chronotrope** | ↑（心率增加）| ↓ |

> [!warning] 纯升压药（如去氧肾上腺素）作为单一一线用药
> **强烈不推荐**——可引起反射性心动过缓、降低 CS 患者心输出量。

## 8.3 循证证据总结

### Cochrane 分析

无充分证据支持任何特定正性肌力药或扩血管药在 CS 中的**死亡率获益**更优。

### DOREMI 试验

| 项目 | 内容 |
|------|------|
| 比较 | 多巴酚丁胺 vs 米力农 |
| 样本 | 192 例 CS 患者（SCAI B~E，排除心脏骤停）|
| 主要终点 | 住院全因死亡、心脏骤停复苏、心脏移植/MCS、 非致死性 MI/TIA/卒中、肾脏替代治疗的复合终点 |
| 结果 | **无差异** |

## 8.4 临床用药决策建议

| 临床场景 | 药物选择建议 |
|----------|--------------|
| **正常血压性 CS**（SVR 增高）| Inodilator 或扩血管药 |
| **肾功能恶化** | 米力农需谨慎（半衰期长，肾脏排泄）|
| **心动过缓性 CS** | Chronotrope（如异丙肾上腺素）|
| **大多数低血压 CS** | **去甲肾上腺素**是一线合理选择 |
| **需要同时升压和增加CO** | Inopressor（去甲肾上腺素/肾上腺素/多巴胺）|

> [!tip] 一线推荐
> 虽然目前没有关于一线血管活性药物选择的明确共识，但写作委员会同意：**去甲肾上腺素是大多数低血压 CS 患者的合理一线选择**。

### 何时考虑升级/转运

Level 2/3 中心患者对初始药物治疗**难治**时，应**考虑转运至 Level 1 中心**以评估 tMCS 支持和/或高级治疗（持久 LVAD、心脏移植）。

# 初步评估

## 4.1 早期识别与诊断

### 早期诊断的重要性

早期诊断 CS 至关重要，因为及时干预可能直接影响患者预后。

传统 CS 诊断标准（用于研究和注册）包括：低血压 + 终末器官低灌注证据 + 充血或心输出量降低指标（如 PAC 测量）。然而，这些标准将一个动态临床实体**二元化**为"有/无"，忽视了以下情况：

> [!warning] 正常血压性心源性休克
> **正常血压性 CS**（即：终末器官低灌注但血压正常）与**死亡率增加相关**，是临床最容易被忽视的陷阱之一。
>
> 诊断 CS 的关键是：**对有风险的患者怀疑心输出量不足**，然后通过 readily obtainable 的临床、实验室和/或影像学数据加以确认。

### 识别触发条件

出现以下任一情况应**高度怀疑 CS**：

| 类别 | 触发条件 |
|------|----------|
| **血压** | SBP < 90 mmHg 或 MAP < 60 mmHg 或较基线下降 > 30 mmHg > 30 min，或需要药物/机械支持维持 SBP > 90 mmHg |
| **心率** | HR > 100 次/分 |
| **脉压** | 窄脉压（< SBP 的 25%） |
| **体征** | 精神状态改变/意识模糊、四肢湿冷、毛细血管再充盈时间 > 2 秒、尿量减少（< 30 mL/h 或 < 0.5 mL/kg/h） |
| **呼吸** | 呼吸困难、端坐呼吸、Spo2 下降 |
| **消化道** | 恶心、呕吐、腹痛、早饱、食欲下降（提示胃肠道缺血） |

### 实验室检查

| 检查项目 | 意义 |
|----------|------|
| 综合代谢面板（CMP） | 评估急性肾/肝损伤 |
| 静脉或动脉血气 | 代谢性酸中毒、乳酸 > 2 mmol/L |
| 高敏肌钙蛋白 | AMI 排除 |
| NT-proBNP | 心衰评估 |
| 中心静脉血气（Scvo2） | 氧供需评估 |

> [!note] 有创血流动力学
> CS 的**初步怀疑和诊断不需要有创血流动力学数据**；但有创血流动力学对于明确心室受累类型和充血profile、进而指导治疗决策**非常有价值**。

## 4.2 SCAI 分期系统

> [!info] SCAI 分期
> 确诊 CS 后，应使用 **SCAI（Society for Cardiovascular Angiography and Interventions）CS 分期系统**对 CS 严重程度进行分级。

SCAI 分期将 CS 患者分为 **A~E 五级**（从"风险最小"到"垂死"），是风险分层和治疗决策的核心工具。

| 分期 | 名称 | 临床特征 |
|------|------|----------|
| **A** | 有风险 | 有CS风险但目前无低灌注或循环衰竭证据 |
| **B** | 开始恶化 | 已有低灌注或循环衰竭证据（相对稳定） |
| **C** | 经典CS | 需要小-中等强度干预（如低剂量升压药或标准tMCS）|
| **D** | 极端 | 对初始治疗反应差，需要多重或高级干预 |
| **E** | 垂死 | 濒死状态，循环极不稳定 |

> [!warning] SCAI B期患者
> SCAI B期患者在发病后 24 小时内进展至 C/D/E 期的风险**最高**，需要密切监测。

## 4.3 表型分类

除病因外，可按以下维度进一步分类 CS 表型：

| 分类维度 | 类型 |
|----------|------|
| **病因** | AMI-CS / HF-CS / 心脏术后 CS / 继发性 CS（心律失常/瓣膜病/心包疾病/其他）|
| **AMI-CS 细分** | STEMI vs NSTEMI |
| **HF-CS 细分** | De novo HF-CS vs 急性-on-慢性 HF-CS |
| **心室受累** | 左室优势型 / 右室优势型 / 双心室型 |

> [!note] 心室受累与预后
> 双心室受累和右室优势型充血在 CS 患者中**常见**，且与不良预后相关（死亡、需要持久 LVAD 或心脏移植）。

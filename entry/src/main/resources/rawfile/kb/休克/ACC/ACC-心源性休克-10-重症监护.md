# 重症监护

## 10.1 ICU 监测概览（Figure 5）

CS 患者的 ICU 监测需要在全身器官灌注与血流动力学稳定之间持续平衡：

| 监测维度 | 具体内容 |
|----------|----------|
| 血流动力学 | 有创动脉压、CVP、PAC（如使用）、心输出量、混合静脉血氧饱和度 |
| 呼吸 | SpO2、动脉血气、呼吸机参数（VT/PEEP/RR）|
| 肾脏 | 尿量、血肌酐、肾脏替代治疗需求 |
| 肝脏 | ALT/AST、乳酸、凝血功能 |
| 神经系统 | GCS/镇静 Ramsay 评分、精神状态 |
| 感染 | 体温曲线、白细胞计数、培养数据 |

---

## 10.2 系统并发症（Figure 9）

CS 患者全身灌注不足可导致多器官功能障碍（MODS）：

| 系统 | 并发症 | 临床提示 |
|------|--------|----------|
| 肾脏 | 急性肾损伤（AKI） | 尿量 < 0.5 mL/kg/h 持续 6h 以上 |
| 肝脏 | 缺血性肝炎、转氨酶显著升高 | ALT > 200 U/L 或 > 3×ULN |
| 凝血系统 | 弥散性血管内凝血（DIC）| PLT ↓、PT/aPTT 延长、D-二聚体↑ |
| 消化道 | 肠道缺血、应激性溃疡、再喂养综合征 | 胃潴留、腹胀 |
| 呼吸系统 | ARDS（CS 相关肺损伤）| PaO2/FiO2 < 300 |
| 神经系统 | 意识改变、谵妄 | 排除缺氧性脑病 |

## 10.3 机械通气与 PEEP（Figure 10）

### PEEP 对心肺交互的影响

| PEEP 效应 | 临床影响 |
|-----------|----------|
| 增加胸腔内压 | ↓ 静脉回流 → ↓ 前负荷 |
| 升高肺动脉压 | 右室后负荷增加（对 RV 优势型 CS 特别不利）|
| 改善氧合 | 减少低氧性肺血管收缩 |
| 过度 PEEP | 严重降低心输出量 |

### 机械通气策略建议

| 原则 | 具体建议 |
|------|----------|
| 肺保护性通气 | VT 6-8 mL/kg（理想体重），平台压 < 30 cmH2O |
| 避免高 PEEP | 特别是 RV 优势型或双心室型 CS |
| 俯卧位 | 严重 ARDS（PaO2/FiO2 < 150）时考虑 |
| 撤机评估 | SCAI C/D 期稳定后应每日评估呼吸机撤离可能性 |

## 10.4 营养支持

| 项目 | 建议 |
|------|------|
| 启动时机 | 血流动力学稳定后 24-48 小时内启动肠内营养（EN）|
| 首选途径 | 肠内营养（胃肠道有功能时优先）|
| 再喂养综合征 | CS 患者高风险；启动时磷、钾、镁监测 |
| 目标热卡 | 20-25 kcal/kg/d（急性期避免过度喂养）|
| EN 不耐受 | 胃潴留 > 500 mL/24h → 幽门后喂养 |

---

## 10.5 感染预防

| 措施 | 说明 |
|------|------|
| 每日抗生素评估 | CS 患者常需经验性广谱抗生素，明确感染源后 48-72h 内降阶梯 |
| 器械相关感染 | tMCS 置入、静脉导管、导尿管是主要感染途径 |
| 多药耐药菌风险 | ICU 住院时间长、既往抗生素暴露史增加风险 |

## 10.6 tMCS 撤离（Figure 11）

### 撤离前评估五要素

| 评估要素 | 内容 |
|----------|------|
| 血流动力学稳定 | MAP > 60-65 mmHg，血管活性药物剂量已降至最低 |
| 心脏功能恢复 | LVEF 改善（超声心动图）、PAPi > 1.85（RV 评估）|
| 器官灌注改善 | 乳酸下降、尿量恢复、凝血功能改善 |
| 无继续机械支持指征 | 无严重心律失常、无进行性低氧 |
| 团队评估共识 | 多学科团队确认可安全撤离 |

### 撤离步骤

1. **降低 tMCS 流速**（逐步而非骤停）
2. **评估血流动力学耐受性**（观察 15-30 min/次降低）
3. **超声心动图评估**（确认 LV/RV 功能可独立维持灌注）
4. **正式撤机**（按方案拔管 + 压迫止血）
5. **撤机后 24-48h 监测**（关注反弹性低灌注）

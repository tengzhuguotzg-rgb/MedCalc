# 转运

## 5.1 CS 中心分级体系

> [!info] 三级 CS 中心
> 根据当地资源与专业知识，CS 中心分为 **Level 1、Level 2、Level 3 三级**，类比创伤救治系统的分级体系。

| 级别 | 能力 | 关键资源 |
|------|------|----------|
| **Level 1** | CS 全面照护 | 心脏导管室 + 心脏ICU + 高级 tMCS；多专科团队（高级心衰与移植心脏病学、介入心脏病学、心胸外科、重症监护心脏病学）；可能有持久 LVAD 和心脏移植能力 |
| **Level 2** | STEMI/PCI + tMCS | 心脏导管室 + ICU + 血流动力学监测；tMCS（IABP ± pLVAD ± pVAD）；有咨询团队 |
| **Level 3** | CS 初始诊断能力 | ICU（如有）；普通心脏病咨询（如有）；**无 PCI 能力**；**无 tMCS** |

> [!note] 当前差距
> 目前尚无统一的医院 CS 中心分级标准（不同于创伤系统），现有分类主要基于专家共识。

## 5.2 转运决策建议

### 何时考虑转运

> [!warning] 关键触发
> 以下情况应**立即考虑与区域 Level 1 CS 中心联系**：
> - AMI-CS 血运重建（PCI）后仍处于**难治性 CS**
> - 心脏骤停复苏后，神经系统状态不稳定
> - 已启动 ≥ 1 种血管活性药物
> - 考虑使用 tMCS 设备

> [!tip] 早期沟通原则
> 一旦 CS 初步诊断确认，**永远不会"太早"联系区域 CS 中心**——即使还在初始机构就地稳定患者过程中，提前沟通有助于提前准备接收和远程会诊。

### Level 2/3 中心行动建议

1. **识别当地可用资源**
2. **确定区域 CS 中心**（愿意接收适当选择的 CS 患者进行进一步评估和治疗，并可提供远程会诊）
3. **识别现场"CS champion"**（推动当地 CS 照护流程优化，同时与区域 CS 中心协作）

### Level 1 中心行动建议

1. 欢迎来自区域转诊医院的会诊请求
2. 接受适当选择的转运患者
3. 为经评估认为不适合转运或需要进一步稳定后再转运的患者提供**远程会诊**

## 5.3 转运预后预测因素

### HF-CS 转运预后

| 预测因素 | 预后方向 |
|----------|----------|
| 高龄 | 预后更差 |
| 机械通气 | 预后更差 |
| 肾脏替代治疗 | 预后更差 |
| 多种血管活性药物 | 预后更差 |
| **使用肺动脉导管（PAC）** | **预后更好**（提示更积极的血流动力学评估策略） |

### AMI-CS 转运预后

| 预测因素 | 预后方向 |
|----------|----------|
| 超重（BMI > 28 kg/m²） | 死亡率更高 |
| 肾功能恶化 | 死亡率更高 |
| 乳酸 > 3 mg/dL | 死亡率更高 |
| 血管活性药物数量增加 | 死亡率更高 |
| **使用任何 tMCS 设备** | **预后更好** |

> [!note] 证据局限性
> 上述预测因素来自**观察性研究**，证据级别有限。区域化转运网络与 CS 预后关系的研究是当前重要证据缺口。

## 5.4 转运结果数据

| 研究 | 主要发现 |
|------|----------|
| 多中心回顾性研究 | 转运患者总体死亡率较高，但主要由 HF-CS 驱动 |
| 3州区域网络研究（对比 Level 1 vs Level 2/3） | AMI-CS 和 HF-CS 在短期结局（住院死亡率、30天死亡率）及并发症（出血、血管并发症、卒中）方面**结果相当** |

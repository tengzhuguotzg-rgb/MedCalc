# SRLF-SFC 心源性休克管理专家建议

> **快速参考**：41条推荐，6大领域，23个PICO问题  
> **方法学**：GRADE系统，1=强推荐，2=中等推荐，专家意见=弱推荐  
> **发布机构**：法国重症医学学会（SRLF）+ 法国心脏病学会（SFC），联合SFAR + SFCTCV

## 指南概览

**背景**：心源性休克（CS）是一种危及生命的急性心脏衰竭综合征，短期病死率30-40%，一年病死率超过50%。过去十年多项症状治疗研究（包括正性肌力药和机械循环支持）多为阴性结果，缺乏针对病因治疗的随机试验。

**方法学**：采用GRADE方法学，将证据质量与推荐强度分离。强推荐可仅凭弱证据，只要获益显著大于风险。

**推荐分级**：
- Grade 1+/1−：强推荐/强反对（高质量证据）
- Grade 2+/2−：中等推荐（中等质量证据）
- 专家意见：证据不足，基于共识

**6大领域41条推荐分布**：

| 领域 | 推荐数 |
|------|--------|
| 1. 休克团队与专家中心 | R1, R2 |
| 2. 症状性药物治疗 | R3~R9 |
| 3. 病因治疗 | R10A/B/C, R11A/B, R12A/B, R13, R14A/B, R15 |
| 4. 器官支持 | R16A/B/C/D, R17 |
| 5. 临时机械循环支持 | R18A/B/C, R19A/B, R20A/B, R21 |
| 6. 撤机与早期后CS管理 | R22, R23 |

## CS照护组织架构

!!! tip "三级照护网络（Fig.1）"
    - **社区中心**：初始稳定 + 早期转运
    - **专科中心**：非难治性/短时程休克（<24h），多学科团队（心衰专家+介入心脏病专家+重症医学专家）
    - **高级中心**：难治性或长时程休克候选高级治疗（t-MCS、VA-ECMO、LVAD、心脏移植）

**关键原则**：
- 早期识别 + Shock Team激活是及时升级支持的关键
- 难治性CS定义：优化治疗后仍持续低灌注
- SCAI分期系统有效分层预后

## 与ACC 2025心源性休克专家共识的关系

本SRLF-SFC指南与[[休克/ACC/ACC-心源性休克-0-概述]]均为2025-2026年发布的最新CS指南，但有以下关键差异：

| 维度 | SRLF-SFC | ACC 2025 |
|------|----------|----------|
| 地域 | 法国/欧洲 | 美国 |
| 方法学 | GRADE（结构化） | ACC方法学 |
| 证据分级 | 1+/1−/2+/2−/专家意见 | Level 1/2/3 |
| t-MCS定位 | 需MDT团队决策 | 强调早期介入 |
| 病因治疗 | 瓣膜病详细（AS/AR/MR） | ACS为主 |

### 共同核心推荐
- 去甲肾上腺素作为一线血管活性药
- 多学科Shock Team + 区域网络
- AMI-CS早期血运重建
- t-MCS需谨慎选择、MDT讨论

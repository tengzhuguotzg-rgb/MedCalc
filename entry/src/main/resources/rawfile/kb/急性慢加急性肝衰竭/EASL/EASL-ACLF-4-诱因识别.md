# 诱因识别

## 诱因定义

**ACLF诱因**：急性肝内或肝外损伤，可直接或间接导致器官功能障碍。

- 诱因可是临床明显的（如确诊的微生物感染伴脓毒症、重度酒精性肝炎）
- 也可以无明显诱因

**PREDICT研究**（目前唯一专门设计识别ACLF诱因的前瞻性研究）显示：420例ACLF患者中，273例至少有1个临床明显的诱因，147例无明显诱因。

---

## 推荐意见

### 1. 系统评估流程

**推荐 1**
> 所有因ACLF入院或住院期间发生ACLF的患者，均应接受系统评估（Fig. 3），以识别最常见的诱因：确诊的细菌感染、重度酒精性肝炎、消化道出血伴低血容量性休克、HBV再激活、HEV感染、可导致脑衰竭的近期用药史、可导致肾衰竭的近期用药史（LoE 2，强推荐，强共识）。

**支持文本**：PREDICT研究显示，最常见的单一诱因是确诊的细菌感染和重度酒精性肝炎；最常见的诱因组合是重度酒精性肝炎合并细菌感染。消化道出血伴休克作为单一诱因者仅6例（2%，共273例有明确诱因）。药物诱发的急性脑病从未作为单一诱因出现，总是伴随其他诱因。

系统评估识别出的诱因数量具有重大预后价值：≥2个诱因的患者90天死亡风险高于仅有1个诱因或无明确诱因的患者。

---

### 2. 诱因不明确时的评估

**推荐 2**
> 对于系统评估未能识别出预期诱因的患者，应根据临床背景，结合所有潜在罕见诱因的完整清单（Table 4）进行逐案评估（LoE 5，强推荐，强共识）。

**支持文本**：系统评估在35%的ACLF患者中未能识别出诱因。针对这部分患者，专家组根据文献资料和临床经验整理了Table 4中的罕见诱因清单，涵盖肝外病毒感染（EBV、CMV、HSV、VZV、HIV、SARS-CoV-2、流感）、寄生虫感染（内脏利什曼病）、外科或放射学干预、肝内病毒再激活（HBV、HDV、HAV、HEV、HCV）、药物性肝损伤、Wilson病、AIH发作、缺血性肝炎等。实际识别任何罕见诱因均取决于临床背景，特别是衰竭器官的谱系和诱因的已知频率。

---

## 声明（Statements）

- ACLF的诱因是急性肝内或肝外损伤，可能导致器官功能障碍（LoE 2，强共识）
- 同时存在的诱因数量是ACLF患者短期预后的主要决定因素（LoE 2，强共识）

## 常见诱因（Table 4摘要）

### 细菌感染（确诊）

| 感染类型 | 诊断标准 |
|---------|---------|
| 自发性细菌性腹膜炎（SBP） | 腹水中性粒细胞 ≥ 250/mm³ |
| 自发性细菌性脓胸 | 胸水无肺炎证据 + 中性粒细胞 > 500/mm³或培养阳性 |
| 自发性/继发性菌血症 | 自发性：血培养阳性且无明确感染源；继发性：导管相关感染或侵入性操作24小时内发生 |
| 尿路感染 | 尿沉渣异常（>10个白细胞/视野）+ 尿培养阳性 |
| 肺炎 | 感染临床表现 + 胸部影像新发浸润 |
| 支气管炎 | 感染临床表现 + 无浸润 + 痰培养阳性 |
| 皮肤软组织感染 | 红肿热痛伴肿胀 |
| 胆管炎 | 胆汁淤积 + 右上腹痛/黄疸 + 影像学胆道梗阻证据 |
| 继发性腹膜炎 | 腹水中性粒细胞 ≥ 250/mm³（常≥10,000/mm³）+ 低葡萄糖、高蛋白、LDH升高或多微生物感染 |
| 艰难梭菌感染 | ≥3次不成形便 + 产毒艰难梭菌阳性 |

### 真菌感染

| 类型 | 诊断 |
|------|------|
| 侵袭性念珠菌病 | 念珠菌属血培养阳性（念珠菌血症）或来自正常无菌体液（腹水、胸水） |
| 拟侵袭性曲霉菌病 | 呼吸道样本直接镜检/培养阳性 + 影像学 compatible |

### 重度酒精性肝炎

**诊断标准**（NIAAA criteria，无肝活检时）：
- 血清胆红素 > 3 mg/dl (> 50 μmol/L)
- AST > 50 IU/ml，AST/ALT 比值 > 1.5，AST和ALT < 400 IU/ml

**肝活检**：大泡性脂肪变性伴中性粒细胞浸润、肝细胞气球样变、Mallory-Denk小体

### 消化道出血伴休克

呕血、黑便、血红蛋白低水平或快速下降（≥ 2 g/dl）、低血容量性休克证据 + 内镜确诊

### 药物相关

| 类型 | 诊断标准 |
|------|---------|
| **药物相关脑损伤** | 近期服用镇静剂史（主要为苯二氮卓类或阿片类） |
| **药物相关急性肾损伤** | 近期服用肾毒性药物史：NSAIDs、RAAS拮抗剂、α1受体阻滞剂、静脉造影剂、肾毒性抗生素（万古霉素、氨基糖苷类） |

## 罕见诱因（Table 4续）

### 肝外病毒感染

| 病毒 | 诊断标准 |
|------|---------|
| EBV | AST/ALT > 3× ULN；VCA-IgM、EA-D、EBNA抗体；EBV定量PCR |
| CMV | AST/ALT > 3× ULN；CMV IgG抗体；CMV定量PCR |
| HSV 1/2/6 | AST/ALT > 1,000 IU/ml；HSV 1/2 IgM抗体；HSV定性PCR |
| VZV | AST/ALT > 1,000 IU/ml；VZV IgM抗体；定性PCR |
| HIV | AST/ALT轻度升高；HIV-1/-2抗体；定量PCR |
| 细小病毒B19 | AST/ALT > 3–5× ULN；细小病毒B19 IgM；定性PCR |
| SARS-CoV-2 | AST/ALT轻度升高；呼吸道样本PCR或快速抗原阳性 |
| 流感A/B/RSV | AST/ALT轻度升高；呼吸道样本PCR阳性 |

### 寄生虫感染

**内脏利什曼病**：AST、ALT、ALP、GGT升高；利什曼原虫或DNA检测（骨髓 aspirate > 淋巴结 > 肝活检）；血清IgG/IgM阳性

### 外科/介入

**手术或放射学干预**：7天内手术或侵入性放射学干预

### 肝内病毒感染

| 类型 | 诊断标准 |
|------|---------|
| HBV再激活 | HBsAg阳性或HBc抗体阳性 + HBV DNA升高 |
| HDV叠加（慢HBV） | AST/ALT > 400 IU/ml；HDV IgM和IgG阳性；HDV RNA升高 |
| HAV叠加 | AST或ALT > 400 IU/ml；胆红素 > 3 mg/dl；抗HAV-IgM阳性 |
| HEV叠加 | AST/ALT > 400 IU/ml；胆红素 > 3 mg/dl；抗HEV-IgM（和IgG）；HEV RNA |
| HCV叠加 | AST或ALT > 400 IU/ml；胆红素 > 3 mg/dl；HCV RNA升高 |

### 其他

| 诱因 | 诊断标准 |
|------|---------|
| **药物性肝损伤（DILI）** | 近期服用肝毒性化合物史；ALT ≥ 5× ULN或ALT ≥ 3× ULN + ALP ≥ 2× ULN + 胆红素 > 2× ULN |
| **Wilson病** | Leipzig评分系统（≥4分）；高胆红素（≥10 mg/dl，主要为间接）；Coombs阴性溶血；AST/ALT比值 > 2.2；低ALP；严重凝血障碍；轻-中度肝性脑病；低血清铜蓝蛋白（< 20 mg/dl）；尿铜 > 100 μg/24h（通常 > 500 μg） |
| **AIH发作** | 免疫抑制治疗不依从或减量史；AST、ALT升高；高丙种球蛋白血症；IgG升高；ANA ≥ 1:80、SMA、抗SLA/LP（1型）；抗LKM1/3、抗LC-1（2型）；肝活检（急性AIH时强制要求） |
| **缺血性肝炎** | AST/ALT峰值极高（通常 > 1,000 IU/ml）；胆红素通常 < 3 mg/dl；INR快速升高后迅速改善；腹部超声确认血管通畅；超声心动图评估右/左心室功能 |

## PREDICT研究关键数据

- **420例ACLF患者**：273例有≥1个临床明显诱因，147例无明显诱因
- **单一最常见诱因**：确诊细菌感染、重度酒精性肝炎
- **最常见组合**：重度酒精性肝炎 + 确诊细菌感染
- **胃肠道出血伴休克**：仅2%（6/273）是单一诱因
- **药物诱发脑病**：从未作为单一诱因，总是伴随其他诱因
- **≥2个诱因**：90天死亡风险显著高于单一诱因或无诱因者

# 预测与诊断评估

## 1. 风险预测模型

**Rec 2.1.1**：建议使用经外部验证的临床风险模型识别 AKI 高风险儿童和成人，以指导肾脏保护策略（**2C**）。

**Rec 2.1.2**：推荐使用经外部验证的临床风险模型识别 RRT 高风险成人，以指导患者选择和共同决策（**1C**）。

**AKI 风险分层框架**：

风险由**易感因素（susceptibilities）**和**暴露因素（exposures）**共同决定，两者可呈非线性交互：

| 常见暴露因素 | 常见易感因素 |
|-------------|-------------|
| 脓毒症（社区/医院获得性） | CKD |
| 重大手术（心脏、腹部等） | 糖尿病 |
| 循环性休克 | 慢性心脏病（含先天性心脏病） |
| 肾毒性药物 | 慢性肝病 |
| 放射对比剂 | 贫血 |
| 机械通气 | 高龄 |
| 器官移植 | ICU 住院 |

**Practice Point 2.1.1**：在社区和低资源环境中，简化 AKI 风险评分可支持高风险成人的早期识别和管理。

**Practice Point 2.1.2**：EHR-AKI 风险模型应在部署环境中测试；统一术语和数据结构，并定期重新评估以确保本地环境的预测准确性。

---

## 2. 电子 AKI 预警

**Rec 2.2.1**：推荐在成人日常临床护理中，**不单独发送基于 SCr 的插断式电子 AKI 预警**，而无系统性应对策略（**1B**）。

> 单独预警不结合系统性应对流程，反而可能增加医护负担而不改善预后。

## 3. 肾功能动态检测

### 呋塞米负荷试验（FST）

**Rec 2.3.1**：建议使用呋塞米负荷试验（FST）评估 euvolemic 或 hypervolemic 的1-2期 AKI 进展至3期（含 RRT）风险（成人2C；儿童2B）。

FST 方法：静脉推注呋塞米 1.0-1.5 mg/kg（未使用利尿剂者），2小时后测量尿量：

- **<200 ml/h** → FST 阳性，高风险进展至3期 AKI
- ≥200 ml/h → FST 阴性，低风险

**支持证据**：FST 在心脏手术和 ICU 危重成人中预测 AKI 3期（含 RRT）的 AUROC 可达 0.8以上。

### 动态 eGFR（KeGFR）

**Rec 2.3.2**：建议在住院成人（伴或不伴 AKI 风险）中，当**非稳态 GFR 估计值**将影响临床决策时计算 KeGFR（**2B**）。

**Practice Point 2.3.1**：暴露前评估肾功能储备有助于识别 AKI 高风险人群。

## 4. 生物标志物风险分层

### 主要生物标志物一览

| 标志物 | 类型 | 临床应用 | 特点 |
|--------|------|---------|------|
| **血清胱抑素 C** | 功能 | AKI 早期识别 | 不受肌肉质量影响；eGFR 估计更准 |
| **[TIMP-2]*[IGFBP7]** | 应激 | ICU 成人中重度 AKI 风险 | FDA+CE 批准；AWC 0.82；4-12小时达峰 |
| **NGAL** | 损伤 | 医院内中重度 AKI | 10-20分钟内升高；2-4小时达峰 |
| **KIM-1** | 损伤 | 药物相关 AKI 鉴别 | 12-24小时达峰；FDA 批准用于药物监测 |
| **L-FABP** | 损伤 | AKI 诊断与严重程度 | 4-6小时达峰；日本批准用于糖尿病 Tubular injury |
| **CCL14** | 炎症 | 持续性3期 AKI（>72h） | 36小时内升高；CE 批准 |
| **DKK3** | 损伤 | 心脏手术后 AKI 风险 | 12-48小时达峰 |

**Rec 2.4.1**：在急性照护环境中，建议根据临床验证和监管批准的适应证，使用肾小管应激/损伤生物标志物早期识别中重度 AKI 高风险人群（成人**2B**；儿童**2C**）。

**Practice Point 2.4.1**：基线白蛋白尿（UACR）对成人 AKI 风险评估有额外预后价值。

**Practice Point 2.4.2**：肾应激生物标志物（如尿 [TIMP-2]*[IGFBP7]）可用于评估 ICU 前12小时或高风险事件（重大手术、脓毒症）后中重度 AKI 风险。

**Practice Point 2.4.3**：肾损伤生物标志物（如尿 NGAL）可用于识别医院内中重度 AKI 高风险儿童和成人。

## 5. AKI 病因与诊断评估

### 尿镜检

**Rec 2.5.1**：建议在成人 AKI 诊断评估中，尤其病因不清或怀疑原发性肾病时，将尿镜检纳入（**2D**）。

| 尿沉渣发现 | 临床意义 |
|-----------|---------|
| **泥棕色颗粒管型** | 提示急性肾小管损伤（ATN）；预测 AKI 恶化 |
| **肾小管上皮细胞** | 鉴别 ATN 与血流动力学性 AKI |
| 活动性尿沉渣（细胞、管型） | 与 CKD 进展和死亡率增加相关 |

### 其他尿液检查

**Practice Point 2.5.2**：尿嗜酸性粒细胞检测**敏感性低**（不能用于排除 AIN）。

**Practice Point 2.5.3**：尿炎性细胞因子（如 IL-18、IL-9）有助于鉴别成人 AIN 与其他 AKI 病因。

**Practice Point 2.5.4**：**FeNa** 和 **FeUrea** 仅限用于特定场景（区分血流动力学性 AKI 与 ATI），总体临床效用有限。

**Practice Point 2.5.5**：肝硬化相关 AKI 中，FeNa 可辅助区分 HRS 与 ATI。

**Practice Point 2.5.6**：肝硬化相关 AKI 中，尿/血浆 NGAL 可辅助区分 ATI 与 HRS（尤其临床判断困难时）。

### 影像学

**Practice Point 2.5.7**：病因不明时**肾活检**应考虑进行（预期结果将影响管理决策时）。

**Practice Point 2.5.8**：肾脏多普勒超声（含肾阻力指数 RRI）可辅助 AKI 风险分层和严重程度评估。

**Practice Point 2.5.9**：床旁超声（POCUS）可辅助评估容量状态和鉴别容量反应性 AKI vs 固有 AKI。

## 6. 儿童特殊考量

**Practice Point 2.6.1**：社区获得性 AKI 高风险儿童在急性病期间应进行肾脏健康评估。

**Rec 2.6.1**：建议在危重儿童中使用**肾绞痛指数（RAI）**识别严重 AKI 和 RRT 高风险儿童（**2B**）。

**RAI 评分要素**：

- 易感因素（入住 PICU、eGFR 下降）
- 暴露因素（液体正平衡、机械通气、血管活性药物）

评分 ≥8 提示 AKI 高风险。

# 定义与分期

## 1. AKI 定义

**Practice Point 1.1.1**：急性肾损伤（AKI）指在7天内出现一项或多项肾功能或结构异常，且处于相应临床背景下（表1）。

### 诊断标准（表1）

满足以下**任意一项**即可诊断 AKI：

**功能性标准**

| 标准 | 时间窗 |
|------|--------|
| SCr 升高 ≥0.3 mg/dl（≥26.5 µmol/l） | 48小时内 |
| SCr 升高 ≥1.5倍基线值 | 已知或推测在7天内发生 |
| 血清胱抑素 C 升高 ≥1.5倍基线值 | 已知或推测在7天内发生 |
| 平均尿量 <0.5 ml/kg/h（基于理想体重） | ≥6小时 |

**结构性标准**

| 标准 |
|------|
| 肾损伤生物标志物升高（用于临床验证和监管批准的适应证），已知或推测在7天内发生 |

---

### 血清胱抑素 C 的应用（Rec 1.1.1）

**建议**：在临床情况下 SCr 不够准确时，建议使用血清胱抑素 C 识别 AKI（**2B**）。

**胱抑素 C 优于 SCr 的具体临床情境（表5）**：

- **肌肉质量变化**：长期疾病消耗、神经肌肉疾病、截瘫、素食/低蛋白饮食、 creatine 补充
- **药物干扰**：西咪替丁、甲氧苄啶（抑制肌酐肾小管分泌）
- **特殊情况**：极端运动员、进食障碍、肌肉萎缩

胱抑素 C 在 SCr 上升前6-48小时即可检测到 GFR 下降，敏感性优于 SCr（12-24小时 vs 12-72小时）。

---

### 基线 SCr 的确定

**Practice Point 1.1.2**：所有疑似或确诊 AKI 患者均应确定基线 SCr 值。

**Practice Point 1.1.3**：成人 AKI 事件基线 SCr 或胱抑素 C 采用分层方法选择（图1）：

1. **首选**：入院前门诊 SCr（8-365天内的均值、中位数或最近值）
2. **次选**：入院时 SCr（若入院前值不可及，且无社区获得性 AKI 征象）
3. **备选**：假设基线 eGFR = 75 ml/min/1.73 m²，反推 SCr（CKD-EPI 方程优先于 MDRD）

> ⚠️ 不同基线确定方法会导致 AKI 的过度或不足诊断，影响流行病学研究的可比性。

## 2. AKI 分期

**Practice Point 1.2.1**：AKI 严重程度分期采用 SCr、尿量和生物标志物的个体化组合，按最高（最重）等级进行分期（表6）。

### KDIGO 2026 AKI 三维分期系统

| 分期 | SCr 标准（C） | 尿量标准（U） | 标志物（B） |
|------|--------------|--------------|-----------|
| **C0/U0/B0** | 无升高 | 无减少 | 阴性 |
| **C1** | 升高 ≥0.3 mg/dl 或 1.5-1.9倍基线 | <0.5 ml/kg/h 持续6-12小时 | B1（阳性）/ B0（阴性） |
| **C2** | 升高 2-2.9倍基线 | <0.5 ml/kg/h 持续 >12小时 | B1 或 B0 |
| **C3** | 升高 ≥3倍基线；或 SCr ≥4.0 mg/dl（≥353.6 µmol/l）持续 >12小时 | <0.3 ml/kg/h 持续 >24小时或无尿 | B1 或 B0 |
| **U1** | — | <0.5 ml/kg/h 持续6-12小时 | — |
| **U2** | — | <0.5 ml/kg/h 持续 >12小时 | — |
| **U3** | — | <0.3 ml/kg/h 持续 >24小时或无尿 | — |
| **RRT** | 启动 RRT（等同于 C3） | — | — |

> **注**：C1-C3 = SCr 标准分期；U1-U3 = 尿量标准分期；B0/B1 = 损伤生物标志物阴性/阳性。分期按所收集到的指标中最严重的一项确定。

**分期说明**：

- SCr 和尿量标准之间的一致性较差：51.7%的 AKI 患者若仅靠 SCr 会漏诊，其中30.2%会因加入尿量标准而提高分期
- 尿量标准中加入 B1（损伤标志物阳性）可进一步改善预后风险分层
- NGAL 阳性 vs 阴性患者的 RRT 启动率、ICU 住院时间和院内死亡率均更高

## 3. 瞬时性与持续性 AKI

**Practice Point 1.2.2**：根据 SCr 或胱抑素 C 升高或尿量减少的**持续时间**鉴别：

| 类型 | 持续时间 | 临床意义 |
|------|---------|---------|
| **瞬时性 AKI** | ≤48小时 | 通常因肾脏灌注不足引起，快速纠正预后较好 |
| **持续性 AKI** | >48小时 | 与 CKD 和死亡率显著增加相关 |

> 48小时标准优于72小时：早期评估最具临床意义，且与现有大部分研究的证据一致。

## 4. AKI 病因识别

**Practice Point 1.2.3**：应根据易感因素（susceptibilities）和暴露因素（exposures）、临床背景、病史和体格检查，以及适当的实验室和影像学检查确定 AKI 病因，以指导特异性治疗。

**需要特异性治疗的 AKI 病因（表8）**：

| 病因 | 诊断检查 | 治疗 |
|------|---------|------|
| 肾脏灌注减少 | 容量评估、FeNa/FeUrea | 液体/血管活性药物、超滤减负、解除腹内压升高 |
| 急性肾小球肾炎/血管炎/血栓性微血管病 | 尿沉渣、血清学、肾活检 | 激素、免疫抑制、血浆置换 |
| 尿路梗阻 | 肾脏超声或CT | 泌尿/放射学干预解除梗阻 |
| 急性间质性肾炎 | 尿检、嗜酸细胞、肾活检 | 停用致病原、考虑激素 |
| 肝肾综合征 | 有效血容量评估、尿钠 | 血管收缩素（特利加压素/去甲肾上腺素）、肝移植 |
| 多发性骨髓瘤 | 血尿单克隆蛋白 | 水化、化疗 ± 血浆置换 |
| 横纹肌溶解 | CK/肌红蛋白、尿 Rous 试验 | 液体治疗 |
| 肿瘤溶解综合征 | LDH、尿酸、钙、磷酸盐 | 黄嘌呤氧化酶抑制剂（别嘌醇/非布司他）、rasburicase |

## 5. AKD 定义

**Practice Point 1.3.1**：急性肾脏病（AKD）指在3个月内出现以下任意一项功能或结构性标准（表9）：

### AKD 诊断标准（表9）

满足以下**任意一项**，持续时间 ≤3个月：

| 类型 | 标准 |
|------|------|
| 功能标准 | AKI（功能性标准）；或 GFR <60 ml/min/1.73 m²；或 GFR 较基线下降 ≥35 ml/min/1.73 m²；或 SCr 升高 >50% |
| 结构标准 | 肾损伤标志物（白蛋白尿、血尿或白细胞尿最常见） |

> ⚠️ GFR <60 ml/min/1.73 m² 标准不适用于2岁以下儿童。

**AKD 临床意义**：

- AKD 患病率约为 AKI 的3倍
- 无 AKI 的 AKD 全因死亡率 OR = 3.8（95%CI 2.2-6.7）
- 无 AKI 的 AKD 与新发 CKD 风险显著增加相关
- 在社区/基层医疗中更常见

---

## 6. AKI 与 AKD 的鉴别与消退

**Practice Point 1.3.2**：无 AKI 的 AKD 与 AKI 后的 AKD 应相互鉴别——两者的病因和临床处理策略不同。

### AKI/AKD 消退标准（表10）

|  | 完全缓解 | 部分缓解 |
|--|---------|---------|
| **AKI** | 7天内 SCr 或胱抑素 C 恢复至 <1.2倍基线 | 7天内 SCr 或胱抑素 C ≥1.2倍但 <1.5倍基线 |
| **AKD** | 3个月内 SCr 或胱抑素 C 恢复至 <1.2倍基线，**或** eGFR >80%基线 | 3个月内 SCr 或胱抑素 C ≥1.2倍但 <1.5倍基线，**或** eGFR <80%但 >66%基线 |

**消退定义的临床意义**：

- SCr 恢复至 <1.2倍基线表明大多数患者 GFR 已恢复至基线
- 部分缓解者90天内肾脏不恢复率：完全缓解3.5%、部分缓解10.4%、无部分/完全缓解30.1%
- 超过7天未缓解的 AKI 即进入 AKD 期（持续至3个月）
- 3个月后仍未缓解者诊断为 CKD

---

## 7. 复发性 AKI

**Practice Point 1.5.1**：复发性 AKI 定义为在前一次 AKI 部分或完全缓解后出现新的 AKI 发作。

## 9. 新生儿 AKI 标准

### 新生儿 AKI 诊断标准（表12）

满足以下**任意一项**：

- 出生7天内 SCr 无下降
- 48小时内 SCr 升高 ≥0.3 mg/dl（≥26.5 µmol/l）
- SCr 已知或推测在7天内升高 ≥1.5倍基线
- 血清胱抑素 C 升高 ≥1.25倍基线（已知或推测在7天内）
- 血清胱抑素 C >2.2 mg/dl
- 尿量 <1.0 ml/kg/h 持续 ≥24小时

### 新生儿 AKI 分期（表13）

| 分期 | SCr 标准 | 尿量标准 |
|------|---------|---------|
| C1 | ≥0.3 mg/dl（26.5 µmol/l）升高，**或** 1.5-1.9倍基线 | >0.5且 ≤1.0 ml/kg/h 持续24小时 |
| C2 | 2-2.9倍基线 | >0.3且 ≤0.5 ml/kg/h 持续24小时 |
| C3 | ≥3倍基线；或 SCr ≥2.5 mg/dl（220 µmol/l）；或启动 RRT | ≤0.3 ml/kg/h 持续24小时 |

> ⚠️ 新生儿分期不包含生物标志物（B0/B1）维度，与成人不同。

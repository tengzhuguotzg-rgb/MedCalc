# 随访

## 1. 随访计划制定

**Rec 6.1.1**：推荐对 AKI 后儿童和成人进行协调随访，个性化为个体临床背景定制（**1B**）。

**Practice Point 6.1.1**：从医院向家庭过渡时需要：住院和门诊照护团队之间清晰沟通、指定 AKI 随访医护人员、提供患者参与和教育资源（肾脏健康和肾脏保护知识）。

**Practice Point 6.1.2**：为 AKI 儿童和成人提供**出院小结**，内容包含：AKI 发作详情、AKI 缓解状态、用药管理和随访计划建议。

**Practice Point 6.1.3**：AKI/AKD 相关疾病后儿童和成人的综合临床评估包括：血压（BP）和容量状态评估、用药协调/审查，以及**出院后肾脏功能和蛋白尿评估**。

**Practice Point 6.1.4**：为有 AKI 和/或 AKD 病史的儿童和成人（及其照护者）提供肾脏健康知识、疾病管理指导和肾脏照护教育，以及**心理社会支持**，以增强功能状态和生活质量。

**Rec 6.1.2**：建议在出院时使用**验证工具**对 AKI 存活者进行风险分层**，识别 CKD 高风险儿童和成人**，使其可从肾脏病照护中获益（**2D**）。

---

## 2. AKI/AKD 后药物管理

**Rec 6.2.1**：推荐在无禁忌证的情况下，依据**循证指南适应证**，启动或恢复 RASi 治疗（成人 **1B**；儿童 **1D**）。

**Practice Point 6.2.1**：评估在儿童和成人中启动或恢复 **SGLT2i、GLP-1 RA 和盐皮质激素受体拮抗剂（MRA）** 的临床适应证。

**Practice Point 6.2.2**：在出院时和 AKI/AKD 后 **3 个月**评估启动或恢复指南适应证药物的适宜性。

**Practice Point 6.2.3**：AKI/AKD 后 RASi、MRA 或利尿剂处方和管理应根据 **KDIGO 2024 CKD 评估与管理临床实践指南**执行。

---

## 3. CKD 评估

**Practice Point 6.3.1**：AKI 或 AKD 后 **3 个月**评估肾脏功能和肾脏损伤标志物，以评估是否缓解，以及是否新发或加重既有 CKD。

**Practice Point 6.3.2**：在 SCr 准确性较低的情况下，可使用**基于血清胱抑素 C 的 GFR 估算**评估儿童和成人 AKI 后 CKD。

---

## 4. AKI 后门诊透析患者出院后照护

**Practice Point 6.4.1**：透析项目应纳入**综合透析照护和沟通计划**，为 AKI 后接受 RRT 的儿童和成人量身定制，依据其透析脱管的预期可能性。

**Practice Point 6.4.2**：在从医院向社区过渡之前，**让患者、照护者和门诊透析照护团队参与**制定出院后照护计划。

**Practice Point 6.4.3**：AKI 后接受 RRT 的儿童和成人透析照护需要：容量状态临床评估、实验室评价肾脏恢复证据、用药审查与协调，以及**患者教育**（识别肾脏功能恢复信号和减少肾脏损伤）。

**Practice Point 6.4.4**：为成人、儿童和照护者提供教育，并使其参与**共同决策**，重点关注自我照护、短期和长期永久 RRT 和死亡风险、生活质量和照护目标，以指导透析照护决策。

**Practice Point 6.4.5**：门诊透析停止后，**肾脏病学家应在 7 天内**进行综合评估，并在 **90 天**后进行额外的肾功能评估。

---

## 5. 儿童特殊推荐

### 随访计划

**Practice Point 6.5.1.1**：在出院时评估 AKI 后儿童的**整体功能状态**；在门诊随访时评估**生长参数和功能状态**。

**Practice Point 6.5.1.2**：既往服用 RASi 的儿童，AKI 和/或 AKD 缓解后应恢复用药，**除非存在禁忌证**。

### 门诊透析儿童

**Practice Point 6.5.2.1**：可采用**基于风险的方法**（根据 AKI 病因和严重程度以及出院时和 AKI 发作 1 年后的肾脏健康评估）确定儿童未来肾脏病随访需求。

**Practice Point 6.5.2.2**：从 AKI 治疗（接受 RRT）中恢复的儿童需要**个体化随访**，包括适合年龄的功能和发育评估、生长和神经发育监测，以及肾脏科、儿科和相關亚专科之间的整合照护。

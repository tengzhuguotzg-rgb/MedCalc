# 肾脏替代治疗

## 1. 启动时机

**Rec 5.1.1**：推荐对成人 AKI，采用**延迟启动策略**（deferred timing）而非早期（预防性）启动 RRT（**1C**）。

**Practice Point 5.1.1**：对儿童和成人 AKI，应与患者、家属、照护者及医护人员共同决策 RRT 的利弊。

**Practice Point 5.1.2**：对 AKI 高概率需 RRT 的儿童和成人，**在达到 RRT 指征前**即应考虑规划 RRT，包括向有肾脏病和 RRT 服务的中心转诊（条件允许时）。

**Practice Point 5.1.3**：AKI 儿童和成人出现**医学上难治的液体、电解质、酸碱和/或代谢并发症**（如尿毒症），若与患者偏好和照护目标一致，应启动 RRT。

**Practice Point 5.1.4**：对于因急性或慢加急性肝衰竭导致**肝性脑病（≥2级）**的患者，即使无传统 RRT 指征，亦可考虑启动 CRRT 控制高氨血症。

**Practice Point 5.1.5**：对危重儿童和成人 AKI，若尿毒症和/或代谢并发症可能干扰神经功能评估或影响苏醒机会，可启动**限时 RRT 试验**。

**Practice Point 5.1.6**：对 AKI 儿童和成人，尤其是接受血管活性药物支持者，RRT 启动时应识别和处理可纠正的血流动力学不稳定因素。

---

## 2. RRT 模式选择

**Practice Point 5.2.1**：儿童和成人 AKI 初始 RRT 模式（CRRT、IRRT 或急性腹膜透析）的选择应综合考虑**患者特征、临床背景、当地专业能力和资源配置**。

**Rec 5.2.1**：对**血流动力学不稳定**的危重成人 AKI，建议使用 CRRT 或急性 PD，而非传统 IRRT（**2C**）。

**Practice Point 5.2.2**：对**急性脑损伤**或其他颅内压增高/脑水肿风险或已存在此类问题的儿童和成人 AKI，**CRRT 优于传统 IRRT**。

**Practice Point 5.2.3**：对 AKI 儿童和成人，尤其是需要血管活性支持者，识别并处理 RRT 期间血流动力学不稳定的可纠正因素。

**Practice Point 5.2.4**：**延长 IRRT（PIRRT）**可作为血流动力学不稳定的危重成人 AKI 初始 RRT 模式的可行替代。

**Practice Point 5.2.5**：当儿童和成人 AKI 患者**生理和代谢上能够耐受 IRRT**，且颅内高压或脑水肿风险（如适用）已消除时，可考虑从 CRRT 过渡到 IRRT。

---

## 3. RRT 剂量强度

**Rec 5.3.1**：推荐对接受 CRRT 的成人 AKI，处方** effluent volume 20-25 ml/kg/h**（优于更高剂量）；对接受 IRRT 或 PIRRT 的成人，处方 **Kt/V 每周 3.9**（优于更高尿素清除率）（**1B**）。

**Practice Point 5.3.1**：对**高体重指数（BMI ≥30 kg/m²）**成人 CRRT，初始剂量强度可基于理想体重或调整体重计算，而非实际体重，以避免过量 CRRT 剂量。

**Rec 5.3.2**：推荐对脓毒症或脓毒性休克危重成人 AKI，**不处方大剂量血液滤过（HVHF）**（优于标准 CRRT 剂量强度）（**1B**）。

**Practice Point 5.3.2**：应每日审查 CRRT 的 RRT 剂量强度是否充分纠正电解质、酸碱和代谢紊乱；IRRT 和 PIRRT 每次治疗前审查，必要时调整处方。

**Practice Point 5.3.3**：成人 AKI 使用急性 PD 时，处方**每周 Kt/V 2.2（每日 18-24 L PD 液）**（优于更高剂量）。

**Practice Point 5.3.4**：成人 AKI 初始 PD 处方的透析周期时间和葡萄糖浓度可能需要调整，以快速纠正酸碱、电解质、代谢或液体并发症。

---

## 4. RRT 透析导管

**Practice Point 5.4.1**：RRT 血管通路（类型、尺寸、长度和置入部位）的选择应考虑可行性，权衡**导管定植/感染或出血风险**与维持导管/管路通畅的需求。

**Practice Point 5.4.2**：采用标准化**透析导管置入和维护束**可降低并发症风险。

**Practice Point 5.4.3**：对于 IRRT 的儿童和成人，既往存在的**动静脉瘘/移植物**可避免新置透析导管。

**Practice Point 5.4.4**：对儿童和成人，**避免使用**既有动静脉瘘/移植物进行 CRRT（因长时间穿刺有导管移位或动静脉瘘/移植物损伤风险）。

**Practice Point 5.4.5**：对于性能不佳的无 cuff 非隧道透析导管更换方式应个体化决策，可选择新穿刺或导丝交换。

**Practice Point 5.4.6**：**不常规进行**计划性透析导管更换以降低导管相关感染风险。

**Practice Point 5.4.7**：预期 RRT 需延长时（如预期），可在 AKI 启动 RRT 时即考虑放置 **cuff 隧道式导管**（尤其是儿童）。

**Practice Point 5.4.8**：对于 RRT 依赖且 AKI 恢复可能性低或为出院做准备的患者，应过渡到 **cuff 隧道式导管**。

---

## 5. RRT 抗凝

**Rec 5.5.1**：在条件允许时，推荐对无柠檬酸禁忌证的儿童和成人 CRRT 患者，使用**局部柠檬酸抗凝**而非肝素（**1B**）。

**Practice Point 5.5.1**：如柠檬酸禁忌或不可及，普通肝素、前列环素或甲磺酸萘莫司他可作为儿童和成人 CRRT 的可接受替代。

**Practice Point 5.5.2**：对接受 CRRT 的**高凝状态**儿童和成人，在系统性抗凝基础上**加用局部柠檬酸抗凝**可能适宜。

**Practice Point 5.5.3**：对接受 IHD 或 PIRRT 且无活动性出血的儿童和成人，如未接受系统性抗凝，应考虑使用**普通肝素或低分子肝素**。

**Practice Point 5.5.4**：对接受 RRT 伴**急性肝素诱导血小板减少症（HIT）**且需系统性抗凝的成人，应使用 **阿加曲班、达那帕罗或比伐卢定**（优于其他非肝素抗凝药）。

**Practice Point 5.5.5**：对亚急性 HIT（ HIT-A、HIT-B）或远期 HIT 且未接受系统性抗凝、需抗凝以防止透析回路凝血的成人，**局部柠檬酸抗凝优于肝素或其他非肝素抗凝药**。

---

## 6. RRT 滤器

**Practice Point 5.6.1**：儿童和成人 AKI 接受 CRRT 或 IHD 时，应使用**生物相容性膜滤器/透析器**。

**Practice Point 5.6.2**：维持和最大化回路/滤器寿命的非药物策略包括：优化导管位置（右心房）、使用预稀释血液滤过、CVVDHD 或 CVVHDF（优于 CVVH）。

**Practice Point 5.6.3**：当局部柠檬酸或系统性肝素抗凝禁忌或不可及时，CRRT 回路/滤器用含**普通肝素（UFH）的溶液预充**可能延长回路/滤器寿命。

---

## 7. RRT 置换液和透析液

**Practice Point 5.7.1**：在以下情况，儿童和成人 AKI 的透析液和置换液应使用**碳酸氢盐而非乳酸盐**缓冲：循环休克、肝衰竭或乳酸酸中毒。

**Practice Point 5.7.2**：CRRT 的透析液和置换液应**无菌**。IRRT 的透析液和置换液应符合 AAMI 细菌和内毒素污染标准。

**Practice Point 5.7.3**：本地配制的 CRRT 透析液和置换液应采用标准化质量控制流程；或使用商品化透析液和置换 CRRT 溶液（便于灵活调整以适应各种血浆电解质成分目标）。

**Practice Point 5.7.4**：透析液和置换 CRRT 溶液具体成分的选择应根据**血清电解质和酸碱参数目标**；使用应个体化，并根据患者临床状况频繁重新评估。

---

## 8. RRT 超滤

**Practice Point 5.8.1**：CRRT **目标超滤量**应每日处方；IRRT 每次治疗前处方，并频繁监测。

**Practice Point 5.8.2**：超滤处方强度（如时机和移除速率）应个体化，并考虑患者特征、临床背景和疾病阶段。

---

## 9. RRT 停止

**Practice Point 5.9.1**：当 AKI 已缓解至肾脏功能足以满足患者代谢和液体平衡需求，或 RRT 不再符合患者照护偏好时，**停止 RRT**。

**Practice Point 5.9.2**：临床和代谢参数、透析间 SCr 变化、尿量、液体平衡、AKI 缓解其他指标和残余肾功能可指导 RRT 停止决策。

**Practice Point 5.9.3**：尿量 **>450 ml/24h**（未使用利尿剂）或 **>2300 ml/24h**（使用利尿剂），且 2 小时定时肌酐清除率 **≥23 ml/min**，可用于预测成人成功停止 RRT。

**Practice Point 5.9.4**：利尿剂治疗在 RRT 停止前后使用可能有助于增加尿量和液体管理，但**未证明可促进 AKI 缓解**。

---

## 10. RRT 质量保证

**Practice Point 5.10.1**：RRT 项目应包括以下质量标准以减少不必要变异、促进循证实践落实、支持持续质量改进：

- 专属领导团队，理想为**医学（医师）和护理二元 partnership**
- 所有开具和实施急性 RRT 的医护人员的认证培训和继续教育
- 急性 RRT 处方的**标准化医嘱集**（含剂量强度、超滤和抗凝策略决策支持）
- 常规和电子健康记录的**标准化 RRT 图表**（含患者和机器溶质清除及液体平衡参数）
- 质量和关键绩效指标的**监测与报告**

**Practice Point 5.10.2**：RRT 关键质量绩效指标包括：有效治疗时间（即停机时间）、滤器寿命、处方-实施剂量强度和超滤。

---

## 11. RRT 与体外生命支持（ECLS）

**Practice Point 5.11.1**：在患者接受 ECLS 时整合 RRT 的策略取决于临床背景、当地专业能力和可用技术与资源。

**Practice Point 5.11.2**：大多数接受 ECLS 的患者同时存在血流动力学不稳定。对这些患者，初始支持**优选 CRRT 而非 IRRT**。

**Practice Point 5.11.3**：与肾脏功能无关的**难治性液体蓄积**可能是接受 ECLS 患者启动 RRT 的合理指征。

---

## 12. 儿童特殊推荐

### 启动时机

**Practice Point 5.12.1.1**：危重儿童严重 AKI 在 ICU 入院或 AKI 发病 **2 天内**启动 RRT 可能改善预后。

**Practice Point 5.12.1.2**：危重儿童**单纯液体超负荷**可作为 RRT 启动指征。

### 模式

**Practice Point 5.12.2.1**：急性 PD 是可接受的 RRT 模式，在特定情况下可能优于 CRRT。

### 剂量

**Practice Point 5.12.3.1**：儿童初始 CRRT effluent volume 可为 **25-30 ml/kg/h**，之后根据电解质、酸碱和代谢紊乱的纠正充分性调整。

**Practice Point 5.12.3.2**：儿童 CRRT 剂量强度处方应基于体重（ml/kg/h），按入院或 ICU 入院时体重或理想体重，**不按体表面积处方**。

**Practice Point 5.12.3.3**：儿童 AKI PD 初始可用 **10-20 ml/kg** 填充量和短周期时间（60-90 分钟），根据耐受情况逐渐增加量和停留时间，前 1-3 天连续治疗 24 小时以达到溶质清除和液体平衡目标。

**Practice Point 5.12.3.4**：高氨血症儿童需增加 CRRT 剂量强度时（如超常规），CRRT 剂量强度可达 **200 ml/kg/h** 以实现充分的氨清除；应监测意外溶质丢失。

### 导管

**Practice Point 5.12.4.1**：小儿童（包括新生儿血管细小时），与硬质临时导管相比，使用**柔软 cuff 导管**可能血栓形成性更低、更优选（即使预期透析时间不确定时）。

### 抗凝

**Practice Point 5.12.5.1**：儿童需小导管和回路血量及较低血流量，CRRT 期间**抗凝往往必要且优于无抗凝**。

### 滤器

**Practice Point 5.12.6.1**：儿童 AKI 接受 CRRT 或 IRRT 时，宜选择与体表面积成比例的**适当尺寸膜**。

### 超滤

**Practice Point 5.12.7.1**：儿童**净超滤（UF_NET）不应超过 2.5 ml/kg/h**；应频繁评估液体平衡（每 4-6 小时）以评价液体移除耐受性并必要时调整液体平衡目标。

### 停止

**Practice Point 5.12.8.1**：尿量超过 **>0.5 ml/kg/h 持续 6 小时**可提示 AKI 缓解，可用于预测儿童成功停止 RRT。

# 3.5 复发性膀胱炎（Recurrent Cystitis）

## 3.5.1 定义

复发性膀胱炎定义为：
- **≥ 3 次膀胱炎发作/年**，或
- **近 6 个月内 ≥ 2 次发作**

复发性膀胱炎对患者生活质量产生负面影响，包括社交和性关系质量、自尊和工作能力下降[169]。

## 3.5.2 诊断评估

复发性膀胱炎常见。初始诊断应以**尿培养确认**。不推荐将膀胱镜、影像学检查等广泛常规检查作为常规手段（诊断收益低）[170]；但出现不典型表现（如疑似肾结石、排尿梗阻、间质性膀胱炎或尿路上皮癌）时应及时进行。

### 复发性膀胱炎风险因素（表4，按年龄分层）

| 年轻/绝经前女性 | 绝经后/老年女性 |
|----------------|----------------|
| 性交 | 绝经前膀胱炎史 |
| 杀精剂使用 | 尿失禁 |
| 新性伴侣 | 萎缩性阴道炎（雌激素缺乏） |
| 母亲有膀胱炎史 | 膀胱膨出 |
| 儿童期膀胱炎史 | 残余尿量增加 |
| 血型抗原分泌型 | 尿导管留置及功能状态（养老院女性） |

## 3.5.3 疾病管理与预防

预防策略按以下顺序尝试：行为咨询 → 非抗菌措施 → 抗菌预防[168, 172]。需识别并处理任何泌尿系统风险因素；残余尿过多应优化处理（包括必要时清洁间歇导尿，CIC）。

### 3.5.3.1 证据来源

文献检索（截止 2021 年 5 月 31 日）共识别 3,604 篇摘要，361 篇全文审查，最终纳入 114 项系统综述/指南和 131 篇原始研究。另有 18 篇从参考文献补充。更新检索（截止 2022 年 6 月 1 日）新增 316 篇摘要，25 篇进入全文审查。研究分入 9 个预防亚组。

### 3.5.3.2 行为干预

在启动长期预防性用药前，应对以下风险行为进行咨询（但现有证据有限）[173, 174]：
- 饮水不足
- 习惯性/性交后延迟排尿
- 大便后从后向前擦拭
- 阴道冲洗
- 穿紧身内裤

一项开放标签 RCT 发现：每日额外饮用 1.5 L 水，可使低饮水量（< 1.5 L/天）的绝经前女性在 12 个月内膀胱炎发作次数和抗生素使用均减少[175]。

### 3.5.3.3 非抗菌预防

#### 3.5.3.3.1 激素替代（绝经后女性）

4 项 meta 分析显示：阴道局部雌激素 therapy（含乳膏或栓剂）对预防复发性膀胱炎有积极趋势[176-179]。所有研究均报告局部应用优于安慰剂，但劣于抗生素。阴道用药因药代动力学特点无全身性副作用，但可能出现局部刺激和轻微出血。

口服雌激素对复发性膀胱炎预防**无效**，且伴随不利的全身性副作用特征[176-179]。一项针对 30 名口服避孕药的绝经前复发性膀胱炎女性的前瞻性非对照研究，报告加用阴道局部雌激素有有益效果[180]。

> [!tip]- 临床要点
> 阴道局部雌激素（乳膏或栓剂）是绝经后女性预防复发性膀胱炎的有效选择，无全身副作用。

#### 3.5.3.3.2 免疫调节

一项纳入 14 项比较研究、共 2,822 例患者的系统综述和 meta 分析[181]，评估了 5 种免疫调节剂（StroVac、OM-89、ExPEC4V、MV140、Solco-Urovac）的疗效：

| 指标 | 数据 |
|------|------|
| 短期（6-12 个月）保持无膀胱炎患者比例 | 合并 RR **1.52**（95%CI：1.05-2.20）vs 安慰剂 |
| NNT（需治数） | **6.45**（95%CI：2.80-64.80） |
| MV140 vs 安慰剂 | RR **2.23**（95%CI：1.43-3.47），但仅来自 1 项 146 人的 RCT |
| OM-89、Solco-Urovac、ExPEC4V | vs 安慰剂无统计学显著差异，但有获益趋势 |

安全性：免疫调节剂总体耐受性良好，严重不良事件罕见，无死亡报告。最常见不良事件：头痛、胃肠不适、阴道炎。

> [!important]- 推荐条件
> 由于现有数据可信度低，**强烈推荐降级为弱推荐**。Panel 建议在设计良好的临床试验中使用。

#### 3.5.3.3.3 益生菌（乳杆菌属）

5 项 meta 分析和 11 项系统综述[177, 182-195]。其中 2 项报告有效乳杆菌株 vs 安慰剂有显著预防获益[186, 188]。

**有效菌株**：L. rhamnosus GR-1、L. reuteri B-54、L. reuteri RC-14、L. casei shirota、L. crispatus CTV-05[177, 184, 186, 188]。

11 项系统综述中 7 项结论：阴道局部益生菌对预防复发性膀胱炎有临床获益[178, 179, 182, 185, 187, 189-192, 194]。现有数据质量不足以推荐具体的给药途径、最佳剂量和疗程。

#### 3.5.3.3.4 蔓越莓

7 项 meta 分析和多项系统综述[177, 196-201]。Cochrane 系统综述结论：vs 安慰剂、水或无干预，蔓越莓产品**未显著减少**有症状膀胱炎总体发生率或复发性膀胱炎亚组[196]。但 6 项后续 meta 分析认为蔓越莓对特定人群有保护作用[177, 197-201]（差异源于纳入研究的临床和方法学异质性[202]）。

1 项 RCT（145 名女性，高剂量 vs 低剂量 PAC 提取物）：两剂量组有症状膀胱炎发作次数无显著差异[203]。

> [!note]- Panel 共识
> 鉴于获益/危害比有利，临床医生可向充分了解弱证据基础的女性推荐蔓越莓产品；但尚无明确推荐剂量和疗程。

#### 3.5.3.3.5 D-甘露糖

1 项 meta 分析（含 1 项 RCT、1 项随机交叉试验、1 项前瞻性队列研究，390 例）报告 D-甘露糖 vs 安慰剂有效，且疗效与抗生素预防相当[142]。另 1 项系统综述[182]结论 D-甘露糖对膀胱炎有显著效果，但需进一步研究证实。包含 695 例患者的系统综述[204]报告 D-甘露糖改善生活质量，显著减少导尿和非导尿患者的复发性膀胱炎。

但 Cochrane 系统综述（719 例）[205]无法确定 D-甘露糖 vs 无治疗、其他补充剂或抗生素是否显著减少复发性膀胱炎发作次数。**证据整体质量低**。

> [!warning]- 证据警示
> D-甘露糖预防复发性膀胱炎的证据**相互矛盾**，质量低，应告知患者。

#### 3.5.3.3.6 膀胱内灌注（GAG 层补充疗法）

透明质酸（HA）和硫酸软骨素（CS）用于膀胱内灌注，补充糖胺聚糖（GAG）层，治疗间质性膀胱炎、膀胱过度活动症、放射性膀胱炎及预防复发性膀胱炎[206]。

**研究证据：**
- 第一项 meta 分析（n=143，2 项 RCT + 2 项非 RCT）：HA/HA-CS 显著减少每位患者每年膀胱炎发作次数，显著延长复发间隔时间[207]
- 第二项 meta 分析（n=800，2 项 RCT + 6 项非 RCT）：HA（含或不含 CS）显著降低每位患者每年平均膀胱炎发作率，显著延长复发时间[208]

HA+CS 联合疗法的证据质量高于 HA 单药（因 HA+CS 有 RCT 支持）。

#### 3.5.3.3.7 乌洛托品 hippurate（甲胺乌洛托品）

- **2012 年 Cochrane 综述**（13 项研究，异质性高）：甲胺乌洛托品可能对无泌尿系异常患者短期预防有效[209]
- **2021 年 meta 分析**（6 项研究）：甲胺乌洛托品与对照之间无统计学显著差异，有获益趋势[210]
- **后续 RCT**（240 名女性，1:1 分配）：每日低剂量抗生素预防 vs 甲胺乌洛托品 1 g bid，共 12 个月：
  - 甲胺组：1.38 次/人/年
  - 抗生素组：0.89 次/人/年
  - 绝对差异 0.49 —— 证实甲胺 hippurate **非劣效于** 抗生素预防
  - 两组不良事件发生率相似，停药后 6 个月两组均维持持续获益[211, 212]

### 3.5.3.4 抗菌预防

#### 3.5.3.4.1 连续低剂量抗菌预防与性交后预防

4 项 meta 分析和多项系统综述/指南[179, 213-223]：所有可用 meta 分析均结论**抗菌预防是最有效的预防复发方案**，优于安慰剂或无干预[213-215]。

抗菌药物可作为连续低剂量长期预防，或性交后预防。两种方案疗效无显著差异。连续预防最佳疗程尚无共识，研究报告 3-12 个月不等。停药后膀胱炎倾向于复发，尤其是每年 ≥ 3 次感染者。

> [!important]- 告知义务
> 提供咨询后，必须同时告知连续低剂量抗菌预防和性交后预防两种选择，仅在行为调整和非抗菌措施失败后才开始。

**常用方案：**

| 药物 | 方案 | 备注 |
|------|------|------|
| 呋喃妥因 | 50 mg 或 100 mg，每日一次 | |
| 磷霉素 trometamol | 3 g，每周一次 | |
| 甲氧苄啶 | 100 mg，每日一次 | |
| 妊娠期 | 头孢氨苄 125 mg 或 250 mg，或头孢克洛 250 mg，每日一次 | |

**妊娠女性**：有频繁膀胱炎史的妊娠女性应考虑性交后预防，以降低膀胱炎风险[226]。

#### 3.5.3.4.2 自我诊断与自我治疗

对依从性良好的患者，应考虑自我诊断和短疗程抗菌自我治疗[227]。药物选择同单纯急性膀胱炎（见 3.4.4.4）。

## 3.5.4 证据与建议汇总

| 证据 | 等级 |
|------|------|
| 广泛常规检查（含膀胱镜、影像学）在 < 40 岁无风险因素的复发性膀胱炎女性中诊断收益低 | 3 |
| 增加饮水量对低饮水量（< 1.5 L/天）绝经前高风险女性有效 | 3 |
| 阴道局部雌激素对绝经后女性预防复发性膀胱炎有积极趋势 | 1b |
| 免疫调节剂在成人女性短期减少膀胱炎复发方面效果有限 | 1a |
| OM-89 和 MV140 是研究最多的免疫调节剂，MV140 结果最有前景 | 1a |
| 含 L. rhamnosus GR-1、L. reuteri B-54/RC-14、L. casei shirota、L. crispatus CTV-05 的益生菌对阴道菌群重建和预防复发性膀胱炎有效 | 1b |
| 蔓越莓产品证据质量低且结论矛盾，尚无明确剂量和疗程 | 1a |
| D-甘露糖单独使用可缓解急性膀胱炎症状（vs 复方新诺明） | 1b |
| 膀胱内 GAG 治疗可减少每位患者每年膀胱炎发作次数，延长复发间隔 | 2 |
| 甲胺乌洛托品每日两次非劣效于每日抗生素预防 | 1b |
| 连续低剂量抗菌预防和性交后抗菌预防均可减少复发性膀胱炎 | 1b |
| 前瞻性队列研究显示间歇性自我起始治疗对复发性膀胱炎女性有效、安全、经济 | 2b |

| 建议 | 推荐强度 |
|------|---------|
| 通过尿培养诊断复发性膀胱炎 | **强** |
| < 40 岁无风险因素女性：不推荐常规广泛检查（如膀胱镜、腹部超声） | **弱** |
| 建议低饮水量绝经前女性增加饮水量（可减少复发风险） | **弱** |
| 绝经后女性使用阴道局部雌激素预防复发性膀胱炎 | **强** |
| 在设计规范的临床试验中使用免疫调节预防（弱推荐，降级前为强推荐） | **弱** |
| 建议使用含有效菌株的阴道或口服益生菌，促进阴道菌群重建预防膀胱炎 | **弱** |
| 建议使用蔓越莓产品（优选果汁）缓解急性膀胱炎症状和预防复发；需告知患者证据质量低且结论矛盾 | **强** |
| 使用 D-甘露糖减少复发性膀胱炎发作；需告知患者证据整体弱且矛盾 | **弱** |
| 对无泌尿系结构异常的女性的复发性膀胱炎使用甲胺乌洛托品 | **强** |
| 非侵入性预防措施失败后，使用透明质酸或透明质酸+硫酸软骨素膀胱内灌注预防复发性膀胱炎；需告知患者仍需更多研究确认初步结果 | **弱** |
| 非抗菌干预失败后，使用连续低剂量或性交后抗菌预防；需告知可能副作用 | **强** |
| 对依从性良好的患者，考虑自我给药的短疗程抗菌治疗 | **强** |

# EAU Guidelines on Urological Infections（泌尿系感染指南）

## 指南简介

欧洲泌尿外科学会（EAU）泌尿系感染指南旨在提供关于**尿路感染（UTI）**及**男性附属性腺感染**预防和治疗的实用建议，同时关注感染控制和抗菌药物管理的重要公共卫生方面。

> [!warning]- 重要说明
> 指南呈现的是专家可获得的最佳证据，但遵循指南建议不一定会带来最佳结局。指南不能替代临床专业知识，而是帮助聚焦决策——同时考虑患者个人价值观和具体情况。**指南不是强制令，也不构成法律责任标准。**

另有专门指南涵盖儿科泌尿系感染 [1] 和神经源性下尿路功能障碍患者的感染 [2]。

## 专家组

EAU 泌尿系感染指南专家组由国际多学科团队组成，包括泌尿外科专家、感染病专家和临床微生物学家。所有专家均提交了利益冲突声明，可在 EAU 网站 Uroweb 查看。

### 推荐强度形成因素

专家组在形成推荐时考虑以下关键要素[3, 4]：

1. 证据的总体质量
2. 效应量（个体或综合效应）
3. 结果的确定性（精确性、一致性、异质性等）
4. 期望结局与不期望结局之间的平衡
5. 患者价值观和偏好对干预的影响

**强推荐（Strong）：** 通常表示高质量证据和/或利弊比及患者偏好有利。
**弱推荐（Weak）：** 通常表示低质量证据，和/或利弊平衡不明确，及患者偏好存在不确定性或变异性。

### 证据等级（Levels of Evidence, LE）

| 等级 | 说明 |
|------|------|
| **1a** | 随机对照试验的系统综述（同质性） |
| **1b** | 单项随机对照试验（窄可信区间） |
| **2a** | 系统综述（队列研究） |
| **2b** | 单项队列研究或低质量 RCT |
| **3** | 病例对照研究 |
| **4** | 病例系列、低质量队列或病例对照研究 |
| **5** | 专家意见 |

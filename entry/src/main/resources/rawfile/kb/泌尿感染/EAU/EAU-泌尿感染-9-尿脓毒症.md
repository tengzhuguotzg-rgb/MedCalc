# 3.9 尿脓毒症（Urosepsis）

## 3.9.1 引言

尿脓毒症应尽早诊断。既往发热或体温过低、白细胞增多或减少、心动过速和呼吸急促被认定为 SIRS（全身炎症反应综合征）警示症状[304, 305]，但 SIRS 已不再列入当前脓毒症术语（见表 7）。

脓毒症死亡率随严重程度增加而显著升高。尿脓毒症治疗包括：适当生命支持、及时抗菌治疗、辅助措施及泌尿系障碍的最佳管理[307]。**解除梗阻引流脓肿等源头控制是核心**——这是急诊。建议泌尿外科医师与重症医学和感染科医师协作治疗。

## 3.9.2 流行病学、病因与病理生理

UTI 可表现为从菌尿（临床症状有限）到脓毒症或严重脓毒症的多种形式。患者可能在很短时间内从几乎无害的状态进展为严重脓毒症。

**死亡率：** 因器官来源不同而异，泌尿系脓毒症总体死亡率**低于其他来源**[308, 309]。男性多于女性[310]。近年来脓毒症总体发病率每年增加 **8.7%**，但相关死亡率下降（院内死亡率从 1995-2000 年的 27.8% 降至 17.9%）[311]。革兰阴性菌在尿脓毒症中仍占主导地位[301, 312]。

**易感人群：** 老年患者、糖尿病患者、免疫抑制患者（移植受者、化疗或糖皮质激素治疗患者）。泌尿系局部因素：尿路结石、任何部位梗阻、先天性泌尿系疾病、神经源性下尿路功能障碍（NLUTD）、内镜操作。

## 3.9.3 诊断评估

### 表7：脓毒症和脓毒性休克的定义与标准[304-306]

| 疾病 | 定义 |
|------|------|
| **脓毒症（Sepsis）** | 感染引起的危及生命的器官功能障碍。由失调的宿主反应导致。临床应用：器官功能障碍可表现为 Sequential [Sepsis-related] Organ Failure Assessment（**SOFA**）评分增加 ≥ 2 分。快速识别用 **qSOFA**（quick SOFA）：呼吸频率 ≥ 22 次/分、精神状态改变、收缩压 ≤ 100 mmHg |
| **脓毒性休克（Septic shock）** | 脓毒症的一个亚型，存在显著的循环、细胞和代谢异常，死亡率高于单纯脓毒症。临床识别：需要血管活性药物维持平均动脉压 ≥ 65 mmHg + 血清乳酸 > 2 mmol/L（> 18 mg/dL）（排除低血容量） |

### 微生物学采样

- 尿培养
- **两套血培养**[313]
- 引流液（如适用）

### 影像学

早期进行超声和 CT 扫描[314]。

| 证据 | 等级 |

| 源头控制干预应尽快实施，以控制或消除已诊断和/或疑似感染灶 | 3 |

|------|---------|
| 进行 qSOFA 评分识别潜在脓毒症患者 | **强** |

| 临床假设脓毒症后 **1 小时内**肠外给予高剂量广谱抗菌药物 | **强** |
| 根据培养结果调整初始经验性抗菌治疗 | **强** |
| 实施源头控制：移除异物、解除梗阻、引流泌尿系脓肿 | **强** |
| 立即提供适当生命支持措施 | **强** |

## 3.9.4 病理生理与生化标志物

### 3.9.4.1 细胞因子与脓毒症反应

细胞因子参与脓毒症发病[309]，调控宿主炎症反应的幅度和持续时间。严重脓毒症中促炎和抗炎反应之间的复杂平衡被破坏。脓毒症可能表明免疫系统严重受损、无法清除病原体，或炎症过度激活，或两者兼有[309]。器官衰竭和死亡的机制目前仍不完全清楚。

### 3.9.4.2 生化标志物

**降钙素原（PCT）：** 严重全身性感染（细菌、寄生虫、真菌）伴全身表现时，PCT 水平升高；严重病毒感染或非感染性炎症反应时，PCT 仅轻度升高或无升高[315]。PCT 监测有助于脓毒症可能性评估及鉴别严重非细菌性炎症状态[315, 317]。

**肾上腺髓质素中段肽（MR-proADM）：** 脓毒症早期高动力学循环建立的关键诱导物，与脓毒性休克进展相关[316]。

**血清乳酸：** 器官功能障碍标志物，与脓毒症死亡率相关[318]。严重感染患者应监测血清乳酸。

## 3.9.5 治疗

### 3.9.5.1 预防

最有效的预防措施同其他医疗相关感染[319, 320]：
- 隔离多重耐药菌患者
- 审慎使用抗菌药物（预防和治疗）
- 缩短住院时间
- 尽早移除留置导尿管
- 使用封闭式引流系统、最小化系统完整性破坏
- 使用最小侵入性方法解除尿路梗阻（患者稳定前）
- 注意日常无菌技术（洗手、保护性手套、交叉感染防控）

> [!warning]- 支架置入的特殊注意
> 抗菌预防**不能预防**支架定植——永久支架患者 100%、临时支架患者 70% 出现支架定植[321]。

### 3.9.5.2 治疗

#### 早期目标导向治疗（EGDT）

最初随机对照单中心研究显示 EGDT 可改善急诊脓毒性休克患者生存[322]，但后续改善急诊背景的多中心研究未能复制该结果[323-325]。个体患者数据 meta 分析结论：EGDT **不优于常规治疗**，且增加住院费用[326]。

#### 抗菌治疗

初始经验性抗菌治疗应提供针对所有可能致病菌的**广谱覆盖**，并根据培养结果调整[307, 314]。**脓毒症患者抗菌药物剂量应足够高**，并根据肾功能调整。**必须在临床假设脓毒症后 1 小时内给药**[307]。

**表8：尿脓毒症推荐抗菌方案**

| 抗菌药物 | 每日剂量 | 疗程 |

#### 源头控制

尿路梗阻是尿脓毒症最常见的泌尿系病因。**引流梗阻和脓肿、移除异物（如导管、结石）是最重要的源头控制策略**，是绝对急诊[307, 314]。

#### 辅助措施[307, 314]

| 措施 | 要点 |
|------|------|
| 液体治疗 | 晶体液；若晶体液不能充分升高血压，加白蛋白 |
| 血管活性药物 | **去甲肾上腺素**首选；心肌功能障碍时加多巴酚丁胺 |
| 糖皮质激素 | 仅在液体和血管活性药物无法维持平均动脉压 ≥ 65 mmHg 时使用氢化可的松 |
| 血液制品 | 目标血红蛋白 7-9 g/dL |
| 机械通气 | 潮气量 6 mL/kg，平台压 ≤ 30 cmH₂O，高呼气末正压（PEEP） |
| 镇静 | 最小化；避免神经肌肉阻滞剂 |
| 血糖目标 | ≤ 180 mg/dL |
| DVT 预防 | 低分子量肝素皮下注射 |
| 应激性溃疡预防 | 质子泵抑制剂（高危患者） |
| 营养 | 早期（< 48 小时）肠内营养 |

# 3.11 细菌性前列腺炎（Bacterial Prostatitis）

## 3.11.1 引言与 NIH 分类

建议使用美国国立糖尿病、消化和肾脏疾病研究所（NIDDK/NIH）分类[360-362]：

| 类型 | 名称与描述 |
|------|-----------|
| **Ⅰ** | 急性细菌性前列腺炎（ABP） |
| **Ⅱ** | 慢性细菌性前列腺炎（CBP） |
| **Ⅲ** | 慢性非细菌性前列腺炎/慢性盆腔疼痛综合征（CPPS） |
| ⅢA | 炎症性 CPPS（精液/EPS/VB3 中有白细胞） |
| ⅢB | 非炎症性 CPPS（精液/EPS/VB3 中无白细胞） |
| **Ⅳ** | 无症状炎症性前列腺炎（组织学前列腺炎） |

EPS = expressed prostatic secretion（前列腺按摩液）；VB3 = voided bladder urine specimen 3（前列腺按摩后尿液）。

## 3.11.2 流行病学、病因与发病机制

前列腺炎是常见诊断，但**确诊细菌感染的比例 < 10%**[228]。

**急性细菌性前列腺炎（ABP）：** 肠杆菌，尤其是 **大肠杆菌（E. coli）** 是主要病原体[376]。

**慢性细菌性前列腺炎（CBP）：** 病原体谱更广，可包括非典型微生物[370]。1 项纳入 1,442 例患者的回顾性研究（4 年）：感染病因确定率 74.2%，其中沙眼衣原体 37.2%、阴道毛滴虫 10.5%、解脲支原体 5%、大肠杆菌仅 6.6%[375]。胞内菌（如 C. trachomatis）的临床意义尚不确定[378]，但有研究提示其为 CBP 致病菌[379, 380]。

**免疫缺陷或 HIV 患者：** 可由结核分枝杆菌、白色念珠菌、芽生菌、组织胞浆菌等机会性病原体引起[377]。

## 3.11.3 诊断评估

### 3.11.3.1 病史与症状

**急性细菌性前列腺炎（ABP）：** 突然起病，排尿症状+定位困难的疼痛，常伴不适和发热。直肠指检（DRE）前列腺可肿大和压痛。**禁忌前列腺按摩**（可诱发菌血症和脓毒症）[364]。

**慢性细菌性前列腺炎（CBP）：** 症状持续 **≥ 3 个月**[381-383]。主要症状：疼痛（会阴、阴囊、阴茎、大腿内侧）及下尿路症状（LUTS）[360-362]。

### 3.11.3.2 症状问卷

慢性前列腺炎症状指数（CPSI）已验证，但临床实用性尚不确定[372]。

### 3.11.3.3 临床检查

**尿试纸检测：** 亚硝酸盐和和白细胞，ABP 患者阳性预测值 **95%**，阴性预测值 **70%**[386]。

**血液检查：** 血培养和全血细胞计数对 ABP 有用。

**影像学：** 经直肠超声可检测疑似前列腺脓肿，但作为诊断工具**不可靠**[389]。

### 3.11.3.4 尿培养与前列腺按摩液（EPS）

**ABP：** 中段尿培养是最重要的检查[370]。

**CBP：** Meares 和 Stamey 四杯法或两杯法的分段尿定量细菌培养和显微镜检查仍然是分类前列腺炎的重要检查[373, 374]。Meares-Stamey 试验的精确微生物学分析也可提供非典型病原体（沙眼衣原体、阴道毛滴虫、解脲支原体）的信息[375]。两杯法与四杯法诊断敏感性相似[388]。

### 3.11.3.5 前列腺活检

不推荐作为常规检查；**未治疗的细菌性前列腺炎患者不宜进行前列腺活检**（脓毒症风险增加）。

### 3.11.3.6 其他检查

**精液培养：** 敏感性约 50%，不作为 CBP 常规诊断检查[373, 374]。膀胱流出道梗阻和尿道梗阻应通过尿流率测定、逆行尿道造影或内镜排除。

**首次尿（First-void urine）：** 是男性通过 NAAT 诊断泌尿生殖道沙眼衣原体感染的首选标本（无创，可检测感染上皮细胞）[390]。

**前列腺特异性抗原（PSA）：** ABP 和 CBP 患者分别约 60% 和 20% 的 PSA 升高；抗生素治疗后约 40% 患者 PSA 下降，与临床和微生物学改善相关[365]。但前列腺炎时 PSA 检测**无实际诊断价值**，应避免[391]。

### 诊断证据与建议汇总

| 证据 | 等级 |
|------|------|
| ABP 患者尿试纸亚硝酸盐和白细胞阳性预测值 95%，阴性预测值 70% | 3 |
| 四杯 Meares-Stamey 试验是 CBP 最佳诊断测试；两杯法敏感性相似 | 2b |
| 首次尿是男性 NAAT 诊断泌尿生殖道衣原体感染的首选标本 | 2b |
| 经直肠超声作为前列腺炎诊断工具不可靠 | 3 |
| 精液培养敏感性约 50%，不作为 CBP 常规诊断 | 3 |
| 前列腺炎活动期 PSA 可升高，对前列腺炎诊断无实际价值 | 3 |

| 建议 | 推荐强度 |
|------|---------|
| ABP 患者不要进行前列腺按摩 | **强** |
| 临床疑似 ABP：检查中段尿亚硝酸盐和白细胞 | **弱** |
| ABP 患者留中段尿培养以指导诊断和定制抗菌治疗 | **弱** |

| 选择性病例进行经直肠超声排除前列腺脓肿 | **弱** |
| 不要常规单独使用精液微生物学分析诊断 CBP | **弱** |

## 3.11.4 疾病管理

### 3.11.4.1 抗菌治疗

**ABP：** 肠外给药高剂量杀菌性抗菌药物（如广谱青霉素、第三代头孢菌素或氟喹诺酮）。初始治疗可联合氨基糖苷类[376-385, 392-396]。辅助措施包括充足补液和尿液引流[228]。感染参数正常后，可过渡到口服，总疗程 **2-4 周**[397]。

**CBP：** 氟喹诺酮类（尽管尿路病原体耐药率高）仍推荐作为一线药物，因其药代动力学特性优越、安全性良好、对革兰阴性菌（含铜绿假单胞菌）和沙眼衣原体有抗菌活性[363, 399]。但细菌耐药性增加是关注点。阿奇霉素和多西环素对衣原体和生殖支原体有活性[366, 375]。左氧氟沙星在 CBP 患者中未显示对沙眼衣原体的显著清除率[400]。甲硝唑适用于滴虫性前列腺炎[367]。

**CBP 疗程：**
- 氟喹诺酮：**≥ 14 天**
- 阿奇霉素/多西环素：**≥ 3-4 周**
- 确诊后抗菌治疗：**4-6 周**[370]
- 检测到胞内细菌：大环内酯或四环素类[363, 398, 401]

> [!warning]- 胞内菌优先用大环内酯/四环素
> 胞内病原体 CBP 患者，大环内酯微生物学和临床治愈率高于氟喹诺酮[1a]。

### 3.11.4.2 前列腺内注射抗菌药物

无对照试验评估，不推荐[402, 403]。

### 3.11.4.3 联合治疗

氟喹诺酮联合植物提取物可能改善生活质量和症状[368]，但不增加不良事件。氟喹诺酮联合伐地那非与单用氟喹诺酮相比，微生物学根除率或疼痛/排尿症状无改善[369]。

### 3.11.4.4 引流与手术

约 **10% 的 ABP 男性会发生尿潴留**[404]。近年前瞻性证据表明耻骨上导管引流可减少 CBP 发生风险[405]。

**前列腺脓肿：** 引流和保守治疗均可行，取决于脓肿大小[406, 407]。脓肿直径 < 1 cm 保守治疗成功；更大脓肿建议单次抽吸或持续引流[407]。

### 治疗证据与建议汇总

| 证据 | 等级 |
|------|------|
| ABP：肠外抗生素治疗（系统症状明显者优选），感染参数正常后过渡到口服，总疗程 2-4 周 | 3 |
| 系统综述：氟喹诺酮（环丙沙星、左氧氟沙星、普卢利沙星）微生物学根除、临床疗效和不良事件无显著差异 | 1a |
| 甲硝唑 500 mg t.i.d × 14 天，滴虫性 CBP 患者微生物学根除率 93.3% | 1b |
| 胞内致病菌 CBP，大环内酯临床和微生物学治愈率高于氟喹诺酮 | 1a |
| 选药时应考虑本地耐药模式 | 3 |

| 建议 | 推荐强度 |
|------|---------|
| **ABP：** 按全身性 UTI 方案治疗（见 3.7.4） | **强** |
| **CBP：** 氟喹诺酮（如环丙沙星、左氧氟沙星）一线治疗 | **强** |
| 若胞内细菌为 CBP 致病菌：大环内酯（如阿奇霉素）或四环素（如多西环素） | **强** |
| 滴虫性 CBP：甲硝唑 | **强** |

## 3.11.5 随访

无症状患者治疗后无需常规尿分析和/或尿培养（细菌性前列腺炎无验证的治愈标准测试，除症状缓解外）[370]。持续症状和重复阳性微生物学结果的性传播感染病原体患者，建议筛查其性伴侣并治疗。抗生素治疗可重复，疗程更长、剂量更高和/或换用其他药物[370]。

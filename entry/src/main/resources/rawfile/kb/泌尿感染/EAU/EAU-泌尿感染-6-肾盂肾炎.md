# 3.6 肾盂肾炎（Pyelonephritis）

## 3.6.1 诊断评估

### 3.6.1.1 临床诊断

肾盂肾炎典型表现：发热（> 38°C）、寒战、腰痛、恶心、呕吐、肋椎角压痛，可伴或不伴膀胱炎典型症状[228]。

> [!warning]- 妊娠妇女特殊注意
> 妊娠期急性肾盂肾炎不仅对母体有不良影响（贫血、肾功能不全、呼吸功能不全），还可影响胎儿（早产和出生更频繁）[229]。

### 3.6.1.2 鉴别诊断

关键在于尽快区分有无风险因素的肾盂肾炎（如梗阻性肾盂肾炎，后者可迅速进展为尿脓毒症）。需通过适当影像学手段完成鉴别诊断（见 3.6.1.4）。

### 3.6.1.3 实验室诊断

推荐常规尿分析（含白细胞和红细胞评估及亚硝酸盐）[230]。所有肾盂肾炎病例均应进行**尿培养和药敏试验**。

### 3.6.1.4 影像学诊断

对有尿路结石史、肾功能异常或尿 pH 高的患者，应行上尿路**超声（US）**排除尿路梗阻或肾结石疾病[231]。

**进一步影像学指征：**
- 经验性治疗 72 小时后仍发热 → 考虑增强 CT 或排泄性尿路造影
- 临床状况恶化 → 立即 imaging

**妊娠女性**：首选 US 或 MRI（避免对胎儿辐射风险）[231]。

### 诊断证据与建议汇总

| 证据 | 等级 |
|------|------|
| 所有肾盂肾炎病例均应在尿分析基础上进行尿培养和药敏试验 | 4 |
| 前瞻性观察队列研究：一项简单临床预测规则可选择性应用于上尿路全身性 UTI 成人，无需损失临床重要信息 | 2b |
| 经验性治疗 48-72 小时仍发热，或疑似并发症（如脓毒症）时，应行额外影像学检查（如无创增强螺旋 CT） | 4 |

| 建议 | 推荐强度 |
|------|---------|
| 常规诊断：尿分析（试纸法），含白细胞/红细胞和亚硝酸盐 | **强** |
| 肾盂肾炎患者：尿培养和药敏试验 | **强** |
| 影像学检查排除紧急泌尿系统疾病 | **强** |

## 3.6.2 门诊治疗

**口服经验性治疗：**
- **氟喹诺酮类**和**头孢菌素类**是唯一可推荐的口服经验性药物[232]
- 口服头孢菌素血药浓度和尿药浓度显著低于静脉制剂
- **禁用**：呋喃妥因、口服磷霉素、 pivmecillinam（数据不足以证明疗效）[233]

**氟喹诺酮过敏或已知耐药时的替代选择：**
- TMP-SMX（160/800 mg）
- 已知敏感的 oral beta-lactam
- 若在无药敏结果情况下使用：应给予一剂长半衰期肠外抗菌药物（如头孢曲松）作为起始

**短疗程门诊方案**：急性肾盂肾炎门诊短疗程已被证实与较长疗程在临床和微生物学成功方面效果相当；但伴随 4-6 周内较高的感染复发率，需结合本地政策和耐药情况制定[234]。

## 3.6.3 住院治疗

住院患者初始应使用**静脉抗菌方案**[235]：
- 氟喹诺酮
- 氨基糖苷类（± ampicillin）
- 广谱头孢菌素或青霉素

**新型抗菌药物：**

| 药物 | 证据 |
|------|------|
| **Ceftolozane/tazobactam** | 临床反应率 > 90%；对氟喹诺酮耐药病原体疗效显著优于左氧氟沙星[236-238] |
| **Ceftazidime-avibactam** | 对 ceftazidime 耐药 Enterobacterales 和铜绿假单胞菌有效[239] |
| **Imipenem/cilastatin** | 2 期随机试验显示良好临床反应率[240] |
| **Cefazidime-avibactam 和 doripenem** | 对 ceftazidime 非敏感病原体疗效与碳青霉烯类相似[241] |
| **Meropenem-vaborbactam** | 3 期 RCT 非劣效于哌拉西林-他唑巴坦[242]；对碳青霉烯耐药 Enterobacterales 治愈率 65%（vs 最佳可用治疗）[243] |
| **Plazomicin** | 每日一次，非劣效于美罗培南，治疗由 Enterobacterales（包括 MDR 菌株）引起的复杂性 UTI 和肾盂肾炎[244] |
| **Cefiderocol** | 2 期 RCT 非劣效于亚胺培南/西司他丁，用于 MDR 革兰阴性菌感染的复杂性 UTI[245] |

> [!important]- 使用原则
> 碳青霉烯类和新型广谱抗菌药物**仅限于**早期培养结果显示 MDR 生物时才考虑使用。选药应基于本地耐药模式并根据药敏结果优化。

**疑似尿脓毒症**：经验性覆盖 ESBL 产生菌[246]。

**肠外转口服**：初始肠外治疗后临床改善且能耐受口服者可过渡到口服抗菌治疗[247]。

**妊娠女性肾盂肾炎**：门诊管理可在症状轻微且可密切随访的情况下考虑适当肠外抗菌药物[248, 249]；较严重者需住院和支持治疗，好转后可过渡到口服，总疗程 7-10 天。

**男性**：发热性 UTI、肾盂肾炎或复发性感染，或疑似前列腺受累，最短疗程建议 **2 周**，优选氟喹诺酮类[250]。

### 治疗证据与建议汇总

| 证据 | 等级 |
|------|------|
| 氟喹诺酮类和头孢菌素类是唯一可推荐的口服经验性肾盂肾炎治疗药物 | 1b |
| 静脉抗菌方案可包含：氟喹诺酮、氨基糖苷类（± ampicillin）、广谱头孢菌素或青霉素 | 1b |
| 碳青霉烯类仅在早期培养显示 MDR 生物时才考虑 | 4 |
| 选药应基于本地耐药模式和药敏结果 | 3 |

| 建议 | 推荐强度 |
|------|---------|
| 非住院患者：一线短疗程氟喹诺酮 | **强** |
| 需要住院的患者：初始静脉抗菌治疗 | **强** |
| 临床改善且能耐受口服的肠外治疗患者：过渡到口服 | **强** |
| **禁用**呋喃妥因、口服磷霉素、 pivmecillinam 治疗肾盂肾炎 | **强** |

## 3.6.4 随访

无症状患者治疗后无需常规尿检或尿培养。

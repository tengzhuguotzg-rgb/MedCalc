# 3.3 无症状菌尿（Asymptomatic Bacteriuria, ABU）

## 3.3.1 背景

无症状菌尿（ABU）指无症状个体尿中细菌生长，对应共生定植而非感染。临床研究显示 ABU 可防止重复感染（superinfecting UTI），因此仅在证实对患者有益时才应治疗，以避免选择耐药菌及破坏潜在保护性 ABU 菌株的风险。

> [!important] 核心原则
> ABU 治疗需权衡利弊：仅在明确获益时方可干预。

## 3.3.2 流行病学

| 人群 | ABU 发生率 |
|------|-----------|
| 健康绝经前女性 | 1-5% |
| 健康老年女性/男性 | 4-19% |
| 糖尿病患者 | 0.7-27% |
| 妊娠妇女 | 2-10% |
| 养老机构老年人 | 15-50% |
| 脊髓损伤患者 | 23-89% |
| 年轻男性 | 少见（若检出需排查慢性细菌性前列腺炎） |

病原谱与局部或系统 UTI 相似，取决于风险因素存在与否。

## 3.3.3 诊断标准

- **女性**：连续两次中段尿培养细菌生长 ≥ 10⁵ cfu/mL[24]
- **男性**：单次尿培养阳性即可诊断[25]
- 无泌尿道症状

> [!note] 进一步检查指征
> - 若分离出产脲酶细菌（如奇异变形杆菌），需排查泌尿系结石[26-28]
> - 男性需行直肠指检排查前列腺疾病

## 3.3.4 疾病管理

### 3.3.5.1 无风险因素人群

ABU **不会导致肾脏疾病或损伤**[30]。仅有 1 项面向非糖尿病、非妊娠女性的**前瞻性非 RCT**[31]探讨了治疗 ABU 的效果，结果显示有症状 UTI 发生率无差异。

鉴于在大多数高风险亚组患者中已证实治疗 ABU **不必要**，panel 共识：将高风险亚组结论外推至无风险因素人群，形成以下建议：**不推荐筛查和治疗 ABU**[30]。

### 3.3.5.2 ABU 伴复发性 UTI（其他健康人群）

1 项 RCT 显示：ABU 治疗反而增加后续有症状 UTI 风险（RR 0.28，95%CI 0.21-0.38；n=673）。保护性 ABU 可作为复发性膀胱炎预防的一部分，**不推荐治疗 ABU**。

### 3.3.5.3.1 治疗是否有益（妊娠妇女）

指南纳入 12 项比较抗菌治疗与安慰剂或无治疗的 RCT[32-43]，10 项发表于 1988 年以前，1 项发表于 2015 年。11 项 RCT（n=2,002）报告有症状 UTI 结局：抗菌治疗 vs 安慰剂或无治疗，**平均 RR 0.22（95%CI：0.12-0.40）**。

6 项 RCT 报告菌尿清除：平均 RR 2.99（95%CI：1.65-5.39；n=716）。
8 项 RCT 报告低出生体重：平均 RR 0.58（95%CI：0.36-0.94；n=1,689）。
4 项 RCT 报告早产：平均 RR 0.34（95%CI：0.18-0.66；n=854）。

另有 3 项系统综述和 meta 分析[45-47]报告类似有益结果，但均强调证据质量低或极低。

> [!warning]- 重要警示
> - 现有研究主要来自 **1960-80 年代**，方法学质量低，诊断和治疗方案、医疗服务可及性均已发生根本变化
> - 一项较新（方法学质量更高）的研究[44]中，抗菌治疗的益处**不那么明显**
> - 因此强烈建议参考本国指南

**结论：** 基于对母婴的有益效果，建议筛查并治疗妊娠期 ABU；但证据质量低，需结合本国指南。

### 3.3.5.3.2 疗程比较：单次 vs 短疗程

**16 项 RCT** 比较不同抗生素方案[48-63]。研究间异质性显著，疗程从单次到持续（至分娩）。按已发表的 Cochrane 综述分组策略分以下几组：单次、短疗程（2-7 天）、长疗程（8-14 天）、持续（至分娩）。仅单次 vs 标准短疗程数据有临床参考价值。

**单次 vs 短疗程疗效比较：**

| 结局 | 证据数 | 效应量（RR） | 结论 |
|------|--------|-------------|------|
| 有症状 UTI | 3 项 RCT | 1.07（95%CI：0.47-2.47；n=891） | 无显著差异 |
| 菌尿清除 | 9 项 RCT | 0.97（95%CI：0.89-1.07；n=1,268） | 无显著差异 |
| **副作用** | 6 项 RCT | **0.40（95%CI：0.22-0.72；n=458）** | 单次剂量副作用显著更少 |
| 早产 | 3 项 RCT | 1.16（95%CI：0.75-1.78；n=814） | 无显著差异 |
| **低出生体重** | 1 项 RCT | **1.65（95%CI：1.06-2.57；n=714）** | 单次剂量低出生体重发生率显著更高 |

> [!note]- 临床要点
> 单次剂量副作用更少（RR 0.40），但低出生体重风险更高（RR 1.65）。5 项 RCT、577 例患者的亚组分析显示：单次磷霉素 trometamol 治疗妊娠期 ABU 的菌尿清除率与其他抗生素无显著差异（OR 1.32，95%CI：0.78-2.22，p=0.30）。
> **推荐：标准短疗程或单次剂量磷霉素 trometamol**（强调证据质量整体偏低）。

### 3.3.5.4 有风险因素人群

| 人群 | 建议 | 证据等级 |
|------|------|---------|
| 糖尿病（控制良好） | 不推荐筛查/治疗 | 1b |
| 绝经后女性 | 不推荐治疗 | 1a |
| 养老机构老年人 | 不推荐筛查/治疗（副作用增加，无临床获益） | 1a |
| 肾移植患者 | 不推荐治疗 | 1a |
| 下尿路功能障碍/重建患者 | 不推荐治疗（ABU 菌株有保护作用） | 1a |
| 留置导尿管/输尿管支架 | 不推荐常规治疗 | - |
| 免疫抑制/重症患者 | 个体化评估 | - |
| 真菌尿 | 不推荐治疗 | - |

### 3.3.5.5 泌尿外科手术前

膀胱切除术等黏膜破裂的经尿道泌尿外科手术前：**必须尿培养**，若 ABU 阳性推荐术前治疗**（强推荐）**。

其他不进入尿路的诊断性操作：ABU 不视为风险因素，无需筛查/治疗。

### 3.3.5.6 骨科手术前（关节置换）

现有高质量证据不支持术前筛查/治疗 ABU 可降低假体关节感染（PJI），**不推荐**。

### 3.3.5.7 心血管手术前

回顾性研究提示无显著获益，**不推荐**（证据质量极低）。

### 3.3.5.8 药物管理

若决定根除 ABU，治疗应根据病原体药敏试验结果个体化制定，而非经验性治疗。

## 3.3.5 随访

无专项研究。

## 3.3.6 证据与建议汇总

| 建议 | 推荐强度 |
|------|---------|
| 以下情况不推荐筛查/治疗：无风险因素女性、控制良好的糖尿病、绝经后女性、养老机构老年人、下尿路功能障碍/重建患者、肾移植患者、关节置换术前、复发性 UTI 患者 | **强** |
| 心血管手术前不筛查/治疗 ABU | **弱** |
| 黏膜破裂的泌尿外科手术前筛查并治疗 ABU | **强** |
| 妊娠妇女筛查并治疗 ABU，推荐标准短疗程或单次磷霉素 | **弱** |

> [!warning] 重要警示
> - 男性 sex **不再**视为独立风险因素
> - ABU 治疗对复发性 UTI 患者**有害**（增加症状发作风险）

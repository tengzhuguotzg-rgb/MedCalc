## 4.1 气道、呼吸与氧合

### 推荐意见

| COR | LOE | 推荐内容 |
|-----|-----|---------|
| 1 | C-LD | 1. 急性卒中伴意识障碍或球麻痹患者，应根据需要提供气道支持和通气辅助，以维持气道通畅，保护和充分通气氧合。 |
| 1 | C-LD | 2. AIS 伴低氧患者应给予补充氧气，维持 SpO₂ >94%。 |
| 2b | B-R | 3. AIS 发病 6 小时内、NIHSS 10–20、CT ASPECTS ≥6、前循环 LVO（M1 或 ICA 末端）、计划 EVT（±IVT）的患者，EVT 前给予正常气压高氧（NBO）可能改善 90 天功能预后。 |
| 2b | B-NR | 4. AIS 由动脉空气栓塞引起，高压氧（HBO）可能改善临床预后。 |
| 3: No Benefit | B-R | 5. AIS 无低氧且不符合 EVT 条件，补充氧气**不推荐**用于改善功能预后。 |
| 3: No Benefit | B-R | 6. 不伴空气栓塞的 AIS，HBO **不推荐**改善功能预后。 |

**支持文本：**

- 低氧在卒中患者中常见（入院后某时高达 63%），与死亡风险增加相关
- SpO₂ <93% 死亡率高于 SpO₂ 93%–95%（12.5% vs 6.9%）；最佳预后下限约为 SpO₂ 96%–97%
- **NBO + EVT（正常气压高氧）：**
  - OPE-NS-2 试验：NBO 组 90 天 mRS 中位数 **2** vs 对照组 **3**（调整 cOR 1.65，P=0.018）
  - 3 项小规模 RCT 显示 NBO 可安全减少梗死体积增长
- **HBO（高压氧）：** 空气栓塞 Meta 分析（10 项研究 n=263）：HBO 开始越早预后越佳（立即开始 65% vs 延迟 15 小时 30%）

> [!warning] 氧疗警示
> - 无低氧患者常规补充氧气无获益（SO₂S 2 试验 n=8003）
> - 高氧可能有害：Liberal O₂（FiO₂ 中位 0.52）增加死亡率；卒中 ICU 患者动脉高氧与院内死亡独立相关

## 4.2 头部体位

### 推荐意见

| COR | LOE | 推荐内容 |
|-----|-----|---------|
| 3: No Benefit | B-R | 1. AIS 患者，相对 30° 体位，24 小时内常规 0° 平卧体位**无功能预后获益**。 |
| 3: No Benefit | B-R | 2. 可能为大动脉粥样硬化病因且无再灌注治疗选择的 AIS 患者，常规 Trendelenburg 体位（−20°）**无功能预后获益**。 |

**支持文本：**

- **HeadPoST 试验**（集群随机交叉，n=11283）：0° vs ≥30° 维持 24 小时，90 天 mRS 无差异；主要纳入轻症卒中（中位 NIHSS 4）
- **HOPES-2 试验**：大动脉粥样硬化患者 −20° Trendelenburg 位 2 周，90 天功能预后无差异

> [!tip] 临床提示
> 0° 或头低位体位在特定 LVO 患者 EVT 前短暂使用可能改善脑灌注，但需进一步研究证实。

## 4.3 血压管理

### 推荐意见

| COR | LOE | 推荐内容 |
|-----|-----|---------|
| **一般原则（无再灌注治疗）** | | |
| 1 | C-LD | 3. AIS 患者，低血压和低血容量应纠正以维持支持器官功能所需的全身灌注。 |
| 1 | C-EO | 4. AIS 伴需要紧急降压的合并症（急性冠脉事件、急性心衰、主动脉夹层、溶栓后 sICH、子痫）时，应早期治疗高血压。 |
| 2b | C-EO | 5. BP ≥220/120 mmHg 且未接受 IVT/EVT 且无需紧急降压的合并症患者，48–72 小时内启动或重新启动降压治疗的**获益不确定**。 |
| 3: No Benefit | A | 6. BP <220/120 mmHg 且未接受 IVT/EVT 且无需紧急降压的合并症患者，48–72 小时内启动或重新启动降压治疗**无效**。 |
| **再灌注治疗前** | | |
| 1 | B-NR | 7. BP 升高且符合 IVT 条件的患者，SBP 应降至 **<185 mmHg**、DBP **<110 mmHg** 后再启动 IVT。 |
| 2a | B-NR | 8. 计划 EVT 且未接受 IVT 的患者，术前维持 BP ≤185/110 mmHg 是合理的。 |
| **IVT 后** | | |
| 1 | B-R | 9. IVT 后至少 24 小时内维持 BP **<180/105 mmHg**。 |
| 3: No Benefit | B-R | 10. 轻至中度 AIS 伴 IVT 患者，强化降压（目标 <140 mmHg vs <180 mmHg）**不改善**功能预后，不推荐。 |
| **EVT 后** | | |
| 2a | B-NR | 11. EVT 患者，术中及术后 24 小时内维持 BP ≤180/105 mmHg 是合理的。 |
| 3: Harm | A | 12. 前循环 LVO 成功 EVT（mTICI 2b/2c/3）且无其他 BP 管理指征的患者，首个 72 小时内强化降压目标 <140 mmHg **有害**，不推荐。 |

**支持文本：**

- **BP <220/120 mmHg**：48–72 小时内强化降压无预防死亡或依赖的证据（多项 RCT）
- **IVT 后强化降压**：ENCHANTED 试验证明强化降压（目标 <140 mmHg）不改善功能预后且 sICH 风险数值更高
- **EVT 后强化降压**：BP SPEED 试验证明 72 小时内强化 SBP <140 mmHg **有害**（更多早期神经功能恶化、更差的功能预后）

> [!warning] EVT 后血压警示
> 成功取栓后（前循环 LVO，mTICI ≥2b），首个 72 小时内强化 SBP <140 mmHg 目标**有害，不推荐**。
> 合理目标：**≤180/105 mmHg**（术中及术后 24 小时）。

## 4.4 体温管理

### 推荐意见

| COR | LOE | 推荐内容 |
|-----|-----|---------|
| 1 | B-R | 1. AIS 伴发热（体温 >37.5°C）应积极查找感染源并治疗，同时考虑退热治疗改善神经预后。 |
| 2b | C-EO | 2. AIS 伴体温过低（轻度低温 32–35°C）的安全性和获益尚未确立，在临床试验之外**不推荐**。 |

**支持文本：**

- 多项 RCT 和大样本观察性研究显示：卒中早期高体温与更差的功能预后和更高的死亡率相关
- 护士驱动的发热监测和治疗协议有 RCT 证据支持，可改善短期和长期预后及死亡率
- 诱导低温在正常体温患者中未显示获益，反而可能增加不良事件

## 4.5 血糖管理

### 推荐意见

| COR | LOE | 推荐内容 |
|-----|-----|---------|
| 1 | C-LD | 3. AIS 伴高血糖，住院期间应持续监测血糖，维持**140–180 mg/dL** 范围内是合理的。 |
| 3: No Benefit | B-R | 4. AIS 伴高血糖，IV 胰岛素治疗将血糖控制在 80–130 mg/dL 范围**不改善** 3 个月功能预后，不推荐。 |

**支持文本：**

- **SHINE 研究**：强化血糖控制（80–130 mg/dL）vs 标准控制（80–179 mg/dL），3 个月功能预后无差异
- 严重低血糖（<40 mg/dL）：强化组 **11.2%** vs 标准组 **3.4%**（增加 3 倍以上）
- 低血糖可模拟卒中样表现，且与死亡率和不良预后呈 J 型关系

> [!tip] 血糖管理目标
> **合理目标：140–180 mg/dL**，避免低血糖（<70 mg/dL）

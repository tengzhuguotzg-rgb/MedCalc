## 4.8 抗血小板治疗

### 推荐意见

| COR | LOE | 推荐内容 |
|-----|-----|---------|
| 1 | A | 1. 非心源性 AIS，推荐**早期启动抗血小板治疗**（发病 24–48 小时内），降低死亡率和残疾。 |
| — | B-R | 2. 阿司匹林（160–325 mg 负荷）用于急性缺血性卒中抗血小板治疗。 |
| 2a | B-R | 3. 轻型卒中（NIHSS ≤3）或高危 TIA（ABCD² ≥4），发病 24 小时内启动 **DAPT（阿司匹林+氯吡格雷）21 天**，之后改为单药，可减少 90 天卒中复发。 |
| 2a | B-R | 4. 发病 24 小时内且基线 DWI 病变体积 <25 mL 的轻型非致残性卒中，DAPT 可能是一线替代方案（等效于 IVT）。 |
| 2b | B-R | 5. 发病 24 小时内不适合 DAPT 的患者，**阿司匹林+西洛他唑** 联合治疗可能优于单用阿司匹林。 |
| 2b | B-NR | 6. 阿司匹林过敏患者，**氯吡格雷单药** 是合理替代。 |
| 2b | B-NR | 7. 已知阿司匹林抵抗或阿司匹林治疗失败患者，氯吡格雷或西洛他唑可作为替代。 |
| 3: Harm | B-R | 8. AIS 发病 24 小时内，**口服抗凝药（DOAC/华法林）单药** 不推荐作为初始治疗（增加出血风险）。 |

**DAPT 重要时限（POINT / CHANCE 试验）：**

> [!warning] DAPT 时限警示
> DAPT 获益集中于**前 21–30 天**；超过 90 天后双抗血小板治疗的出血风险超过获益，不推荐长期 DAPT。

## 4.9 抗凝治疗

### 推荐意见

| COR | LOE | 推荐内容 |
|-----|-----|---------|
| **心房颤动相关卒中** | | |
| 1 | A | 1. 非瓣膜性房颤相关急性缺血性卒中，推荐**新型口服抗凝药（DOAC）** 优于华法林。 |
| 1 | C-EO | 2. 房颤相关 AIS，启动口服抗凝的时机应个体化，平衡再发卒中风险与出血转化风险（出血转化风险高的患者延迟抗凝）。 |
| 2a | B-NR | 3. 房颤相关 AIS，**小缺血核心 + 无显著出血转化**（出血性梗死 1 型或 2 型），发病 **3 天内** 启动 DOAC 合理。 |
| 2a | B-NR | 4. 房颤相关 AIS，**中等缺血核心或出血转化风险较高**（PH 型出血或 HI-2），发病 **7–8 天内** 启动 DOAC 合理。 |
| 2b | C-LD | 5. 房颤相关 AIS，**大面积梗死**（ASPECTS ≤5 或 DWI 病变体积 >100 mL）或需要半球减压术的患者，发病 **10–14 天后** 启动 DOAC 是合理的。 |
| **华法林** | | |
| 2b | B-NR | 6. 机械心脏瓣膜相关 AIS 或中重度二尖瓣狭窄相关房颤，**华法林**（INR 2.0–3.0）是抗凝首选。 |
| **AIS 伴 DOAC 暴露** | | |
| — | C-EO | 7. AIS 伴近期 DOAC 暴露（<48 小时）且有致残性缺陷，若无逆转药物且肾功能正常，IVT 可能是合理选择。 |
| — | C-EO | 8. AIS 伴 DOAC（直接 Xa 因子抑制剂或直接凝血酶抑制剂），已知或怀疑抗凝相关出血转化，**andexanet alfa**（抗 Xa 逆转剂）或 **idarucizumab**（达比加群逆转剂）可能获益。 |
| **AIS 后启动抗凝的影像评估** | | |
| 2b | C-EO | 9. AIS 后启动抗凝决策，应参考初始脑影像评估缺血核心体积和出血转化程度。 |

**心房颤动相关 AIS 抗凝时机——决策框架：**

| 缺血严重程度 | 影像特征 | 推荐启动时机 |
|-------------|---------|------------|
| 小核心 + 无显著出血 | HI-1/2 或无出血 | 发病 **3 天内** |
| 中等核心 | 较大 DWI 病变，HI-1/2 | 发病 **7–8 天内** |
| 大核心 | ASPECTS ≤5，DWI >100 mL | 发病 **10–14 天后** |
| 需要去骨瓣减压 | — | 发病 **10–14 天后** |

**DOAC 逆转药物：**

| DOAC 类型 | 逆转药物 |
|-----------|---------|
| 抗 Xa 抑制剂（利伐沙班、阿哌沙班、艾多沙班） | **Andexanet alfa** |
| 直接凝血酶抑制剂（达比加群） | **Idarucizumab** |

# ESICM-ARDS-8-神经肌肉阻滞剂

## 问题8.1：常规连续输注NMBA vs 无常规NMBA（中重度ARDS）

### 背景

NMBA可减少呼吸功、改善人机不同步，可能影响预后。但长期使用与ICU获得性肌无力相关，且需要深度镇静（本身可导致不良结局）。

2010年ACURASYS试验（中重度ARDS，PaO₂/FiO₂ < 150 mmHg + PEEP ≥ 5）：48小时NMBA输注，调整后死亡率显著降低。另有3项小规模试验显示类似获益。

但近年ICU实践演变，更强调浅镇静和早期恢复自主呼吸。2019年ROSE试验：同样48小时NMBA输注+深度镇静 vs 常规护理（无常规NMBA + 浅镇静目标），90天死亡率无显著差异。

**关键差异**：
- 俯卧位使用：ROSE 15.8% vs ACURASYS 44.8%
- 对照组镇静深度：ROSE更浅（严重心血管事件更少）
- PEEP策略：ROSE使用更高PEEP

COVID-19期间NMBA使用增加（消除强烈自主努力、减少可能加重P-SILI的高跨肺压），但无COVID-19 RCT。

### 证据总结

5项试验（28天/ICU死亡率）：汇总RR 0.80（95% CI 0.57–1.04），p=0.086，无显著降低。

2项大型试验比较（90天死亡率）：
- ACURASYS：保护效应（但样本量计算基于15%死亡率降低，实际9%不显著，检验效能仅42%）
- ROSE：无显著结果

Meta分析汇总：高异质性（I²=56%），总体证据质量中等，反对使用。

**气压伤保护作用**：4项研究中NMBA组气胸发生率显著低于对照组。

### 推荐8.1

**强推荐反对**：常规连续输注NMBA降低中重度ARDS（非COVID-19）患者死亡率。（强推荐；中等质量证据）

**无推荐**：无法对中重度ARDS（COVID-19）常规连续输注NMBA降低死亡率做出推荐。（无推荐；无证据）

### 专家临床应用意见

ACURASYS试验中，无NMBA时使用俯卧位需要更深度镇静，可能促进了对照组死亡率增加。NMBA组对气胸的明确保护作用可能支持在高气压伤风险患者中使用NMBA。

### 未解决的问题和研究缺口

未来研究应优先关注其他结局：成功拔管、再插管、麻痹回忆、ICU获得性肌无力、健康相关生活质量，以及NMBA在俯卧位中的特定作用。识别人机不同步差的患者也很重要（可能是NMBA的潜在适应证）。

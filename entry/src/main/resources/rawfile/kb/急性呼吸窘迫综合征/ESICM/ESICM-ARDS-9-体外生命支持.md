# ESICM-ARDS-9-体外生命支持

## 问题9.1：VV-ECMO vs 传统通气（严重ARDS）

### 背景

VV-ECMO用于支持或替代气体交换。血液通过"膜肺"进行氧合和CO₂扩散。技术进步改善了气体交换并减少了并发症。严重ARDS患者（包括COVID-19）使用ECMO，高容量经验中心报告更好结局。ECMO需要与肺保护性通气和俯卧位联合的整体照护包。

### 证据总结

2项RCT（CESAR和EOLIA）：
- **CESAR**（n=180）：主要终点死亡或重度残疾
- **EOLIA**（n=249）：PaO₂/FiO₂ < 50 mmHg（>3h）或 < 80 mmHg（>6h）或pH < 7.25 + PaCO₂ ≥ 60 mmHg（>6h），平台压 ≤ 32 cmH₂O

EOLIA入组患者平均PaO₂/FiO₂约75 mmHg，均为严重ARDS。CESAR存在高偏倚风险（干预组约1/4未接受ECMO）。

**Meta分析**：VV-ECMO组60天死亡率显著降低（RR 0.72，95% CI 0.57–0.91，中等质量证据）。90天死亡率及60天死亡+治疗失败复合终点保护效应一致。观察性研究（含EOLIA后验贝叶斯分析）大多确认ECMO保护效应。

COVID-19：无RCT。观察性研究显示ECMO对短期生存的保护效应，但证据间接。

**ECMO风险**：EOLIA报道干预组出血事件和严重血小板减少发生率更高，但缺血性卒中减少（绝对风险降低5%）。

### 推荐9.1

**强推荐**：符合EOLIA试验入组标准的严重ARDS（非COVID-19）患者，应在满足明确组织标准的ECMO中心接受治疗，采用与EOLIA试验相似的管理策略。（强推荐；中等质量证据）

COVID-19相关严重ARDS同样适用。（强推荐；低质量证据，间接性）

### 专家临床应用意见

有效提供ECMO需要具备该技术经验的网络ECMO中心。从非ECMO中心集中患者需要具备ECMO转运团队（移动ECMO）。严重ARDS的COVID-19患者在有经验中心早期死亡率与非COVID相似，但长期多维残疾可能显著（虽然尚不清楚是ECMO还是严重ARDS本身导致）。

### 问题9.2：体外二氧化碳清除（ECCO₂R）vs 传统通气

### 背景

ECCO₂R通过体外回路去除CO₂。使用较低血流量（通常200–1500 mL/min），因为去除CO₂所需血流量远低于实现充分氧合所需。ECCO₂R的主要目的是促进有害机械通气的减少。

### 证据总结

2项RCT（Xtravent和REST）：
- **Xtravent**（n=79，PaO₂/FiO₂ < 200 mmHg）：无泵动脉-静脉（1–2 L/min）CO₂去除
- **REST**（n=412，PaO₂/FiO₂ < 150 mmHg，约60%为ARDS）：静脉-静脉低流量（~450 mL/min）CO₂去除

Meta分析：ECCO₂R未降低死亡率（RR 1.03，95% CI 0.82–1.3，高质量证据）。ECCO₂R组28天机械通气免费天数更少（均数差 −1.21天，95% CI −3.77至1.34，中等质量证据）。

**REST试验严重不良事件**：9例（4.5%）脑出血，6例（3%）颅外出血（对照组分别为0和1例[0.5%]）。

### 推荐9.2

**强推荐反对**：在RCT之外，不推荐使用ECCO₂R治疗非COVID-19 ARDS以降低死亡率。（强推荐；高质量证据显示无效果）

COVID-19相关严重ARDS同样适用。（强推荐；中等质量证据显示无效果）

### 专家临床应用意见

Xtravent血流量1–2 L/min，REST约450 mL/min。较低血流量（约500 mL或约25% CO₂清除）可能不足以实现充分减少有害通气。REST试验中ECCO₂R资源需求估计与CRRT相当；但更高血流量需要与ECMO中心类似的能力。

### 未解决的问题和研究缺口

尽管当前证据不支持ECCO₂R有效性，但其作用仍不确定。需要进一步研究确定特定可能获益的ARDS人群。技术可能取决于设备和安全性。正在进行中。

# ESICM-ARDS-5-低潮气量通气

## 问题5.1：低潮气量通气 vs 传统大潮气量通气

### 背景

1960年代初，临床使用大Vt（12–15 ml/kg）预防低氧血症，但导致肺容积逐步丧失。后续认识转变推动了肺保护性通气：

1. **高碳酸血症和呼吸性酸中毒**：若患者氧合充足，可良好耐受
2. **VILI机制**：允许肺泡塌陷-复张和/或过度膨胀的正压通气可加重现有肺损伤或导致新发肺损伤
3. **"婴儿肺"概念**：ARDS患者有效通气肺容积减小，"正常Vt"也可导致过度膨胀

**结论**：ARDS患者需要肺保护性通气策略——小Vt（4–8 ml/kg PBW）+ 适当PEEP水平，以限制肺扩张和萎缩伤。

### 证据总结

纳入7项RCT，主要分析（3项高质量试验）：死亡率**无显著差异**（RR 0.96，95% CI 0.72–1.28，p=0.768）。二次分析（全7项）：RR 0.82（95% CI 0.66–1.02，p=0.069）。机械通气免费天数和气压伤在各试验中均无显著差异。

**重要方法学说明**：
- 1项试验未正式发表，降级处理
- 2项试验使用明确协议将干预组PEEP保持在低位拐点以上2 cmH₂O或13 cmH₂O，样本小，提前终止，夸大效应量
- 各试验Vt计算方式不统一（ABW/IBW/PBW混用），组间达到的梯度也不同

### 推荐5.1

**强推荐**：非COVID-19 ARDS患者使用低潮气量通气策略（4–8 ml/kg PBW），对比传统大Vt（以血气正常化为目标），以降低死亡率。（强推荐；高质量证据，尽管统计不显著，但生理学依据极强）

**强推荐**：COVID-19相关ARDS同样适用。（强推荐；中等质量证据，中度间接性）

### 专家临床应用意见

使用RCT时，重要的是强调：当气道压力达到限值时才减少Vt。对照组使用的Vt已不能称为"传统"。此外，该领域缺乏共识，未来不太可能进行其他RCT。

目前支持ARDS患者的标准方法：限制Vt在4–8 ml/kg PBW，保持平台压 < 30 cmH₂O。理想体重的计算应使用PBW而非IBW（两者不应互换）。

COVID-19患者虽未纳入RCT，但呼吸力学相似，VILI生理机制相同，低Vt有生物学合理性。但需注意：COVID-19患者严重长期多维残疾发生率高，可能因长时间深度镇静和/或麻痹而进一步加重。

### 未解决的问题和研究缺口

1. 评估某种通气策略是否可能加重VILI的最佳方式
2. 如何确定最佳Vt（基于PBW、驱动压或其他方法）
3. 基于个体患者生理的个性化肺保护性通气策略
4. 当当前通气策略很可能加重VILI时，移除PaCO₂的其他方法

未来研究应纳入患者和家属的长期多维结局，患者和照护者的意见应成为确定未来研究问题和结局的核心。

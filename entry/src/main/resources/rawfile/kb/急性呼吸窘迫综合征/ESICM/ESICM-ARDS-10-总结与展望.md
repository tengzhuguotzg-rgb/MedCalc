# ESICM-ARDS-10-总结与展望

## 指南总结

本指南提出**21条循证推荐**，涵盖ARDS定义、表型分类及呼吸支持策略。

### 推荐强度速查

| 领域 | 推荐数量 | 关键推荐 |
|------|---------|---------|
| HFNO vs COT | 1强+1无推荐 | 强推荐HFNO降低插管率 |
| HFNO vs NIV | 1无推荐 | 无推荐；COVID-19弱推荐CPAP/NIV替代HFNO |
| CPAP/NIV vs COT | 1无推荐+1弱推荐 | COVID-19弱推荐CPAP；头盔CPAP弱推荐 |
| 低潮气量通气 | 1强推荐 | 强推荐低Vt（4–8 ml/kg PBW） |
| PEEP策略 | 2无推荐 | PEEP/FiO₂表和呼吸力学指导均无推荐 |
| 肺复张操作 | 2强推荐反对 | 反对持续高压RM和短暂高压RM |
| 俯卧位 | 2强推荐 | 强推荐中重度俯卧位≥16h；清醒俯卧位弱推荐 |
| NMBA | 1强推荐反对 | 反对常规连续输注NMBA |
| VV-ECMO | 1强推荐 | 强推荐严重ARDS使用ECMO |
| ECCO₂R | 1强推荐反对 | 强推荐反对常规使用ECCO₂R |

### 与2017年指南比较

本指南相比2017年版本（Fan et al., AJRCCM 2017）在范围和推荐上均有更新：

- **新增Domain**：表型分类（Domain 2）、HFNO（Domain 3）、CPAP/NIV（Domain 4）作为独立章节
- **更新推荐**：俯卧位、低潮气量、ECMO推荐等级调整
- **COVID-19**：每个推荐均明确COVID-19适用性
- **21条推荐 vs 16个Recommendation块**：部分块含多条独立声明

## 未来研究重点

1. **ARDS定义改进**：结合影像学和生物标志物的前瞻性研究，AHRF作为过渡综合征
2. **亚表型**：稳定性、可重现性、快速分类准确性、精准治疗策略
3. **HFNO**：最佳输送和停用方式，长期功能结局
4. **CPAP/NIV**：P-SILI预防，最佳患者选择
5. **PEEP个体化**：复张可能性测试，食道压指导，高PEEP的血流动力学代价
6. **俯卧位**：应答预测指标，最佳持续时间和停止时机
7. **NMBA**：在其他结局（成功拔管、再插管、ICU获得性肌无力）中的角色
8. **ECMO/ECCO₂R**：长期多维结局，ECMO特异性并发症

## 临床实施建议

- 遵循指南推荐的循证策略
- 注意患者亚表型和个体差异
- 在经验中心建立ECMO网络
- 关注患者和照护者意见，将其作为研究问题和结局的核心

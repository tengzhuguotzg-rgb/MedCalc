# ATS-ARDS-2-与柏林定义对比

## 五项关键更新

| 柏林定义 | 更新内容 | 全球定义如何解决 |
|----------|---------|----------------|
| 急性发作在已知损伤或新发/恶化呼吸道症状1周内 | 部分损伤（如COVID-19）起病可能更缓慢 | HFNO纳入将捕获更多缓慢病程患者，时限标准未改变 |
| 双侧胸片或CT影像学改变，不完全由胸腔积液、肺不张或结节解释 | 胸片和CT并非所有临床环境可用 | 超声可识别含气丧失（多B线/实变），条件是操作者经过充分培训 |
| 三级严重程度由PaO2:FiO2定义 | SpO2:FiO2作为PaO2:FiO2替代指标已广泛验证使用 | SpO2:FiO2可用于诊断和严重程度评估（限SpO2≤97%） |
| 所有严重程度（除轻度）要求NIV或MV + PEEP≥5cmH2O | HFNO在低氧血症患者中的应用随FLORALI试验后大幅增加 | 新增**非插管ARDS**类别：HFNO≥30L/min即可满足标准 |
| 有创和无创机械通气在资源受限环境不可用 | 资源受限环境改良定义不需要PaO2:FiO2、PEEP或HFNO | Kigali改良版正式纳入全球定义 |

## 每项更新的详细理由

### HFNO 纳入

- FLORALI试验（2015）后HFNO使用大幅增加
- COVID-19期间大量患者接受HFNO治疗
- 研究显示93%接受HFNO的COVID-19患者在插管后仍满足ARDS氧合标准
- HFNO30L/min可提供低水平PEEP（约5cmH2O）
- 早期识别ARDS可使早期干预试验更可行

### SpO2:FiO2 替代

- ABG在资源受限环境可用性不一致，高收入国家ABG监测频率也在下降
- Rice线性方程计算SpO2:FiO2截断值，敏感性和特异性与非线性插补相当
- SpO2≤97%时SpO2:FiO2才有效（氧解离曲线特征）
- **局限性**：脉氧仪在深肤色和休克患者中可能敏感性不足；深肤色患者隐匿性低氧血症发生率更高
- 若临床高度怀疑ARDS但脉氧未达标，建议获取ABG

### 超声纳入

- 超声特别适用于资源受限环境（放射学不常规可用时）
- 培训合格的操作者可可靠地识别与非心源性肺水肿或实变相关的双侧含气丧失征象
- 局限：单独依赖B线和/或实变可能导致过度敏感和适度特异性
- 建议将超声发现与胸膜线异常等其他超声征象整合

### 资源受限环境（Kigali改良）

- 不要求PEEP、具体氧流量或呼吸支持设备
- 允许任何呼吸设备（而非要求HFNO、NIV或MV）
- 但一致认为在所有环境中允许任何呼吸设备会损害ARDS作为危重症综合征的面容效度
- 因此创建单独的资源受限环境类别

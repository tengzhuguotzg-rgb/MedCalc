# ATS-ARDS-5-总结

## 共同诊断标准（所有类别适用）

| 标准 | 内容 |
|------|------|
| 诱因 | 急性易感危险因素（肺炎、非肺部感染、外伤、输血、误吸、休克等）；肺水肿非主要归因于心源性/液体负荷 |
| 时限 | 1周内急性发作或恶化 |
| 影像 | 双侧胸片/CT/超声改变，不完全由胸腔积液、肺不张或结节解释 |

## 三类 ARDS 严重程度

### 非插管 ARDS（新增）

| 标准 | 阈值 |
|------|------|
| HFNO | ≥30L/min |
| PaO2:FiO2 | ≤300mmHg |
| SpO2:FiO2 | ≤315（限SpO2≤97%） |

### 插管 ARDS（沿用柏林体系，补充SpO2截断值）

| 严重程度 | PaO2:FiO2（mmHg） | SpO2:FiO2（SpO2≤97%） | PEEP要求 |
|----------|-------------------|----------------------|---------|
| 轻度 | 200< ≤300 | 235< ≤315 | ≥5cmH2O |
| 中度 | 100< ≤200 | 148< ≤235 | ≥5cmH2O |
| 重度 | ≤100 | ≤148 | ≥5cmH2O |

### 资源受限环境（Kigali改良，正式纳入）

| 标准 | 阈值 |
|------|------|
| SpO2:FiO2 | ≤315（限SpO2≤97%） |
| 要求 | 无PEEP/氧流量/呼吸设备要求 |

## 与柏林定义的五项关键更新

| # | 柏林定义 | 全球定义更新 |
|---|---------|------------|
| 1 | 时限1周 | 不变（HFNO纳入可早期识别） |
| 2 | 仅胸片/CT | **+超声**（培训合格操作者） |
| 3 | 仅PaO2:FiO2 | **+SpO2:FiO2**（SpO2≤97%） |
| 4 | NIV/MV + PEEP≥5 | **+非插管HFNO≥30L/min**（新增类别） |
| 5 | 资源受限不可用 | **Kigali改良正式纳入** |

## SpO2:FiO2 使用要点

- **仅SpO2≤97%有效**；>97%时无效（氧解离曲线）
- 估算FiO2 = 0.21 + 0.03 × 氧流量（L/min）
- 海拔>1000m：校正 = （PaO2或SpO2）/FiO2 × （大气压/760）
- 深肤色/休克患者：建议获取ABG

## FiO2 估算

```
估计FiO2 = 环境FiO2（如0.21）+ 0.03 × 氧流量（L/min）
例：氧流量6L/min → FiO2 ≈ 0.21 + 0.18 = 0.39
```

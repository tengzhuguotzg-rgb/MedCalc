# ATS-ARDS-3-各论

## 概念模型

委员会同意柏林定义的概念模型（微调后）继续反映当前的病理生理学理解和证据：

- **本质**：急性、弥漫性、炎症性肺损伤
- **诱因**：肺炎、非肺部感染、外伤、输血、烧伤、误吸或休克
- **病理生理**：肺血管和肺泡上皮通透性增加 → 肺水肿；重力依赖性肺不张 → 含气肺组织丢失
- **临床特征**：动脉低氧血症 + 双侧影像学透光度降低，伴分流增加、死腔增加、顺应性下降
- **可受医学管理影响**：PEEP初始设置、液体管理策略、镇静和神经肌肉阻滞、俯卧位
- **组织学发现**：多样，可有肺泡水肿、炎症、透明膜形成、肺泡出血（弥漫性肺泡损伤），但并非总存在，非临床诊断必要条件

与柏林定义的区别：强调缺乏一致的组化发现或生物标志物，以及初始临床管理对ARDS表现的重要性。

## 时限

**维持1周标准**：急性发作或低氧性呼吸衰竭恶化定义为在易感危险因素发作或新发/恶化呼吸道症状后1周内发生。

- COVID-19等部分损伤的起病可能更缓慢，但HFNO纳入允许更早期识别，因此时间窗未改变

**排除条件**：低氧血症和肺水肿**不主要**归因于心源性肺水肿或液体负荷过多、肺不张或胸腔积液、PE。ARDS可在这些疾病存在时诊断，前提是同时存在ARDS易感危险因素，且临床医生认为这些疾病（如液体负荷过多）不太可能是低氧血症的主要原因。

**合并慢性肺病**：可在COPD、间质性肺病或肺动脉高压患者中诊断，前提是急性低氧性呼吸衰竭不主要归因于这些基础疾病。

## 影像学

### 双侧影像学标准

- **胸片**：最常用但观察者间可靠性差（ARDS识别率低、LUNG SAFE研究）
- **CT**：更敏感但危重患者实际不可行
- **超声**：可接受作为识别含气丧失征象的影像学方式，条件是操作者经过充分培训

### 超声标准

双侧B线征和/或实变影，提示非心源性肺水肿或肺实变，**不完全**由胸腔积液、肺不张或结节/肿块解释。

**培训要求**：操作者须经过识别双侧实变和非心源性肺水肿的充分培训。

**优势与局限**：
- 特别适用于资源受限环境
- 若操作者培训充分，可靠性高
- 单独依赖B线和/或实变可能过度敏感、特异性中等
- 与胸膜线异常等超声征象整合可提高诊断准确性

## 氧合

### 三分类体系

**非插管 ARDS**（新增类别）：
- HFNO≥30L/min 或 NIV/CPAP PEEP≥5cmH2O
- PaO2:FiO2≤300mmHg 或 SpO2:FiO2≤315（SpO2≤97%）
- 意义：允许在插管前即识别ARDS

**插管 ARDS**：
- 最低 PEEP 5cmH2O
- 轻度：200<PaO2:FiO2≤300；中度：100<≤200；重度：≤100mmHg
- 对应SpO2:FiO2截断值：235/148

**资源受限环境**：
- SpO2:FiO2≤315（SpO2≤97%）
- 不需要PEEP、HFNO或呼吸支持设备
- Kigali改良版正式纳入

### SpO2:FiO2 的使用条件

1. **SpO2≤97%**：超过此值SpO2:FiO2不能反映氧合状态（氧解离曲线特征）
2. **Rice线性方程**：PaO2:FiO2 = 4.88 × SpO2:FiO2 − 27.79（敏感性/特异性与非线性插补相当）
3. **ABG获取指征**：临床高度怀疑ARDS但SpO2:FiO2未达标；分类错误会影响管理决策或临床试验入组
4. **肤色偏差**：深肤色患者隐匿性低氧血症发生率高出4倍，建议有疑问时获取ABG

## 讨论要点

- ARDS亚表型（高炎症/低炎症）未整合入当前定义，需要前瞻性验证
- ARDS是动态综合征——患者可快速改善或持续病程，研究者可根据研究目的选择特定人群
- LUNG SAFE显示40% ARDS未被识别——新定义需要类似的可靠性评估
- 长期功能结局未被纳入初始诊断——未来研究优先级
- NHLBI已资助六大学术中心联盟，前瞻性研究ARDS/肺炎/脓毒症的临床和生物学决定因素

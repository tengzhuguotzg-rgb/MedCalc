# ATS-指南-3-NMBA

## PICO 3 推荐

**推荐内容**：建议对早期重度ARDS患者使用神经肌肉阻滞剂。

**推荐强度**：条件性推荐 | **证据确定性**：低

## 背景

- NMBA是ARDS常用辅助治疗
- 可能的获益机制：减少呼吸机诱导肺损伤（通过改善人机同步）、减少氧消耗、减轻炎症、减少肺泡液体
- 2017年指南未涉及此议题

## 证据总结

- **7项RCT，1,598例患者**
- **死亡率**：NMBA可能降低中重度ARDS患者死亡率（RR 0.74；95% CI 0.56–0.98；低确定性）
  - 亚组：与深度镇静相比降低死亡（RR 0.72）；与浅镇静相比无差异（RR 0.99）
- **气压伤**：可能减少（RR 0.55；中等确定性）
- **无呼吸机天数**：可能略增加（低确定性）
- **ICU获得性肌无力**：可能增加（RR 1.16；中等确定性）

## 实施要点

- 推荐限于**早期**（ARDS发病 <48小时）**重度**（PaO2/FiO2 ≤100 mmHg）ARDS
- 推荐在深度镇静下使用，或在浅镇静下仍有显著呼吸机不同步伴临床恶化时使用
- 顺式阿曲库铵在最大规模试验中使用，可能有抗炎额外获益
- 持续输注或间歇推注均可；疗程不超过48小时
- 需平衡ICU获得性肌无力风险

## 不确定性

- 最优NMBA药物选择
- 启动时机（早期 vs 晚期；符合标准后立即 vs 稳定后）
- 剂量（部分阻滞 vs 完全阻滞；间歇 vs 持续）
- 长期结局数据不足

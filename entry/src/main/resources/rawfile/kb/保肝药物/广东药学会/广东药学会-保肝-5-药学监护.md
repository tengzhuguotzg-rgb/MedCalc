## 1. 药物相互作用

### 甘草酸二铵、复方甘草酸苷、异甘草酸镁

- **排钾利尿药**（呋塞米、氢氯噻嗪等）：增加甘草酸二铵的排钾作用，增加低钾血症风险 → 注意监测血清钾值
- **莫西沙星**：甘草酸苷排钾作用可引起血钾下降，与莫西沙星合用可致室性心动过速（包括尖端扭转型室速）、QT间期延长 → 合用须谨慎

### 还原型谷胱甘肽

- 与多种药物存在**配伍禁忌**：维生素B₁2、甲萘醌、泛酸钙、乳清酸、抗组胺制剂、多烯磷脂酰胆碱（均不宜混合使用）
- 青霉素、氨苄青霉素、先锋霉素、红霉素乳糖酸盐、四环素：存在配伍禁忌，须分别滴注，冲管

### N-乙酰半胱氨酸（NAC）

- 可**降低部分抗生素活性**，不宜配合使用或混用，给药时间应与抗生素间隔至少1小时
- 可**降低卡马西平清除**，增加其血药浓度，增加癫痫发作风险 → 密切监测卡马西平血药浓度
- 可**增强硝酸甘油扩张动脉效应**，导致严重低血压和头痛 → 合用时监测血压

### 熊去氧胆酸

- **考来烯胺、考来替泊**：阻碍熊去氧胆酸吸收，影响疗效，如必须合用应间隔至少2小时
- **磺酰脲类口服降糖药**：熊去氧胆酸可抑制血清白蛋白与磺酰脲类结合，可能增强降血糖作用

### 其他

- 硫普罗宁：与具有氧化作用的药物联用须注意
- 水飞蓟宾：与甲硝唑的相互作用尚不明确
- 双环醇：可能降低环孢素血药浓度，谨慎合用
- 联苯双酯：与肌苷合用可减少联苯双酯的降酶反跳现象

## 2. 不良反应

保肝药物不良反应大多症状较轻，不影响治疗（如胃肠道反应：纳差、恶心、呕吐、腹胀等，一般减量或停药后可消失；静脉制剂可减慢输液速度以减少用药部位不良反应）。

**各类系统器官分类及主要表现**：

| 系统器官分类 | 不良反应 |
|-------------|---------|
| 心脑血管 | 心悸、血压升高、心律失常 |
| 代谢/内分泌 | 水钠潴留、水肿 |
| 免疫 | 瘙痒、荨麻疹、皮疹、过敏性休克 |
| 神经系统 | 头痛、头晕 |
| 胃肠系统 | 纳差、恶心、呕吐、腹胀、腹痛、腹泻 |
| 呼吸系统 | 呼吸困难、咳嗽 |
| 全身 | 胸闷、胸痛、高热、寒战、面部潮红、横纹肌溶解 |
| 电解质 | 低钾血症、高钾血症、高钠血症 |
| 其他 | 注射部位静脉炎、注射部位疼痛 |

> 注：心悸、血压升高、瘙痒、荨麻疹、皮疹、恶心、呕吐、头晕、面部潮红为常见不良反应。

## 2.1 假性醛固酮症

长期持续服用甘草和含有甘草酸制剂可导致：

**机制**：收缩压和舒张压升高，以及血浆钾、血浆肾素活性和血浆醛固酮的抑制

**临床表现**：血钾降低、血压升高、血浆肾素活性与醛固酮水平降低

**高风险人群**：老年患者、肾脏功能受损、心脑血管疾病患者

**处理**：
- 密切监测电解质、血压、醛固酮水平
- 如出现严重不良反应，应根据病情**停药或调整改用其他保肝药物**

## 2.2 过敏反应

甘草酸制剂、硫普罗宁、门冬氨酸钾镁发生率较高。

**轻度过敏反应**：皮肤瘙痒、荨麻疹、皮疹等
- 减量或停药可消失
- 可选择抗组胺药物治疗

**严重过敏反应（过敏性休克）**：血压下降、意识不清、呼吸困难、心肺衰竭、面部浮肿
- 用药期间应密切监测
- 一旦发现异常应**立即停药并给予处理**

# 低镁血症治疗与管理

## 推荐意见7（Ⅱa，推荐，A）

低镁血症的管理应根据不同专科患者的特点**个体化制定**。定期检测血镁并根据低镁血症的程度适时补充；注意药物相互作用；补镁治疗应**从低剂量开始**。儿童按体重精确计算剂量；孕妇使用硫酸镁需遵循产科指南。

---

## 常规管理

**治疗适应证：**
- 明确诊断的低镁血症伴神经肌肉兴奋增强、抽搐或心律失常
- 难治性低钾血症或低钙血症（镁缺乏常为潜在诱因）
- ICU/术后患者、高强度利尿剂/胺碘酮/PPI使用者的预防性应用
- 糖尿病、慢性胃肠道疾病、慢性酒精依赖或营养不良者

**口服补充：**
| 制剂 | 剂量 |
|------|------|
| 氧化镁 | 250\~500 mg/d，分2\~3次 |
| 甘氨酸镁 | 生物利用度更高 |

**静脉补充：**
| 情形 | 方案 |
|------|------|
| 轻度至中度缺乏 | 硫酸镁1\~2 g，稀释后静滴≥60 min |
| 重度或症状性缺乏 | 初始：硫酸镁1\~2 g于5\~60 min缓慢输注；后续：4\~8 g于12\~24 h持续静滴 |
| 特殊情形（如TdP） | 硫酸镁2 g于2\~15 min静脉推注，后持续输注 |

## 各专科应用特点

**心血管科：** 冠脉介入、心外科手术、抗心律失常药物（尤其胺碘酮）前建立血镁基线；TdP一线用药硫酸镁；术前补镁可降低术后房颤；出院后血镁+心电随访。

**肾脏科：** 警惕"总镁正常而iMg不足"；透析/CRRT人群可因滤液丢失持续低镁；补镁小剂量缓速间断，严密监测；透析液镁浓度建议升至1 mmol/L。

**重症医学科：** ICU发生率20%\~65%；iMg作为首选干预指标，联合血钾、血钙与心电高频动态监测；早期个体化补镁（与补钾/补钙同频校准）；完成"ICU→过渡病房→门诊"闭环管理。

**内分泌/糖尿病科：** 初诊即建立血镁基线；PPI使用超3个月者定期评估血镁；以口服为主，症状明显或高危时静脉序贯；每3\~6个月随访（高危用药者缩短间隔）。

**老年医学科：** 血镁纳入老年综合评估；"低剂量起步、慢速递增"原则；口服氧化镁250\~500 mg/d分次；加强用药审查（利尿剂、PPI等）。

**儿科：** 静脉2\~5 mg/kg元素镁，输注速度≤0.05\~0.1 mmol/kg/h，防高镁血症致呼吸抑制；遗传性肾小管疾病需制定长期个体化维持方案。

**产科：** 子痫前期/子痫按产科指南用硫酸镁（负荷量4\~6 g静注，继以1\~2 g/h维持）；非子痫前期以口服为主。

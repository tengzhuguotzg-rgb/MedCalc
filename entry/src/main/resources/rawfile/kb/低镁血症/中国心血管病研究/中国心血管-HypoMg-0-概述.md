# 低镁血症概述

## 2.1 流行病学

低镁血症发生率在各类人群中存在显著差异：

| 人群 | 发生率 |
|------|--------|
| 正常人群 | 约 **2%** |
| PPI使用者 | 约 **6%** |
| 普通住院患者 | 约 **11%**（合并低钙血症时高达23.3%） |
| 医院获得性低镁血症 | 约 **15%** |
| ICU患者 | 约 **52%**，伴较高病死率 |
| 术后入ICU者 | 高达 **61%** |
| 老年 60\~70岁 | **23.4%** |
| 老年 71\~80岁 | **28.7%** |
| 老年 81岁以上 | **53.1%** |

---

## 2.2 病因

**摄入不足：** 营养不良、厌食、饮食缺乏绿叶菜/豆类/坚果/粗粮；长期酗酒（损害小肠吸收 + 促进镁向细胞内转移）。

**肠道吸收障碍：** 炎性肠病、短肠综合征、胰腺炎；长期服用PPI（抑酸影响镁的主动转运）。

**肾脏丢失：** 肾小管酸中毒、肾小管间质性肾病；Gitelman/Bartter综合征；利尿剂（噻嗪类/袢利尿剂）、渗透性利尿剂、氨基糖苷类抗生素、两性霉素B、铂类化疗药。

**内分泌相关：** 糖尿病（渗透性利尿+胰岛素促镁向细胞内转移）、甲状旁腺功能亢进（过多PTH促肾脏排镁）、甲状腺功能亢进（甲状腺素促进肾脏排镁）。

## 2.3~2.9 各系统关联

**心血管系统：** 低镁血症可诱发房性心动过速、房颤、室上速、TdP；心电图可见QRS波增宽、T波低平或倒置、QT间期延长。心脏外科术前补镁可降低术后房颤发生率。基础血镁越低，未来心衰发生率越高。

**内分泌代谢：** 血镁浓度降低与2型糖尿病风险增加显著相关，与HbA1c水平呈负相关。糖尿病可因高血糖损害肾小管重吸收、胃肠吸收减少、二甲双胍致腹泻而诱发低镁。

**肾脏疾病：** 低镁血症是CKD最常见的电解质紊乱；透析液镁浓度从0.5 mmol/L升至1 mmol/L可有效延缓CKD进展、降低IL-6和TNF-α。

**神经骨骼肌：** 症状从头痛、乏力到惊厥、抽搐、肌无力；体征包括Chvostek征、Trousseau征、肌震挛、震颤、眼球震颤。慢性低镁血症（>6个月）可致骨密度降低（骨质疏松症）。

**其他系统：** 消化系统（便秘、麻痹性肠梗阻）、呼吸系统（支气管痉挛、加重哮喘/COPD）、生殖系统（妊娠期高血压/先兆子痫/子痫）。

**电解质紊乱：** 低镁血症常与难治性低钾血症和代谢性碱中毒并存——低镁时低钾也难以纠正。

**特殊人群：** 儿童（Gitelman/Bartter综合征，临床表现可不典型，仅行为异常）；妊娠（硫酸镁是子痫一线解痉药）；老年人（症状常不典型，易漏诊）。

---

## 推荐意见1（强推荐，A）

对出现肌肉震颤、抽搐、惊厥、乏力，或 Chvostek 征、Trousseau 征阳性等神经肌肉兴奋性增高表现的患者，**常规评估血镁水平**。尤其在症状无明确诱因或反复发作的情况下，应高度重视低镁血症作为潜在病因。

---

## 推荐意见2（强推荐，A）

对发生房性心动过速、心房颤动、QT间期延长、尖端扭转型室性心动过速等心律失常，或在术后以及胺碘酮使用过程中出现复杂或新发心律失常的患者，**常规检测血镁水平**。低镁血症应作为心律失常的重要可逆诱因纳入临床鉴别诊断。

---

## 推荐意见3（Ⅱa，推荐，B）

在合并难治性低钾血症、低钙血症、2型糖尿病、慢性便秘、哮喘、COPD、妊娠期高血压、慢性肾病或透析治疗背景的患者中，若病情控制不良或反复波动，应警惕低镁血症，适时开展血镁筛查及动态监测。

## PICO 问题（Table 1）

共 7 个 PICO 问题，涵盖两大主题：**降温方式**（5个问题）和**影响温度控制的药物**（2个问题）。

---

### PICO 1 — 主动降温 vs 被动降温

| 要素 | 内容 |
|------|------|
| **P**（患者） | 热射病患者 |
| **I**（干预） | 主动降温方法 |
| **C**（对照） | 被动降温方法 |
| **O**（结局） | 6个月良好神经功能生存、住院死亡率、降温速率、达到目标温度时间、30分钟内达标、器官功能障碍、不良反应/并发症、住院时长 |

---

### PICO 2 — 冷水浸没 vs 其他主动方法

| 要素 | 内容 |
|------|------|
| **P** | 热射病患者 |
| **I** | 冷水浸没（或其他非冷水浸没的主动方法） |
| **C** | 非冷水浸没的其他方法 |
| **O** | 同 PICO 1 |

---

### PICO 3 — 经典型热射病的冷水浸没

| 要素 | 内容 |
|------|------|

| **I** | 冷水浸没（或其他主动方法） |
| **C** | 非冷水浸没的其他方法 |
| **O** | 同 PICO 1 |

---

### PICO 4 — 劳力型热射病的冷水浸没

| 要素 | 内容 |
|------|------|

| **I** | 冷水浸没（或其他主动方法） |
| **C** | 非冷水浸没的其他方法 |
| **O** | 同 PICO 1 |

---

### PICO 5 — 30分钟内达标

| 要素 | 内容 |
|------|------|
| **P** | 热射病患者 |
| **I** | 30分钟内达到目标温度 |
| **C** | 未在30分钟内达到目标温度 |
| **O** | 6个月良好神经功能生存、住院死亡率、器官功能障碍、不良反应/并发症、住院时长 |

---

### PICO 6 — 快速降温率（≥ 0.155°C/min）

| 要素 | 内容 |
|------|------|
| **P** | 热射病患者 |
| **I** | 达到快速降温率（≥ 0.155°C/min） |
| **C** | 未达到快速降温率（< 0.155°C/min） |
| **O** | 6个月良好神经功能生存、住院死亡率、器官功能障碍、不良反应/并发症、住院时长 |

### PICO 7 — 影响温度控制的药物

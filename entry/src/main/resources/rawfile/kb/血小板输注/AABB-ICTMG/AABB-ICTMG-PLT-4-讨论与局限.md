# 讨论与局限

## Discussion（讨论）

### 核心发现

限制性血小板输注策略在所有人群中均未显示出血或死亡预后改善方面的获益，这与既往研究结果一致。

**关键结论：** RCT未提供一致证据支持血小板对临床预后的重要获益。

### 限制性阈值的变化性

各研究中"限制性"与"宽松"策略的定义因人群而异，即使同一类人群（如HPT）中不同试验的方案也存在差异。这意味着无法对所有人群制定单一的统一阈值推荐。

最严格的限制性策略是"无预防"策略（仅在出现出血时治疗性输注），但该策略尚未在所有人群中测试。

### 与既往指南的差异

| 人群 | 本指南 | 既往指南 |
|------|--------|---------|
| HPT | 限制性（<10×10⁹/L） | 一致 |
| 介入放射学 | 限制性（<20/<50×10⁹/L） | 基本一致 |
| 重大非神经轴手术 | 限制性（<50×10⁹/L） | 基本一致 |
| 心血管手术 | 不预防 | 基本一致 |
| **新生儿** | 新增强推荐 | 既往无 |
| **登革热** | 新增强推荐（不输） | 既往无 |
| **腰椎穿刺** | 限制性（<20×10⁹/L） | 较既往更宽松（既往更保守） |
| **自体SCT** | 无预防策略 | 既往建议更宽松 |
| **CVC置入** | 限制性（<10×10⁹/L） | 既往更宽松 |
| **颅内出血** | 不输（PLT >100，无推荐） | 既往无 |

对于诊断性腰椎穿刺，部分临床医生可能希望采用较宽松策略以减少穿刺创伤，但鉴于脊髓血肿发生率极低，且该预后不太可能影响治疗决策，该考量权重有限。

### 早产儿特别说明

早产儿（1.2）强推荐主要基于单一大型研究结果；虽存在异质性（胎龄、出生后年龄），二次分析未发现基线风险对效果有显著影响。2026年有新试验启动。

### CVC置入最新证据

2023年一项RCT（31）显示：按解剖部位分层的2~4级出血发生率存在效果修正，可压迫部位在限制性和宽松策略间无差异。国际多中心RCT评估多种血小板阈值的研究正在进行（68,69）。

## 局限性（Limitations）

- **患者异质性：** 血小板减少症患者在出血风险相关因素方面存在高度异质性，试验纳入标准和基线特征可能未能覆盖所有重要因素。强调临床判断和良好实践声明的重要性。

- **极低确定性证据：** 部分临床场景的证据确定性极低。

- **基线风险不明确：** 部分研究中事件率存在差异时，基线风险并不总是一致。

- **MID的panel判断：** panel根据价值观和偏好预设了MID，影响了关键结局的证据确定性评级。更保守的MID可能在某些场景降低证据确定性，导致可能出现条件性而非强推荐。

- **间接证据整合：** panel整合了间接证据（假设跨人群相对效果无显著差异），这可能不成立。

- **AGREE II：** panel采用了AGREE II框架确保指南质量，计划在未来实施工作中解决适用性领域问题。

---

## 未来研究方向（Future Directions）

- 针对目前缺乏或确定性极低的临床场景（包括不同年龄组）开展RCT
- 制定个性化血小板输注方法，纳入多种个体因素
- 为出血风险持续存在的患者开发替代方法（如局部止血、氨甲环酸）
- 未来有新重要试验数据时考虑更新指南

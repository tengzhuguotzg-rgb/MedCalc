## 1. INTRODUCTION

### 核心流行病学数据

- 全球aSAH发病率约**6.1/10万人年**，全球患病率约809万（95%不确定性区间：702-972万）
- 地区差异极大：日本28/10万人年、芬兰16.6/10万人年（最高）
- 1955-2014年全球发病率每年下降1.7%，北美每年下降0.7%
- 但美国2007-2017年发病率升至11.4/10万人年（不降反升）
- 院前死亡率22%-26%；院内死亡率无明显改善（美国：2006年13.7%→2018年13.1%）
- 年龄标准化死亡率最高地区：大洋洲、安第斯拉丁美洲、中亚
- 发病率随年龄增长，女性>55岁后尤其明显
- 女性相对风险（RR）1.3（vs 男性）
- 美国Black人群发病率不成比例地更高且持续增长

### 1.1 方法学与证据审查

- 检索时间：2022年3月至2022年6月（2012年后文献）
- 数据库：MEDLINE、PubMed、Cochrane Library等
- 2022年7月至2023年1月间的重要新研究已纳入（影响推荐、COR或LOE者）
- 所有推荐经全体撰写组充分讨论后投票，采用改良Delphi流程达成共识
- 有相关利益冲突的撰写组成员在涉及利益的推荐上回避投票
- 所有推荐获得80.9%-100%的投票成员同意

### 1.2 撰写组构成

撰写组成员包括：神经重症医生、血管神经科医生、血管神经外科医生、神经介入医生（背景多样：放射/神经/神外）、麻醉科医生、康复医生、急性护理执业护士、住院医师和患者代表。

联署机构：AHA/ASA、AANS/CNS、美国神经病学学会（AAN）、Neurocritical Care Society、SNIS、SVIN。

### 1.3 文档审核与批准

文件审核机构：AHA卒中委员会科学声明监督委员会、AHA科学顾问与协调委员会、AHA执行委员会；AANS/CNS、AAN、NCS、SNIS、SVIN及39位个体内容审核者。

### 1.4 指南范围

本指南**仅限于aSAH**，不涵盖：
- 创伤性SAH
- 血管畸形所致SAH
- 易出血性肿瘤所致SAH

本指南**不与以下指南重叠**：脑出血（ICH）、脑动静脉畸形（AVM）、未破裂颅内动脉瘤的治疗指南。

**新增章节**：护理（8.1节）、康复（9节）、复发风险因素（10节）。

**新增重点强调**：共同决策（shared decision-making）、健康公平（health equity）、系统化照护（systems of care）。

### 1.5 COR与LOE体系

本指南采用AHA/ASA推荐的**COR + LOE双栏体系**：

| COR | 含义 |
|-----|------|
| 1 | 强推荐（获益 >> 风险） |
| 2a | 中等推荐（获益 > 风险） |
| 2b | 弱推荐（获益 ≥ 风险） |
| 3 (No Benefit) | 不推荐（无明确益处） |
| 3 (Harm) | 禁忌/有害（风险 > 获益） |

| LOE | 含义 |
|-----|------|
| A | 源自≥1项高质量RCT |
| B-R | 源自≥1项中等质量RCT或其事后分析 |
| B-NR | 源自良好设计的观察性研究 |
| C-LD | 随机或观察性研究的局限性数据 |
| C-EO | 专家共识 |

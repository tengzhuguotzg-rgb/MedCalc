## 4. CLINICAL MANIFESTATIONS AND DIAGNOSIS OF aSAH

### aSAH评估推荐

| COR | LOE | 推荐内容 |
|-----|-----|---------|
| 1 | B-NR | 1. 急性起病剧烈头痛患者，推荐及时诊断性检查以确诊/排除aSAH，最大限度降低发病率/死亡率 |
| 1 | B-NR | 2. 发病>6小时或伴有新发神经功能缺损的急性剧烈头痛患者，行非增强CT，CT阴性时行腰穿(LP)，以确诊/排除aSAH |
| 2a | B-NR | 3. 发病<6小时且无新发神经功能缺损的急性剧烈头痛患者，高质量CT（由委员会认证的神经放射科医生阅片）足以确诊/排除aSAH |
| 2b | B-NR | 4. 无新发神经功能缺损的急性剧烈头痛患者，应用Ottawa SAH Rule识别高危患者是合理的 |
| 1 | B-NR | 5. 自发性SAH且动脉瘤性原因高度怀疑、CTA阴性或不明确时，DSA是确诊/排除脑动脉瘤的必要手段 |
| 2a | B-NR | 6. 已确认脑动脉瘤SAH患者，DSA有助于确定动脉瘤干预的最佳策略 |

### 经典临床表现

- **雷击样头痛（Thunderclap headache）**：突然起病、立即达最大强度——清醒患者aSAH的典型表现
- **先兆/哨兵头痛（Sentinel headache）**：10%-43%的病例在aSAH前出现
- **误诊或延迟诊断**：可能导致死亡和严重残疾

### 诊断流程建议（图2）

```
剧烈头痛或其他aSAH相关症状患者
         ↓
┌─ 发病<6小时 ─┬─ 无神经功能缺损 ─→ 高质量CT → 神经放射科医生阅片
│              └─ 有神经功能缺损 ─→ CT + CTA
│
└─ 发病>6小时 ─→ CT + LP（黄变染色评估）
         ↓
      CT阳性 ─→ CTA ± DSA
      CT阴性 ─→ 临床高度怀疑？→ LP → DSA
```

### Ottawa SAH Rule（表3）

适用于>15岁清醒患者，新发严重非外伤性头痛，1小时内达峰。

**需进一步评估SAH的标准（符合任意1项即需进一步检查）：**

| 编号 | 标准 |
|------|------|
| 1 | 年龄≥40岁 |
| 2 | 颈部疼痛或僵硬 |
| 3 | 目击意识丧失 |
| 4 | 发病于用力时 |
| 5 | 雷击样头痛（立即达峰） |
| 6 | 查体颈部屈曲受限 |

**验证数据：**
- 初始研究（Perry等）：2131例，132例SAH（6.2%），敏感度100%，特异度15.3%
- 前瞻验证（6医学中心）：1153例，67例SAH，敏感度100%，特异度13.6%
- 外部验证（Bellolio等）：454例，9例SAH，敏感度100%，特异度7.6%

### 影像学选择

**非增强CT：**
- 6小时内使用高质量CT，敏感度98.7%，特异度99.9%
- 漏诊率：<1.5/1000
- 需注意：不典型表现（单纯颈痛、晕厥、癫痫、新发局灶性神经功能缺损）不适用此规则

**CTA：**
- 敏感度≈97.2%
- 对<3mm破裂动脉瘤敏感度仅61%
- 不能直接评估SAH，仅评估脑血管病变

**DSA（金标准）：**
- 弥漫性SAH无论CTA结果如何均应进行
- 可评估脑血管解剖和动脉瘤几何形态
- 指导治疗方式选择

### 知识缺口与未来研究方向

- **MRI应用**：多种MRI序列检测和表征aSAH的诊断准确性研究亟需
- **中脑周围SAH**：CTA alone vs 导管DSA的诊断路径尚有争议
- **新兴技术**：双能CT和光子计数CT是SAH和动脉瘤检测的新兴技术

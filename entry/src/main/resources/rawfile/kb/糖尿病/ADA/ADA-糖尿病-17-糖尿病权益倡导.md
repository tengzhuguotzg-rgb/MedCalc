# 17. 糖尿病权益倡导

> **来源**：Diabetes Care 2026;49(Suppl. 1):S356–S357 | [doi:10.2337/dc26-S017](https://doi.org/10.2337/dc26-S017)  
> **无临床推荐**，为 ADA 权益倡导立场声明

ADA 战略目标：让更多儿童、青少年和成人糖尿病患者免受歧视之累。另一个重点：确保**费用不成为成功的糖尿病管理的障碍**。以下为 ADA 发表的部分权益倡导声明（完整列表见 professional.diabetes.org/content/key-statements-and-reports）。

## 权益倡导声明

### 学校环境中的糖尿病护理

> 对应 ADA 正式声明：Cogen F et al., *Diabetes Care* 2024;47:2050–2061

- 确保学校环境中配备有接受过糖尿病管理培训的人员
- 所有学龄儿童应有权在学校监测血糖、使用 CGM、给药和进食
- DMMP（糖尿病医疗管理计划）应包含设备使用说明和备用方案
- 学校应允许学生私自使用 CGM/胰岛素泵而不受限制

### 驾驶与糖尿病

> 对应 ADA 正式声明：Cox DJ et al., *Diabetes Care* 2024;47:1889–1896

- 糖尿病患者不应因糖尿病而被限制驾驶，除非因低血糖/高血糖导致意识或认知障碍
- 对使用胰岛素或磺脲类等高低血糖风险药物者进行评估和咨询
- 低血糖感知障碍者应咨询驾驶风险评估

### 羁押环境中的糖尿病管理

> 对应 ADA 正式声明：Lorber DL et al., *Diabetes Care* 2024;47:544–555

- 拘留机构必须提供符合 ADA 标准的糖尿病护理
- 确保可获得 BGM/CGM、胰岛素/降糖药、膳食计划和糖尿病教育
- 应有书面政策和程序，涵盖安全需求、转运、医疗人员可及性、用品供应和糖尿病患者自我管理赋权
- 不得因糖尿病限制患者置于隔离或单独拘禁

### 幼童糖尿病在托幼和社区环境中的护理

> 对应 ADA 正式声明：March C et al., *Diabetes Care* 2023;46:2102–2111

- 托幼机构应有接受过糖尿病管理培训的工作人员
- 确保幼童（<5 岁）可获 CGM、胰岛素给药和餐食调整
- 需与家长/监护人和糖尿病医疗专业人员建立沟通系统

### 胰岛素获取与可负担性

> 对应 ADA 正式声明：Cefalu WT et al., *Diabetes Care* 2018;41:1299–1311

- ADA 倡导胰岛素价格合理且可及
- 呼吁立法限制自付费用（如 Medicare Part D 和商业保险的 $35/月封顶）
- 反对非专利和生物类似药准入障碍

### 糖尿病与就业

> 对应 ADA 正式声明：Anderson JE et al., *Diabetes Care* 2014;37(Suppl. 1):S112–S117

- 糖尿病患者不应因糖尿病被拒绝就业或限制就业（无论是否使用胰岛素）
- 就业环境应提供合理便利（如时间安排血糖监测、用药）
- 消防、执法、商业驾驶等特定职业应基于**个体化评估**而非自动排除

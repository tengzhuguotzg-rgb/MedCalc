## 原发性 HLH 的基因缺陷

原发性（遗传性）HLH 的致病基因变异频率因种族而异，**PRF1、UNC13D、STXBP2** 变异最常见。

大多数变异与重型、起病早相关；部分变异可能引起表型较轻、起病较晚。

**其他可导致 HLH  predisposition 的遗传缺陷**：
- **Chédiak–Higashi 综合征**（LYST）：溶酶体生物合成障碍，细胞毒性缺陷
- **X连锁淋巴组织增生综合征 1 型**（XLP1，SH2D1A）：SLAM相关蛋白缺陷
- **代谢性出生缺陷**：如 Wolman 病（LIPA）、赖氨酸尿蛋白不耐受（SLC7A7）
- **NLRC4**：炎症小体失调，导致 NLRP3 活化异常
- **XLP2**（BIRC4）：T细胞凋亡调控缺陷

## 家族性 HLH 的细胞毒性缺陷机制

### 核心概念：免疫应答启动正常，但**无法终止**

**正常免疫应答**（Fig. 1A）：
1. 巨噬细胞被病原体激活，分泌促炎细胞因子并呈递抗原
2. CD8+ T 细胞被招募→增殖→清除感染细胞→分泌 IFN-γ→刺激固有免疫
3. NK 细胞通过诱导凋亡减少活化 CD8+ T 细胞，下调免疫应答
4. 病原体清除后，免疫细胞程序性死亡，仅保留记忆 T 细胞

**家族性 HLH**（Fig. 1B）：
1. 免疫应答**正常启动**
2. 但细胞毒性缺陷导致：感染细胞无法被清除，病原体持续存在加剧 IFN-γ 产生
3. 巨噬细胞被过度激活，细胞毒性 T 细胞无法清除过多的抗原呈递细胞
4. NK 细胞无法清除有缺陷的活化 CD8+ T 细胞
5. 结果：**过度免疫应答→焦亡（pyroptosis）**→DAMP/PAMP 释放→再次激活巨噬细胞
6. **恶性循环**→细胞因子风暴→多器官损伤和多器官衰竭

### 关键细胞因子

| 细胞因子 | 作用 |
|---------|------|
| **IFN-γ** | 核心驱动因子；在原发性和继发性 HLH 中均有关键作用；CXCL-9 是其替代标志物 |
| IL-1β、IL-6、IL-10、IL-18、TNF | 显著升高，参与高细胞因子血症 |

### 穿孔素-颗粒酶途径缺陷

正常情况下，NK 细胞和细胞毒性 T 细胞通过**穿孔素-颗粒酶细胞死亡途径**诱导靶细胞程序性凋亡（Fig. 2）。

家族性 HLH 中，该过程因以下原因缺陷：
- 穿孔素产生不足
- 含穿孔素颗粒从细胞毒性细胞中释放减少

**图 2（基因-蛋白-功能）**：

| 基因 | 蛋白 | 功能 |
|------|------|------|
| PRF1 | 穿孔素（Perforin） | 在免疫突触形成多孔通道，允许颗粒酶进入靶细胞 |
| UNC13D | Munc13-4 | 促进细胞毒性囊泡在免疫突触对接 |
| STX11 | Syntaxin-11 | 促进囊泡与质膜融合，释放穿孔素和颗粒酶 |
| STXBP2 | Munc18-2 | 协助 Syntaxin-11 转运至突触 |
| RAB27A | Rab27a | 介导囊泡沿微管向免疫突触转运 |
| LYST | Lyst | 促进分泌性溶酶体的生物合成 |

### 重要临床认识

- 家族性 HLH 证明了**颗粒介导的细胞毒性在人类免疫调控中不可或缺**
- **双等位基因 PRF1 突变携带者**（存活至10岁以上未发生家族性 HLH）在儿童或青少年期发生至少一种血液系统恶性肿瘤的风险接近 **50%**
- **UNC13D 突变单等位基因携带者**：淋巴瘤风险是非携带者的 **3 倍**

## 继发性 HLH 的发病机制

### 阈值模型（Brisse et al.）

多种因素共同作用，达到阈值时炎症失控、暴发性 HLH 发生（Fig. 3）：

| 因素类型 | 具体内容 |
|---------|---------|
| 遗传因素 | 影响细胞毒功能的基因变异（如双基因突变） |
| 背景炎症 | 基础炎症状态 |
| 潜在免疫抑制 | 免疫抑制状态 |
| 感染触发 | 病原体感染 |

### 继发性 HLH 与家族性 HLH 的差异

| 特征 | 继发性 HLH | 家族性 HLH |
|------|-----------|-----------|
| NK 细胞和细胞毒性 T 细胞 | **数量常减少** | 功能缺陷但数量正常 |
| 细胞毒性缺陷性质 | 短暂、可逆（多为功能性） | 遗传性、持续性 |
| 其他 | 可有耗竭标志物高表达（如 PD-1） | - |

感染性病原体（如 EBV、H5N1 流感病毒）可直接抑制细胞毒性功能。

### MAS-HLH 的特殊机制

- MAS-HLH 的病理生理机制**部分不同于**其他 HLH 形式
- **IL-18 过量产生**是其标志性发病机制，也见于其前驱疾病（Still 病）

---

## 核心结论

> **HLH 的本质是免疫调控失败**：免疫应答启动正常，但终止机制缺陷→IFN-γ 主导的细胞因子风暴→多器官损伤。家族性 HLH 源于遗传性细胞毒性缺陷，继发性 HLH 为多因素阈值模型，两者最终殊途同归。

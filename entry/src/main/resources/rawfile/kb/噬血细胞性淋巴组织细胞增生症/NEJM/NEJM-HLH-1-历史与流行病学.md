## 历史

| 年份 | 事件 |
|------|------|
| 1939 | Scott 和 Robb-Smith 首次描述一种致死性疾病（发热、消瘦、血细胞减少、肝脾肿大、活跃吞噬现象），命名为"组织细胞性骨髓网状细胞增生症"（histiocytic medullary reticulosis），可能为恶性肿瘤相关的 HLH |
| 1951/1952 | 家族性 HLH 首次被描述（1951年 Reese 和 Levy 报告两例；1952年 Farquhar 和 Claireaux 正式报告） |
| 1979 | 病毒相关噬血细胞综合征（即继发性 HLH）被正式报告 |
| 1999 | 首个家族性 HLH 致病基因 **PRF1** 被发现，是该领域的重要突破 |

1999年后，明确家族性 HLH 由以下四种基因的双等位基因变异引起：

| 基因 | 对应蛋白 | 家族性HLH类型 |
|------|---------|-------------|
| **PRF1** | 穿孔素（Perforin） | 类型2 |
| **UNC13D** | Munc13-4 | 类型3 |
| **STX11** | Syntaxin-11 | 类型4 |
| **STXBP2** | Syntaxin-binding protein 2 (Munc18-2) | 类型5 |

**Griscelli 综合征 2 型（GS2）** 由 **RAB27A** 基因变异引起，该基因编码的小 GTP 酶 Rab27a 参与溶酶体转运。

上述蛋白均对 NK 细胞和细胞毒性 T 细胞的正常功能至关重要，证实了家族性 HLH 的核心发病机制：**淋巴细胞细胞毒性缺陷**。

## 流行病学

### 原发性 HLH

- **发病率**：瑞典数据显示儿童发病率约 **1/50,000 活产婴儿**
- 与严重联合免疫缺陷病（SCID，发病率 1/38,500 活产婴儿）相当；两者可能是人类最常见的遗传性、迅速致死性免疫缺陷病
- 因属常染色体隐性遗传，在**近亲结婚常见地区**发病最多
- 发病中位年龄：**3~6 月龄**

### 继发性 HLH

- 总体发病率：2018 年英格兰数据显示所有类型 HLH 发病率约 **4.2/百万人口**
- 恶性肿瘤相关 HLH：2012-2018 年瑞典数据显示年发病率至少 **6.2/百万成人**，地区差异可能与对该病认识程度有关
- **全球范围内仍普遍漏诊**

### 触发因素（继发性 HLH，661例危重症成人回顾）

| 触发因素 | 占比 |
|---------|------|
| 感染 | 50% |
| 恶性肿瘤 | 28% |
| 自身免疫病 | 12% |

**最常见感染病原体**：EBV（25%）、细菌（20%）、CMV（7%）

**恶性肿瘤触发**：淋巴瘤占恶性触发的 76%，白血病 8%

**自身免疫触发**：系统性红斑狼疮（SLE，39%）和成人 Still 病（21%）最常见

## 预后

### 家族性 HLH（历史数据）

- **不治疗**：中位生存期仅 1~2 个月（Janka 1983）
- **HLH-94 方案**：5年生存率约 **50%**
- **HLH-2004 方案**：5年生存率约 **59%**
- **现代数据（国际注册）**：移植前生存率高达91%，HSCT后3年生存概率 **77%**

### 继发性 HLH 预后（分型）

| 类型 | 预后 |
|------|------|
| **肿瘤相关 HLH** | **最差**，2年生存率仅 **20~30%**（部分因基础癌症预后差） |
| **MAS-HLH** | 儿童病死率 **5~10%**，成人 **10~15%** |
| 感染相关（有效抗菌治疗时） | 较好（针对病原体治疗后） |

### 影响预后的关键因素

- **早期识别和启动治疗**：延迟治疗可导致不可逆器官损伤（CNS、肝脏等）
- **基础病因**：肿瘤相关最差，感染相关若能针对性治疗则较好
- **CNS 受累**：可导致不可逆神经损伤，影响长期预后
- **移植时机**：HSCT 前存活是关键；无症状患者行 HSCT 100%存活（凸显新生儿筛查价值）

## 未来方向

### 当前挑战

1. **漏诊问题**：HLH（尤其是继发性）仍普遍漏诊，提高认识可挽救许多生命
2. **发病机制未明**：继发性 HLH 的病理生理机制尚未完全阐明
3. **最佳实践缺乏**：各类继发性 HLH 的诊断和治疗最佳实践有待深入研究

### 基因治疗

- 在小鼠模型中，**PRF1 和 UNC13D 缺陷**的基因治疗已显示有效
- 希望在 **1~2 期临床试验**中明确基因治疗的角色

### 新兴研究方向

| 方向 | 说明 |
|------|------|
| **新病因识别** | 发现更多可引起继发性 HLH 的疾病状态，有助于早期识别和干预 |
| **CAR-T/免疫效应细胞治疗相关 HLH** | 新型治疗带来的并发症，需深入研究 |
| **精准医学** | 基于基因型和细胞毒性检测的分层治疗 |

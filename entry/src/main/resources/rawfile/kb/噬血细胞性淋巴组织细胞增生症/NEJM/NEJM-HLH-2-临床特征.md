## 家族性 HLH

### 典型临床表现

典型起病：发热性疾病婴幼儿出现**脓毒症样表现**伴**血细胞减少**和**肝脾肿大**。

**血液系统表现**：
- 血小板减少、贫血常见，中性粒细胞减少相对少见
- ⚠️ 噬血现象（Hemophagocytosis）**并非始终存在**，尤其病程早期；且噬血现象并非 HLH 特异性表现

**肝脏表现**：
- 肝脾肿大几乎普遍存在
- 谷氨酰转肽酶（GGT）和乳酸脱氢酶（LDH）升高常见
- 高胆红素血症（以结合胆红素为主）
- 一项英国研究纳入78例<24月龄急性肝功能衰竭儿童，其中30例为HLH，19例有基因变异

**凝血功能**：
- 弥散性血管内凝血（DIC）、严重急性出血、低纤维蛋白原血症常见

→ 相关：[[凝血病/解放军医学杂志/军医-凝血病-0-概述]]（DIC 是 HLH 常见并发症，可参考凝血病共识进行标准化评估）

**神经系统表现**：
- 约**1/3**患者诊断时存在神经系统改变，可为重度（癫痫发作、意识减退、脑膜刺激征）
- 共济失调和精神运动发育迟缓可逐渐出现
- 青少年和成人中，神经系统症状（尤其是共济失调/步态障碍和癫痫）可能为首发表现
- 约**一半**患儿脑脊液淋巴细胞数或蛋白含量中度升高
- MRI：常见弥漫性多灶性白质病变和小脑受累
- 罕见情况：可逆性后部脑病综合征（PRES）与 HLH 相关

**其他表现**：
- 瘀点、瘀斑、短暂性斑丘疹、淋巴结肿大
- 白蛋白降低、钠降低、甘油三酯升高
- **可溶性 CD25（sCD25）/白介素-2受体α**水平常显著升高，与铁蛋白联用作为疾病活动度监测生物标志物

**Griscelli 综合征 2 型（GS2）**：发色灰银、眉毛和睫毛呈银灰色，头发光学显微镜下见大块色素颗粒；STXBP2 变异可导致严重腹泻，可能在 HSCT 成功后仍持续存在

## 继发性 HLH

### 临床表现

所有 HLH 形式的** signs and symptoms 基本相似**，原发与继发难以区分。主要区别在于：原发性多见于婴幼儿，继发性多见于较大儿童和成人。

继发性 HLH 常表现为**脓毒症样危重症**，对脓毒症治疗无效；因与危重症患者其他炎症性疾病表现重叠，识别困难。

### 触发因素（661例危重症成人回顾）

| 触发因素 | 占比 |
|---------|------|
| 感染 | **50%** |
| 恶性肿瘤 | **28%** |
| 自身免疫病 | **12%** |

**感染触发**：EBV（25%）最常见，其次为细菌（20%）、CMV（7%）

**恶性触发**：淋巴瘤占 76%（T/NK 细胞淋巴瘤常见），白血病 8%

**自身免疫触发**：SLE（39%）和成人 Still 病（21%）最常见；其他包括系统性血管炎、炎症性肠病

**儿童特点**：感染相关 HLH 最常见，其次为自身免疫相关 HLH（见于约10%的全身型幼年特发性关节炎患儿和 SLE 患儿），恶性肿瘤相关 HLH 在儿童中少见得多

**神经系统**：发生率低于家族性 HLH，约 **10%~25%** 患者有神经系统症状；较大规模研究中约一半 MRI 异常

## HLH-2024 诊断标准（修订版）

### HLH-2004 诊断标准（需满足5/8条）

| 指标 | 阈值 |
|------|------|
| 发热 | ≥38.5°C |
| 脾肿大 | ≥2cm |
| 血细胞减少 | ≥2系（血红蛋白<90g/L+血小板<100×10⁹/L+中性粒细胞<1×10⁹/L） |
| 低纤维蛋白原血症或高甘油三酯血症 | 纤维蛋白原≤1.5g/L 或 甘油三酯≥3.0mmol/L |
| 高铁蛋白血症 | ≥500μg/L |
| 噬血现象 | 骨髓/其他组织 |
| NK 细胞活性 | 降低或缺失 |
| 可溶性 CD25 | ≥2400U/mL |

### HLH-2024 修订要点

2024年组织细胞学会修订诊断标准（HLH-2024），提出**三种诊断策略**：

1. **基因检测策略**：确认致病基因变异
2. **淋巴细胞细胞毒性检测策略**：流式细胞术筛查
3. **临床标准策略**：5/7条（原8条去掉 NK 细胞活性）

**关键变化**：NK 细胞活性检测从临床标准中移除，作为独立诊断路径

### 辅助诊断工具

- **H Score**：成人加权评分系统（9变量），包含发热、器官肿大、免疫抑制状态、噬血现象、血细胞减少数、铁蛋白/甘油三酯/纤维蛋白原/天冬氨酸氨基转移酶水平；总分 0~337，≥168 分时对 HLH 诊断的特异度为94.1%，灵敏度为100%
- **sCD25/铁蛋白比值≥2.0**：预测淋巴瘤相关 HLH 阳性预测值85%；若联合 sCD25≥5000U/mL，阳性预测值升至96%
- **铁蛋白筛查**：危重患者若经验性脓毒症治疗效果不佳，应考虑 HLH 并检测铁蛋白；儿童高铁蛋白特异性优于成人

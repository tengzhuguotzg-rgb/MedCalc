## 23条推荐总览

| 编号 | 推荐强度 | 证据级别 | 主题 |
|------|---------|---------|------|
| 推荐意见1 | 弱推荐 | 低级别质量证据 | 重症患者发生应激性溃疡（或出血）的高危因素 |
| 推荐意见2 | 弱推荐 | 低级别质量证据 | RRT和机械通气患者密切监测应激性溃疡 |
| 推荐意见3 | 弱推荐 | 低级别质量证据 | 糖皮质激素、抗凝剂、抗血小板药、NSAIDs使用者密切监测 |
| 推荐意见4 | 弱推荐 | 中等级别质量证据 | 高危患者使用抑酸剂预防应激性溃疡 |
| 推荐意见5 | 弱推荐 | 中等级别质量证据 | 推荐PPI预防应激性溃疡（优于H2RA） |
| 推荐意见6 | 弱推荐 | 低级别质量证据 | 无肠内营养禁忌证者使用肠内营养降低出血风险 |
| 推荐意见7 | GPS | — | 不推荐常规早期内镜检查，仅用于药物难以控制的大出血 |
| 推荐意见8 | 弱推荐 | 低级别质量证据 | 酌情采取胃肠减压 |
| 推荐意见9 | 弱推荐 | 低级别质量证据 | 采取禁食，低危患者可尽早进食 |
| 推荐意见10 | GPS | — | 尚无足够证据推荐使用胃黏膜保护剂 |
| 推荐意见11 | **强推荐** | **高级别质量证据** | 推荐使用PPI治疗应激性溃疡及上消化道出血 |
| 推荐意见12 | 弱推荐 | 低级别质量证据 | 不推荐常规使用H2RA（PPI不可及时可作为二线） |
| 推荐意见13 | 弱推荐 | 低级别质量证据 | 不推荐常规使用生长抑素及其类似物 |
| 推荐意见14 | 弱推荐 | 中等级别质量证据 | 建议使用氨甲环酸治疗应激性溃疡合并上消化道出血 |
| 推荐意见15 | 弱推荐 | 低级别质量证据 | 常规治疗无效可尝试凝血酶、止血粉、去甲肾上腺素 |
| 推荐意见16 | GPS | — | 活动性出血建议停用NSAIDs、糖皮质激素、抗凝剂或抗血小板药 |
| 推荐意见17 | GPS | — | 常规治疗失败建议尽快内镜检查并止血 |
| 推荐意见18 | 弱推荐 | 低级别质量证据 | 内镜干预失败建议血管内介入或手术治疗 |
| 推荐意见19 | GPS | — | 休克患者复苏目标MAP不低于65 mmHg |
| 推荐意见20 | GPS | — | 无心脑血管合并症者以Hb 7 g/L为输血指征 |
| 推荐意见21 | GPS | — | 心脑血管/肺部疾病患者个体化制定输血阈值 |
| 推荐意见22 | GPS | — | 出血停止后如无严重血流动力学不稳定可早期肠内营养 |
| 推荐意见23 | 弱推荐 | 中等级别质量证据 | 使用PPI时注意增加肾脏不良事件风险，密切监测肾功能 |

# ERC 2025 成人高级生命支持 生理监测与CPR

## 高监护环境下的特殊考虑

### 高度监护环境定义

ICU、导管室、手术室等有持续心电和血流动力学监测的环境。

### 生理信号预警

- **ETCO2突然下降**：可能提示心脏骤停或心输出量极低状态
- **收缩压降至<50 mmHg**：即使干预后仍低，考虑开始胸外按压

### 有创动脉血压监测患者的肾上腺素给药

- 首剂肾上腺素小量递增（如50-100 μg IV）而非1 mg推注
- 如总剂量1 mg仍无反应，确保无外渗，考虑继续每3-5分钟1 mg

## 生理引导CPR的目标值

### 推荐目标

| 参数 | 目标值 | 说明 |
|------|--------|------|
| 舒张压（Diastolic BP） | ≥30 mmHg | 有创动脉血压监测时 |
| ETCO2 | ≥25 mmHg (3.3 kPa) | 无创监测 |

### 原理

- 舒张压≥30 mmHg反映胸外按压产生的冠状动脉灌注压
- ETCO2≥25 mmHg反映足够的心输出量和肺灌注

### 实际应用

- 在有创监测可用时，以舒张压≥30 mmHg为导向调整按压深度和速率
- 在无创监测时，以ETCO2≥25 mmHg为导向

## 波形碳谱仪（Waveform Capnography）监测

### 推荐

- 使用波形ETCO2确认气管导管位置
- 使用波形ETCO2监测CPR质量
- ETCO2在CPR期间升高可能提示ROSC已发生
- **不要仅凭低ETCO2值决定停止复苏尝试**
- ROSC判断：结合临床和生理征象（意识恢复、有目的动作、动脉波形、ETCO2升高），而非单一指标

## 床旁超声（Point-of-Care Ultrasound, POCUS）

### 推荐

- 仅熟练操作者进行心脏骤停期间POCUS（良好实践声明）
- POCUS**不得导致胸外按压额外中断或延长**（强推荐）
- POCUS可帮助识别心脏骤停的可逆原因（如心脏填塞、张力性气胸）

### 诊断要点

- 右心室扩张单独存在不能用于诊断肺栓塞
- **不要使用POCUS评估心肌收缩力作为终止CPR的唯一指标**

### 扫查方案

建议在按压间隙（每2分钟心律检查时）快速扫查：
- 心尖四腔心（心包填塞）
- 下腔静脉（容量状态）
- 肺滑移（气胸）

## 机械胸腔按压设备

### 推荐

- 仅在高质量手动胸外按压不实际或危及施救者安全时考虑机械胸腔按压设备（弱推荐，低确定性证据）
- 使用机械按压设备时，通过仅使用经过培训的团队来最小化设备应用期间的按压中断

### 设备类型

- 活塞驱动装置（如LUCAS）
- 负载分布式带装置（如AutoPulse）

### 使用指征

- 长时间CPR（如ECPR期间）
- 转运期间
- 施救者疲劳导致按压质量下降

## 复苏性血管内球囊阻断（REBOA）

### 推荐

- **ERC不推荐心脏骤停常规使用REBOA**，除非在临床试验中评估（弱推荐，极低确定性证据）

### 背景

- REBOA用于控制出血（非心脏骤停主要适应证）
- 心脏骤停中证据有限

## 复苏中降温（Intra-Arrest Cooling）

### 推荐

- **不推荐ALS期间进行复苏中降温**，除非严重过热（强推荐，低确定性证据）

### 说明

- 目标温度管理（TTM）在ROSC后进行，而非CPR期间

## 体外心肺复苏（ECPR）

### 推荐

- 对于选定的IHCA和OHCA患者，当传统CPR无法恢复自主循环时，ECPR可作为挽救治疗加以考虑（弱推荐，极低确定性证据）
- 仅在可实施的设置中使用

### ECPR适应证（通常）

| 因素 | 标准 |
|------|------|
| 年龄 | <75岁（部分中心<70岁） |
| 目击 | 目击心脏骤停 |
| 初始心律 | 电击心律 |
| CPR持续时间 | <60分钟（部分<30分钟） |
| 合并症 | 有限严重合并症 |
| 实施地点 | 有ECMO资源和专业知识的中心 |

### ECPR的实质

- 为心脏骤停患者提供循环氧合支持
- 作为通往明确治疗的桥梁

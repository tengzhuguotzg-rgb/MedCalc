# ERC 2025 成人高级生命支持 概述

## 指南基本信息

- **全称**：European Resuscitation Council Guidelines 2025 Adult Advanced Life Support
- **发表期刊**：Resuscitation
- **制定机构**：ERC（European Resuscitation Council）
- **依据**：ILCOR CoSTR（国际复苏联络委员会）2025年共识
- **方法学**：证据系统评价 + 专家共识

## 关键信息

### 指南亮点（Key Messages）

- 强调高质量胸外按压是ALS的基础
- 重视心脏骤停的预防性征象，在发生前加以干预
- 声门上气道和气管插管应循序渐进，只有成功率高者才能进行气管插管
- 尽早启动ALS以挽救更多生命
- 高效氧合与通气配合高质量胸外按压
- 正确的顶侧位 pad 位置对除颤至关重要
- 使用波形二氧化碳图确认气管导管位置

### 主要变化（2021→2025）

| 项目 | 2021版 | 2025版 |
|------|--------|--------|
| 高级生命支持启动时机 | 未强调 | 越早越好 |
| 氧气与通气 | 未强调 | 高质量胸外按压配合有效通气 |
| pad位置 | 未明确 apical (lateral) | 明确 apical (侧) pad 位置 |
| 气管导管位置确认 | 未明确 | 必须使用波形CO2图 |
| 不可电击心律的肾上腺素时机 | 未明确 | 尽早给药 |
| 除颤能量 | 150-360 J biphasic | 首次 ≥150 J（pulsed biphasic 130-150 J） |
| 难治性VF pad位置 | 参考文本 | 三次电击后考虑前后位换向量 |
| 双序贯除颤（DSD） | 无 | 不推荐常规使用 |
| 心脏骤停期间补钙/碳酸氢钠/糖皮质激素 | 未明确说明 | 常规不推荐 |
| 前胸锤击 | 已移除 | 2025版不再包含 |
| 低资源环境ALS | 未提及 | 新增章节 |
| CPR诱导的意识 | 未提及 | 新增章节 |

## 推荐强度说明

本指南采用与ILCOR一致的推荐强度体系：

- **强推荐 (Strong)**：获益确定性高，适用于大多数患者
- **弱推荐 (Weak/Conditional)**：平衡因患者价值观、资源可及性而异
- **良好实践声明 (Good Practice Statement)**：获益明确，基于专家共识，无需正式证据评级
- **证据不足 (Insufficient Evidence)**：无法给出正式推荐

注：ERC 2025 ALS指南大部分内容基于专家共识，而非严格GRADE分级的随机对照试验。

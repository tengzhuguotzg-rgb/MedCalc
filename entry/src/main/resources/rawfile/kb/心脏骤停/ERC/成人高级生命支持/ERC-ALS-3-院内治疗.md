# ERC 2025 成人高级生命支持 院内治疗

## ALS算法（Fig. 2来源）

### 室颤/无脉性室速（VF/pVT）治疗流程

```
VF/pVT识别 → 立即除颤（150-200 J biphasic）
       ↓
 CPR 2分钟 + 气道管理（见8-气道与通气）
       ↓
 再次识别心律 → 仍为VF/pVT
       ↓
 第二次除颤（150-360 J）
       ↓
 CPR 2分钟 + IV/IO肾上腺素（1 mg）
       ↓
 第三次除颤（如需要）
       ↓
 CPR期间考虑：H's/T's 可逆病因 + 高级气道
       ↓
 考虑：难治性VF → 转换pad位置（前-后位）/ 胺碘酮300 mg
       ↓
 持续循环：2分钟CPR → 识别心律 → 除颤/药物
```

### 非电击心律（Asystole/PEA）治疗流程

```
Asystole/PEA识别 → 立即CPR 2分钟
       ↓
 建立IV/IO通路
       ↓
 肾上腺素1 mg（每3-5分钟重复）
       ↓
 识别并治疗可逆病因（H's/T's）
       ↓
 高级气道管理
       ↓
 持续CPR直至ROSC或终止复苏
```

## 医院内复苏流程（In-Hospital Resuscitation Algorithm）

### 关键要素

1. **早期识别**：使用早期预警评分识别高危患者
2. **快速反应**：立即启动RRS/代码蓝色
3. **高质量CPR**：
   - 按压深度≥5 cm
   - 按压速率100-120次/分钟
   - 按压中断最小化
   - 每次按压后让胸廓完全回弹
4. **早期除颤**：VF/pVT发作后尽快除颤（目标<3分钟）

### 监护环境下的特殊处理

在高度监护环境（如ICU、导管室、手术室）中：
- 可识别VF/pVT发作
- 可尝试连续三次快速除颤（如患者仍有脉搏但已知心律异常）
- 随后进行2分钟CPR再评估

## 团队与领导

### 推荐

- 每个医院应建立心脏骤停响应团队（强推荐，低确定性证据）
- ALS团队领导应协调复苏流程、记录关键时间点、沟通家属（良好实践声明）
- 团队应定期进行模拟培训（强推荐）

### 复苏团队的理想组成

| 角色 | 职责 |
|------|------|
| 团队领导 | 协调复苏、决策 |
| 按压者×2 | 交替进行高质量胸外按压 |
| 通气者 | 管理气道、正压通气 |
| 除颤者 | 操作除颤仪、给药 |
| 记录者 | 记录时间、用药、干预 |
| 家属联络 | 沟通、陪伴 |

## 设备与基础设施

### 推荐

- 医院应确保临床区域随时可取复苏设备和药物（强推荐，中等确定性证据）
- 设备应在全院统一标准化配置，定期检查（强推荐）
- 心脏骤停事件发生后，应立即补充所有消耗性物品（良好实践声明）

### 标准化复苏车内容

| 类别 | 物品 |
|------|------|
| 气道管理 | 面罩、口咽/鼻咽通气道、声门上气道、气管插管（含可视喉镜） |
| 循环 | 手套、注射器、静脉导管、骨穿针、肾上腺素、胺碘酮 |
| 除颤 | 除颤仪（带贴片和导联线） |
| 其他 | 剪刀、纱布、记录表 |

## 记录与数据管理

### 推荐

- 每例心脏骤停事件都应有详细记录（强推荐）
- 记录应包括：
  - 事件时间线
  - 干预措施及其时间
  - 药物剂量和时间
  - 患者结局

### 数据质量改进

- 医院应定期审查心脏骤停数据（弱推荐）
- 使用核查表进行事件后复盘（弱推荐）
- 反馈给团队以改进未来表现

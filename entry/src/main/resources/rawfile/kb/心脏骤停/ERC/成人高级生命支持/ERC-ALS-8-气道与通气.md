# ERC 2025 成人高级生命支持 气道与通气

## 通气原则

- CPR期间给予最高可行的吸入氧浓度
- 尽早开始有效通气，确保适当的频率和潮气量
- 避免通气不足（低通气）和过度通气（高通气）

## 面罩通气（Bag-Mask Ventilation）

### 推荐

- 通过优化面罩密封和气道开放来实现有效的面罩通气
- 必要时使用**两人技术**进行面罩通气（比单人更有效）
- 每次吸气给气1秒，可见胸廓抬起
- 一人负责面罩密封和开放气道，另一人挤压球囊

### 关键点

- 选择适当大小的面罩
- 抬下颌开放气道
- 确保面罩与面部密闭

## 声门上气道（Supraglottic Airway, SGA）

### 推荐

- 使用声门上气道时，**i-gel优于喉气管**
- 按步骤进行：从基础气道技术开始，根据施救者技能逐步进阶
- 仅高成功率（>95% within two attempts）的施救者才能进行气管插管

### i-gel特点

- 即刻放置，无需充气
- 迷路密封设计
- 快速放置，减少中断

### SGA放置后通气

- 频率：10次/分钟
- 持续胸外按压不暂停
- 如气体泄漏导致通气不足，暂停按压进行通气（30:2）

## 气管插管

### 推荐

- 仅由**高成功率施救者**使用连续波形ETCO2监测进行气管插管
- 高成功率的专家共识：2次尝试内成功率>95%
- 目标：气管插管相关胸外按压中断<5秒
- 使用直接喉镜或视频喉镜，根据当地协议和施救者经验

### 视频喉镜（Video Laryngoscopy）

- 在视频喉镜立即可用的环境中，**优先使用视频喉镜**而非直接喉镜
- 可改善声门可视化，减少插管时间

### ETCO2波形图确认

- **必须使用波形ETCO2来排除气管导管的食管位置**
- 持续ETCO2波形确认气管导管在位
- 如ETCO2突然下降，警惕导管移位或故障

### 气管插管流程

```
球囊面罩通气准备
       ↓
 喉镜（直接或视频）
       ↓
 气管插管（目标<5秒内完成）
       ↓
 立即连接ETCO2监测
       ↓
 确认波形持续（排除食管置管）
       ↓
 固定导管，连接呼吸机
```

## 机械通气设置（During CPR）

### 推荐设置

| 参数 | 设置值 |
|------|--------|
| 通气模式 | 容量控制或压力调节模式 |
| 潮气量 | 6-8 mL/kg（预测体重） |
| FiO2 | 最大值 |
| 呼吸频率 | 10次/分钟 |
| 吸气时间 | 1-1.5秒 |
| PEEP | 0-5 cm H2O |
| 峰值压力报警 | 60-70 cm H2O |
| 吸气触发 | 关闭 |

### 监测

- 确保机械通气有效
- 如无效，改用手动通气

## 困难气道处理

### 条件

标准气道管理策略（口咽通气道+球囊面罩/声门上气道/气管导管）失败时。

### 外科气道

- 仅由经过适当培训的施救者实施
- 选择：外科环甲膜切开术或针刺环甲膜穿刺

### 适应证

- 无法通气、无法插管
- 严重上气道梗阻（异物、肿瘤、创伤）

## 含气囊气道管理

- 气囊压力适当（避免过高导致气道缺血）
- 定期监测气囊压力

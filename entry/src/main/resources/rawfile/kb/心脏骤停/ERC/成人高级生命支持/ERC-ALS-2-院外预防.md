# ERC 2025 成人高级生命支持 院外预防

## 院外心脏骤停流行病学

### 欧洲数据

- 欧洲每年约有5000例心脏骤停
- 心脏骤停发生率：每10万人50-100例
- 存活出院率：约10%（不同地区差异显著）

### 危险因素

| 类别 | 因素 |
|------|------|
| 人口学 | 年龄增长、男性 |
| 既往病史 | 冠心病、心衰、心律失常史 |
| 社会经济 | 低收入地区发生率更高 |
| 遗传 | 心律失常家族史 |

## 冠心病与心脏性猝死（SCD）

### 背景

- 冠心病是SCD的主要原因（约80%）
- SCD可能是无症状冠心病的首发表现

### 推荐

- 有冠心病风险人群应进行SCD风险评估（弱推荐）
- 冠心病患者应评估是否需要埋藏式心律转复除颤器（ICD）进行一级预防（强推荐，中等确定性证据）

## 猝死病因（Table 2来源）

### 冠心病相关

- 急性心肌缺血/梗死
- 陈旧性心肌梗死疤痕
- 心肌病变

### 非冠心病

| 病因 | 示例 |
|------|------|
| 遗传性心律失常 | 长QT综合征、Brugada综合征、短QT综合征 |
| 心肌病 | 肥厚型心肌病、致心律失常性右室心肌病 |
| 先天性心脏病 |  |
| 瓣膜病 | 主动脉瓣狭窄 |
| 其他 | 炎症性心肌病、淀粉样变性 |

### 可逆性病因

- 电解质紊乱（低钾、高钾、低镁）
- 药物相关心律失常
- 吸毒
- 溺水

## 年轻人（<35岁）心脏骤停

### 病因构成

| 病因 | 比例 |
|------|------|
| 遗传性心律失常疾病 | 约30-40% |
| 心肌病 | 约20-30% |
| 冠心病 | 约10-20% |
| 其他/不明原因 | 约20-30% |

### 遗传性心律失常疾病

#### 长QT综合征（LQTS）

- 常染色体显性遗传
- 心电图QT间期延长
- 运动或情绪触发尖端扭转型室速
- β受体阻滞剂为一线治疗

#### Brugada综合征

- 特征性心电图改变（V1-V3 ST段穹窿样抬高）
- 发热是已知触发因素
- ICD是主要治疗手段

#### 儿茶酚胺敏感性多形性室速（CPVT）

- 运动或情绪触发双向性室速
- 氟卡尼+β受体阻滞剂治疗

### 推荐

- 年轻人OHCA存活者应进行遗传评估和家族筛查（强推荐）
- 确诊遗传性心律失常者应进行家族级联筛查（强推荐）
- 有心脏骤停家族史者应避免竞技运动直至完成心脏评估（良好实践声明）

## 预防策略

### 社区层面

- 公共场所AED配置（机场、火车站、体育场馆等）
- 公众CPR培训（社区培训项目）
- 心脏骤停登记系统建立

### 高危人群

- 冠心病患者规范治疗
- 心衰患者优化药物治疗
- 遗传性心律失常患者家族筛查

### 特殊人群

- 运动员：运动前心脏筛查
- 围手术期：识别高危患者

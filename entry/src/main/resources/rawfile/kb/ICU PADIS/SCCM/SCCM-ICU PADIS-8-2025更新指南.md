# 2025更新指南

## 概述

自2018年发布PADIS指南以来，多项高质量研究陆续发表。SCCM召集了国际专家、ICU存活者和方法学家，组成了24人跨专业指南工作组，对5个内容领域进行更新：焦虑（新领域）、躁动/镇静、谵妄、不动、睡眠障碍。

**更新时间**：2025年，DOI：10.1097/CCM.0000000000006574

**工作组构成**：24人，包括护士、医生、药师、物理治疗师、心理学家和ICU存活者

**方法学**：GRADE体系，通过系统综述和证据决策框架生成推荐意见

## BENZODIAZEPINES FOR ANXIETY

## Recommendation 1

**无推荐**——苯二氮卓类治疗成人ICU患者焦虑的证据不足，无法给出推荐。

## 问题

在成人ICU焦虑患者中，苯二氮卓类（任何剂量、途径、持续时间或频率）与无苯二氮卓类相比，是否影响患者结局？

## 证据摘要

焦虑是成人ICU患者中最令其痛苦的症状之一（在发生率和强度上均如此）。苯二氮卓类在ICU中常用于治疗急性焦虑。系统检索后，除了一项在烧伤患者中开展的前后对照队列研究（证据有限）外，没有研究直接回答该PICO问题。

## 证据转推荐

由于缺乏证据，panel目前无法对ICU患者使用苯二氮卓类治疗焦虑提出推荐。

## 特别注意事项

- 严重焦虑在插管患者中十分常见——16名患者中有10名在住院期间（无论镇静与否）对气管插管感到焦虑
- 目前迫切需要标准化ICU患者的焦虑评估工具；Faces Anxiety Scale已验证可用于插管和非插管患者
- 非药物治疗方法（如音乐聆听、虚拟现实）已显示前景
- 突然停用治疗慢性焦虑的苯二氮卓类可能诱发急性戒断；因此，入ICU时应考虑继续使用原有的苯二氮卓类

## DEXMEDETOMIDINE FOR SEDATION

## Recommendation 2

**建议在机械通气成人ICU患者中使用右美托咪定优于丙泊酚**——条件性推荐，支持干预，中等质量证据。

适用于：浅镇静和/或减少谵妄为最高优先级时。

## 问题

在成人ICU患者中，右美托咪定（对比丙泊酚）是否适用于镇静？

## 证据摘要

29项RCT（n=3087），大多数明确将药物滴定至浅镇静目标。

| 结局 | 结果 | 证据质量 |
|------|------|---------|
| 谵妄发生率 | RR 0.55（减少15例/100人） | 中等质量 |
| 谵妄持续时间 | MD -25.58小时 | 低质量 |
| ICU住院时间 | MD -0.19天 | 低质量 |
| 机械通气时间 | MD -1.6小时（差异很小） | 低质量 |
| 达到目标镇静的时间 | 改善 | 低质量 |
| 需要补充镇痛药的比例 | RR 0.71（减少） | 中等质量 |
| 长期认知功能 | 不明确 | 极低质量 |
| 28天死亡率 | 不明确 | 极低质量 |
| **不良事件：心动过缓** | RR 1.65（增加6例/100人） | 中等质量 |
| **不良事件：低血压** | RR 1.07（差异很小） | 低质量 |
| 死亡率（最长随访） | RR 0.98（差异很小） | 低质量 |

右美托咪定可能改善6个月时的生活质量和功能状态（均为低质量）。

## 证据转推荐

panel判断：对于大多数ICU患者，右美托咪定的预期益处超过心动过缓的不良反应。但如果患者需要深度镇静或心动过缓风险较高，应考虑其他镇静药物。

成本方面：右美托咪定最初上市时的成本较高，但目前许多医院的获取成本已降低，与丙泊酚大致相当；然而不同地区成本和可及性差异显著，因此本推荐为条件性而非强推荐。

## 特别注意事项

- 目前尚无明确证据显示右美托咪定在65岁以上和以下患者中的安全性或有效性存在异质性
- 所有亚组分析结果仅为假设生成，需进一步研究
- 如果患者需要深度镇静或有高心动过缓风险，应选择其他镇静药物

## ANTIPSYCHOTICS FOR DELIRIUM

## Recommendation 3

**无法给出推荐**——抗精神病药（对比常规治疗）治疗成人ICU患者谵妄，证据不足，无法支持或反对使用（条件性推荐，低质量证据）。

## 问题

在成人ICU谵妄患者中，抗精神病药（任何剂量、途径、持续时间或频率）与无抗精神病药相比，是否影响患者结局？

## 证据摘要

8项RCT（n=1869）。6项使用氟哌啶醇，1项使用典型和 atypical抗精神病药，1项为三组（氟哌啶醇、齐拉西酮、安慰剂）。

| 结局 | 结果 | 证据质量 |
|------|------|---------|
| 无谵妄天数 | MD +1.25天（轻微增加） | 低质量 |
| 28天死亡率 | RR 0.87（轻微降低） | 低质量 |
| 最长随访死亡率 | RR 0.89（轻微降低） | 低质量 |
| 机械通气持续时间 | 差异很小 | 中等质量 |
| ICU住院时间 | 不明确 | 极低质量 |
| 住院时间 | 差异很小 | 低质量 |
| PTSD、功能状态（3个月）、生活质量、认知障碍 | 不明确 | 极低质量 |
| 不良事件：心律失常 | RR 1.57（轻微增加） | 低质量 |
| QT间期延长、锥体外系症状 | 差异很小 | 低质量 |

## 证据转推荐

panel未能给出推荐，原因：
1. 所有潜在获益均为低质量证据（部分因精确性不足）
2. 不清楚抗精神病药如何降低死亡率——唯一检测到的变化是无谵妄天数轻微改善，但其他结局（如机械通气时间、ICU住院时间）无变化
3. 无谵妄结局未考虑昏迷——患者可能因正常精神状态或因昏迷而无谵妄

## 特别注意事项

- 大多数RCT纳入了混合型（低活动型伴和/或高活动型）谵妄患者，因此不清楚哪类患者最受益
- 如使用，必须谨慎在ICU出院时停用新启用的抗精神病药（如果临床不再需要）

## ENHANCED MOBILIZATION FOR IMMOBILITY

## Recommendation 4

**建议在成人ICU患者中提供强化康复/活动优于常规康复/活动**（条件性推荐，支持干预，中等质量证据）。

## 问题

在成人ICU患者中，强化康复/活动（对比常规护理康复/活动）是否影响患者结局？

## 增强的定义

本版采用先前建立的定义：
- **活动（Mobilization）**："改变和维持姿势的过程"，排除肺康复
- **增强（Enhanced）**：超过所在科室常规康复/活动的任何形式（如：骑车、踏步、早期、每日两次、方案化或延长的康复/活动时间）

## 证据摘要

58项RCT（n=8038）。

| 结局 | 结果 | 证据质量 |
|------|------|---------|
| ICU获得性肌无力发生率 | RR 0.77（减少9例/100人） | 高质量 |
| 谵妄持续时间 | MD -1.34天 | 低质量 |
| 机械通气持续时间 | MD -1.1天 | 中等质量 |
| ICU住院时间 | MD -1.01天 | 低质量 |
| 住院时间 | MD -1.16天 | 低质量 |
| 出院回家比例 | RR 1.15（增加） | 低质量 |
| 功能结局（出院后） | 改善 | 中等质量 |
| 生活质量（出院后） | 改善 | 中等质量 |
| 死亡率（最长随访） | 差异很小 | 低质量 |
| **不良事件：心律失常** | RR 2.27（轻微增加） | 中等质量 |
| 意外管路移除 | RR 1.28（差异很小） | 中等质量 |
| 意外拔管 | RR 0.93（差异很小） | 中等质量 |
| 低血压 | RR 1.32（差异很小） | 中等质量 |

## 证据转推荐

强化康复/活动在成人ICU患者中的益处大于不良事件（如心律失常）略微增加的风险。不良事件发生率低。RCT未报告ICU强化康复/活动项目的所需资源，因此panel发出条件性推荐，承认资源限制可能是实施该计划的重要障碍。然而，实施强化康复/活动可能为医疗系统带来重要的节约。此外，RCT主要纳入基线功能独立患者，因此推荐可能不适用于所有患者。panel未能就最佳康复/活动剂量、时机、持续时间或方式给出推荐（因纳入研究的异质性）。

## 特别注意事项

- 活动是一项多因素、团队干预，在复杂环境中进行；基于临床团队专业知识，频率、强度、时机和类型可能因ICU而异
- 无足够数据对心脏手术患者进行亚组分析；因此本推荐适用于一般内外科患者

## MELATONIN FOR PATIENT OUTCOMES

## Recommendation 5

**建议在成人ICU患者中给予褪黑素优于无褪黑素**（条件性推荐，支持干预，低质量证据）。

## 问题

在成人ICU患者中，褪黑素（对比安慰剂）是否影响患者结局？

## 证据摘要

30项RCT（n=3739）。24项比较褪黑素vs安慰剂，2项vs右美托咪定，1项vs苯二氮卓类，3项比较雷美尔通道vs安慰剂。

| 结局 | 结果 | 证据质量 |
|------|------|---------|
| 谵妄发生率 | RR 0.70（减少） | 低质量 |
| 谵妄持续时间 | MD -0.16天（差异很小） | 低质量 |
| 主观睡眠质量 | 改善 | 低质量 |
| 总夜间睡眠时间 | MD +0.04小时（差异很小） | 中等质量 |
| ICU住院时间 | MD -0.5天（轻微减少） | 低质量 |
| 机械通气时间 | 差异很小 | 低质量 |
| 住院时间 | 差异很小 | 低质量 |
| 觉醒次数、焦虑、激动比例、认知功能、PTSD | 不明确 | 极低质量 |
| ICU死亡率、功能状态（出院后） | 不明确 | 低质量 |
| 不良事件（研究者定义） | 轻微减少 | 低质量 |

## 证据转推荐

ICU实践中常使用褪黑素来减轻危重症成人的睡眠和昼夜节律紊乱的负面影响。内源性褪黑素抑制和昼夜节律紊乱在危重症患者中存在，为补充治疗提供了理论依据。条件性推荐支持褪黑素是基于：谵妄发生率降低和睡眠质量主观改善，加上不良反应风险低。由于数据存在局限性，未能给出强推荐。多数RCT未评估睡眠/昼夜节律结局；使用主观测量工具的RCT可能引入回忆偏倚，且只能纳入可应答的患者，不能测量睡眠结构。

## 特别注意事项

- 褪黑素未经美国FDA监管，因此质量可能存在差异
- 雷美尔通（FDA批准的褪黑素受体激动剂）在有该药物的国家可作为褪黑素的替代品
- 纳入试验在褪黑素剂量、持续时间和频率方面存在显著异质性，限制了具体的给药建议
- 不同国家成本可能不同

## 研究日程（Research Agenda）

### 焦虑

- 进一步开发和验证检测/量化危重症成人焦虑的工具
- 测试苯二氮卓类和其他药物治疗危重症成人焦虑安全性和有效性的RCT
- 检查非药物和/或多模式方法治疗危重症成人焦虑的RCT

### 镇静

- 右美托咪定在酒精戒断中的作用
- 镇痛优先方案背景下镇静药物的作用需要更充分的研究
- 评估ICU期间患者与家属沟通价值的研究，以及患者对丙泊酚vs右美托咪定的感知
- 需要深度镇静的患者中右美托咪定的作用
- 明确解决右美托咪定按年龄（65岁以上或以下）效果异质性的RCT

### 谵妄

- 在高活动型谵妄、低活动型谵妄（作为独立人群）和其他谵妄亚型中使用抗精神病药的RCT
- 检查ICU入院时使用抗精神病药患者长期认知效应的RCT
- 在进行性谵妄严重程度中使用抗精神病药的RCT

### 不动

- 确定强化康复/活动作为谵妄预防vs治疗的impact
- 确定让其他利益相关方（如家属）在当前资源短缺情况下协助康复/活动的可能性
- 确定最佳强化康复/活动模式，包括频率、持续时间和从ICU入院开始的启动时间
- 在资源有限的环境中实施ICU强化康复/活动计划

### 睡眠

- 研究夜间给药褪黑素及严格评估睡眠质量和数量的RCT
- 关于褪黑素适当剂量的RCT

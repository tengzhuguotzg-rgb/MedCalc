# 2. Methodology

## 指南概况

本实践指南共包含 **56条推荐意见**，基于2019年发布的ESPEN ICU临床营养指南原文。

## 精简原则

- 精简内容聚焦于证据和文献基础，保留推荐意见的核心依据
- 部分推荐意见根据最新证据进行了修订
- 呈现形式改为流程图，提高临床可操作性

## 证据与推荐分级

方法学参照 **苏格兰校际指南网络（SIGN）** 的标准操作程序（SOP）。

| 分级 | 含义 |
|------|------|
| **1级** | 强证据 |
| **2级** | 中等证据 |
| **3级** | 低证据 |
| **4级** | 专家意见 |

推荐强度分为四类：

| 推荐强度 | 说明 |
|----------|------|
| **A** | 强有力的推荐 |
| **B** | 中等推荐 |
| **0** | 弱推荐/不确定 |
| **GPP** | 最佳实践声明（基于专家共识） |

## 共识过程

所有推荐意见均经过多阶段共识流程达成，并记录共识率（%）。

本次版本更新要点：
- **R21、R22**：根据最新证据更新
- **R33、R35**：根据最新证据更新
- **R36、R37**：合并为一条推荐
- **R26（原）→ 现R30**：待下次修订时重新评估

指南制定过程由ESPEN独家资助。
